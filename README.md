## 📅 本週訓練重點：W20 基礎期重建圓滿週

**1. 訓練重點：基礎期收官與步頻技術內化**
本週是「基礎期重建 (Base Phase Rebuilding)」的第四週，也是該週期的最後一戰。這週的核心任務是將前三週累積的有氧底蘊進行深度的「固化」。在您下週啟程前往日本（預期跑量將大幅降低，進入戰略性減量）之前，我們必須給予身體一次紮實的有氧刺激。同時，這週的另一項重點是「技術動作的內化」，我們要求將高步頻與低觸地時間的技巧，深深刻印在肌肉記憶中。

**2. 核心目標：有氧引擎極大化與關節衝擊極小化**
*   **技術維度：落實步頻革命**。全週所有跑步課表，無論配速快慢，強制要求將步頻穩定在 175-185 spm。練習「腳掌觸地即刻彈起」的體感，想像地面是發燙的，藉由縮小步幅來提升頻率，從根本上降低對膝蓋與足踝的垂直衝擊力。
*   **代謝維度：穩固有氧區間 (Z2)**。透過大量的輕鬆跑，強迫身體在低心率 (153-166 bpm) 的狀態下運作，促進微血管增生與線粒體變大變多，讓身體成為一個更有效率的「燃脂機器」。
*   **心肺維度：維繫乳酸門檻**。週三的 T-Pace 間歇跑旨在不累積過量疲勞的前提下，輕觸無氧系統，保持血液清除乳酸的效率，讓未來的馬拉松配速 (M-Pace) 感覺更為輕鬆。

**3. 關鍵課表：12km 週末長跑 (Long Run)**
本週六安排的 12km 長跑是基礎期以來最具指標性的一練，旨在模擬雪梨馬拉松中後段的體能與心理消耗。
*   **模擬實戰**：這不僅是距離的累積，更是補給策略與專注力的演練。
*   **配速策略**：嚴格執行「負分割 (Negative Split)」，前 3 公里刻意壓抑速度，讓心率平緩上升。
*   **專注力挑戰**：在最後 3 公里，當體力下滑時，正是考驗核心穩定與步頻不掉速的關鍵時刻。

**4. 預期成效：代謝效率的實質躍升**
順利完成本週的訓練後，您將會感受到身體產生的質變。在相同的 Z2 心率負荷下，您的配速將會比數週前更為輕快；而面臨相同配速時，您的呼吸將更為從容。這象徵著您的有氧底盤已經成功拓寬，為未來的高強度專項訓練打下了堅不可摧的基礎。

**5. Execution 執行建議：精準致勝**
*   **心率絕對優先**：在執行 Z2 輕鬆跑時，若因氣溫、濕度或當日體能狀況導致心率飆升，請務必毫不猶豫地降速，切勿被配速數字綁架。
*   **戶外路感適應**：本週六的 12km 長距離請務必移步至戶外進行。雪梨馬拉松的賽道起伏多變，您需要真實柏油路面的反饋來鍛鍊下肢的微小穩定肌群。
*   **動態修復不間斷**：週一、週四與週日的瑜伽課表是這份計畫的隱形功臣。請帶著覺察進行每一個伸展，它們能有效清除肌肉內的微小結節，保持身體的最佳彈性。
