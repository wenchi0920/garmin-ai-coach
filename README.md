## 📅 4. 本週訓練重點 (W20 基礎期重建)

本週 (W20) 是「基礎期重建 (Base Phase Rebuilding)」的第四週，也是該週期的最後一戰。我們的任務是將前三週的訓練效果進行深度固化，為即將到來的減量週（日本旅遊）與後續的強化期做好萬全準備。以下是本週的五大訓練重點與執行方針：

**一、 訓練重點：基礎期收尾與步頻技術徹底內化**
在我們準備進入下個訓練週期之前，本週的重點在於「技術檢核」與「有氧鞏固」。您的心肺能力已經證明能夠勝任當前的負荷，但我們必須將注意力轉移到「跑步經濟性」上。核心任務是強制將高步頻與低觸地時間的技巧刻印在肌肉記憶中，讓這成為您未來面對疲勞時的本能反應。

**二、 核心目標：確立有氧底盤與高步頻動作經濟性**
本週的所有跑步課表，無論是在跑步機還是戶外，首要且唯一的技術目標就是將步頻穩定拉升並維持在 **175-185 spm** 之間。從您近期的數據（如 5/10 跑步機的 158 spm、5/9 戶外的 156 spm）來看，您的步幅普遍偏大（約 0.88m - 0.93m）。這在長距離賽事中，會成倍增加膝蓋與足踝的垂直衝擊力。本週的目標是學習「快速抽腿」，縮小步幅，想像地面發燙，縮短腳掌停留地面的時間（目標觸地時間 <260ms），藉此從根本上降低受傷風險。

**三、 關鍵課表：12km 週末戶外長跑 (Long Run) 及乳酸門檻 T 區間歇**
本週的訓練高潮有兩處：首先是週三安排的 **4km T-Pace 間歇 (4x1km)**，這將測試您在乳酸門檻附近的排除能力與心肺韌性；其次是週六的 **12km 長跑 (Long Run)**。這場長跑不僅是耐力考驗，更是針對虛擬賽事（TATTA RUN 與 ONE TOKYO）的里程累積。長跑要求必須在戶外執行，模擬真實路面與風阻，並嚴格執行「負分割 (Negative Split)」，前半段刻意壓抑速度，後半段在疲勞時專注於核心穩定。

**四、 預期成效：擴張有氧區間，實現代謝效率的質變**
若能確實貫徹本週的高步頻與 Z2 心率控制，您將感受到身體在代謝效率上的顯著躍升。我們預期您在相同的 Z2 輕鬆跑心率（約 145-155 bpm）下，配速能自然變得更加輕快；而在面對 T-Pace 間歇時，恢復期心率下降的速度也會比以往更快。這象徵著您的微血管密度增加，粒線體運作更具效率，為 8 月雪梨馬拉松的高強度專項訓練打造出堅不可摧的防護罩。

**五、 執行建議：心率絕對優先、戶外適應與心靈放鬆**
在執行 Z2 輕鬆跑時，請務必秉持「心率絕對優先」的原則。若當天天氣悶熱或體能略降導致心率逼近 Z3，請毫不猶豫地降低配速，絕不要被手錶上的速度數字綁架。此外，由於多項線上賽事均要求 GPS 軌跡，請盡量將訓練從跑步機轉移至戶外。同時，延續您近期優良的瑜伽習慣，在每次高強度訓練後進行動態修復，帶著覺察去感受每一吋肌肉的拉伸，這不僅是身體的恢復，更是心靈的冥想與準備。
