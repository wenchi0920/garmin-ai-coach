# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個整合運動生理學與大數據分析的頂尖科學化馬拉松訓練系統。本系統以「數據導航，無痛增長」為核心理念，每日自動讀取並解析跑者的 Garmin 穿戴裝置數據。透過對心率、步頻、配速、HRV（心率變異度）與身體能量 (BB) 等關鍵生理指標的長期追蹤，動態評估跑力值 (VDOT)，精準校準個人化的訓練區間（包括輕鬆跑 E、馬拉松配速 M、乳酸門檻 T 與間歇跑 I）。本系統在給予生理系統「適度超載刺激」與確保「超補償恢復」之間取得黃金平衡，協助跑者在不累積傷痛的前提下，穩步提升有氧耐力與跑步經濟性，系統化地實現賽事目標，達成個人體能天花板的突破。本系統更著重於防傷機制的動態調整，當生理指標出現疲勞警訊時，會即時啟動避險機制，引導跑者在安全且科學的框架下前行。

## 📅 4. 本週訓練重點 (W29)

本週（2026-07-13 ~ 2026-07-19）為「進展期後段 (W28-W30)」的第二週。本週的訓練邏輯在於**「穩步擴大有氧容量、拉長乳酸門檻耐受時間，並以高步頻防禦機制應對里程突破」**。本週預計總跑量將穩步爬升至 **39 公里**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：LSD 里程突破與乳酸門檻的疲勞債退避**
    本週的長跑日（週六）將迎來關鍵的突破，LSD 里程由上週的 14km 提升至 **16km**。這 2 公里的增量，是對下肢肌肉耐力與心肺系統的一次重要超載刺激，旨在逐步適應雪梨馬拉松後半段的耐力需求。此外，原定週二質量跑的 T 配速（乳酸門檻跑）總長度將從 6km 提高至 **7km**（維持 5'58"/km）。然而，週二執行時因配速失控爆衝且高溫悶熱，完成 5.62km 後中斷。在週四安靜心率與 HRV 出現疲勞警訊的情況下，系統決定**週四跑步課表不進行補課，週五嚴格休息**，以退為進，全力確保週六 16km LSD 的完成品質。
*   **二、 核心目標：步頻 170-175 spm 的穩態維持與補給窗口演練**
    1.  **步頻穩態控制**：全體跑步課表均搭配手錶步頻警示，強制將雙邊步頻穩定在 **170-175 spm** 之間。透過「高步頻、小步幅」的跑姿，縮短觸地時間，控制垂直振幅在 75mm 以下，從而最大程度降低每次落地的煞車效應與關節衝擊力，尤其在週四完成下肢 950kg 重訓後，肌肉纖維微損，高步頻是唯一的防傷防線。
    2.  **配速與心率區間守恆**：週六 LSD 必須將心率壓制在 Zone 2 (131-143 bpm) 的有氧區間內。尤其週六 LSD 前 3 公里必須壓抑速度，以 **7'45" - 8'00"/km** 慢速起跑，防止心率過早漂移。
    3.  **腸胃補給窗口演練**：利用週六的 16km 長跑，進行雪梨馬拉松的補給模擬。分別於 **第 6 公里與第 12 公里** 定時補給一包能量果膠，每次補給均配合 150ml 的水分，演練腸胃在高運動強度下的吸收能力，防止賽事當天發生腸胃不適。

*   **三、 關鍵課表：下肢重訓的肌肉超補償與 16km LSD 戶外實戰**
    1.  **週四下半身肌力訓練（已執行）**：完成了 34 分鐘的力量訓練，包括深蹲 50kg、硬舉 40kg 與 45度腿推。本次神經元激活非常成功，但為下肢帶來了機械壓力。
    2.  **週五完全休息與活動度保養（今日）**：不跑步，進行 20 分鐘與 15 分鐘的睡前瑜伽，針對髖部（鴿子式/蜥蜴式）與小腿後側進行深度伸展，確保肌肉與神動完全放鬆，為週六備戰。
    3.  **週六 16km LSD 長距離慢跑（核心）**：**強制戶外 GPS 跑**。1km Z1 暖身 + 14km Z2 有氧慢跑 (配速: 7'01"-7'51"/km, 心率 131-143 bpm) + 1km Z1 緩跑。本課表核心在於「穩」，中後段以 170+ spm 步頻防守，嚴格執行 6k / 12k 的補給紀律。
*   **四、 預期成效：提升跑步經濟性與延緩後半程心率漂移**
    透過將步頻穩定在 170-175 spm，學員的平均觸地時間預期能穩定控制在 **270ms 以下**，垂直振幅降至 **75mm 以下**。這將大幅降低股四頭肌的離心收縮負擔，減少肌肉微細撕裂。雖然週二 7km T 門檻跑因爆衝中斷，但週六 LSD 的有氧刺激，將持續拓寬學員的有氧底層容量。若能落實 LSD 前 3 公里壓抑配速與定時補給，將能有效延緩後半程的心率漂移，使心率曲線平緩，並確保在 12km 後腿部肌肉仍有充足的力量支持，不至於因肌肉崩潰而中途停步。
*   **五、 執行建議：手錶步頻輔助、熱應激防護與避險退場機制**
    1.  **步頻節拍器輔助**：跑步時請開啟手錶的步頻節拍器（設定 172 bpm），利用聽覺回饋強制跟上步頻，專注於「小步幅、輕落地、骨盆送髖」的感覺。
    2.  **熱應激防護**：7 月中旬高溫濕熱是心率漂移的最大干擾源。週六 LSD 務必安排在**清晨 05:30 - 07:30** 或**傍晚 18:00 以後**進行，隨身攜帶水袋或選擇有補水點的路線，嚴格防範中暑與嚴重脫水。
    3.  **避險與退場機制**：由於近期出現疲勞（HRV 不平衡），本週執行 LSD 時，若在配速未變的情況下發現心率飆升至 155 bpm 以上 (Zone 3/Zone 4 邊界)，或感覺下肢膝關節、足底有針刺般的銳痛，應立即由跑步切換為快走，甚至中止訓練。科學訓練的第一信條是無傷。
