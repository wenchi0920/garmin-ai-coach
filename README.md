# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個結合運動生理學、穿戴裝置大數據分析與動態自動化課表生成的頂尖科學化馬拉松訓練系統。本系統以「數據驅動，精確校準，無痛增長」為核心理念，每日自動讀取與解析跑者的 Garmin 穿戴裝置數據（包括 .fit 檔、心率、步頻、HRV 心率變異度、身體能量 BB、靜息心率 RHR 與睡眠品質等）。透過對個人化動態跑力 (VDOT) 的動態校準，精準設定輕鬆跑 (E)、馬拉松配速 (M)、乳酸門檻 (T) 及間歇跑 (I) 之配速與心率區間。系統在給予生理超載刺激與確保超補償修復之間取得黃金平衡，協助跑者突破耐力天花板。同時，系統設有強大的防傷避險機制，當生理指標出現疲勞警訊時即時調整強度，陪伴跑者安全、系統化地實現賽事目標。

依據 10 筆訓練數據動態校準，學員當前跑力評估為 **VDOT 34**。在此跑力基準下，個人化訓練區間校準如下：

* **配速強弱區劃**：Z1 ( > 7'51"/km )、Z2 ( 7'01" - 7'51"/km )、Z3 ( 6'23" - 7'01"/km )、Z4 ( 5'58" - 6'23"/km )、Z5 ( < 5'58"/km )。
* **儲備心率 (HRR) 區間**：Z1 ( 119 - 131 bpm )、Z2 ( 131 - 143 bpm )、Z3 ( 143 - 155 bpm )、Z4 ( 155 - 167 bpm )、Z5 ( 167 - 180 bpm )。

賽事發展規劃包含三階段里程碑：核心大賽 2026-08-30 雪梨馬 (Sub 4:30, 6'23"/km)；中期目標 2026-12-06 那霸馬 (Sub 5:00, 7'06"/km，演練控速與補給防抽筋)；長期終極目標 2027-03-07 東京馬 (Sub 4:10, 5'55"/km)，屆時需將 VDOT 逐步提升至 38 左右。

## 📅 4. 本週訓練重點 (W30)

本週（2026-07-20 ~ 2026-07-26）為「進展期後段 (W28-W30)」的第三週（高峰鞏固週）。本週訓練邏輯的核心在於**「穩步擴大有氧容量、拉長乳酸門檻耐受時間，並以高步頻防禦機制與補給演練應對高峰里程突破」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：高峰鞏固期的負荷控制與防禦控速**  
    本週為備戰雪梨馬拉松進展期後段的最高峰週（W30 高峰鞏固週），預計總跑量達到 41 公里，為備賽以來的最高峰。由於上週累積了顯著的自主神經疲勞，且多次出現配速失控教訓，本週的重點不是追求盲目速度突破，而是「在安全受控的配速與心率區間內，穩定完成里程累積」。特別是週六將迎來備賽以來最長的 17km LSD 挑戰，因此平日的非質量跑（輕鬆跑與恢復跑）必須執行「防禦性控速」，將強度鎖定在 Z2/Z1 區間（心率 < 143/155 bpm），促使身體產生超補償修復。

*   **二、 核心目標：17km 實戰補給演練與配速守恆**  
    1. **ONE TOKYO Global Virtual Race 4th 虛擬賽與補給演練**：週六（07/25）安排的 17km LSD 戶外長跑為備賽期單次最長里程突破。要求全程戶外 GPS 記錄，學員必須演練雪梨馬拉松實戰補給策略，於 6km 與 12km 處定時補充能量膠與電解質水，測試胃部高運動量下的吸收力。  
    2. **重塑配速與心率紀律**：週二 11km T 門檻跑（含 7km 門檻區段）已順利以 6'07"/km 均速完成。週四 8km 輕鬆跑心率需嚴格控制在 Z2 區間（< 143 bpm 尤佳）。  
    3. **維持下肢穩定力量**：週一（上半身）與週四（下半身）重訓重點放在單腳穩定性（單腳硬舉、保加利亞分腿蹲），強化臀中肌與後側鏈剛性，為週六 17km 長跑與雪梨馬丘陵地形建立關節抗衝擊能力。

*   **三、 關鍵課表：T門檻跑的紀律重建與LSD長跑實戰**  
    1. **週二 11km T 門檻跑 (07/21)**：已執行 5.64km 跑步機 T門檻跑，平均配速 6'07"/km（主區段 5'45"-5'52"/km），步頻 174 spm，順利建立高配速巡航感與乳酸耐受力。  
    2. **週四 8km Z2 基礎有氧跑 (07/23)**：1km Z1 暖身 + 6km Z2 基礎有氧 (配速 7'01"-7'51"/km，心率控制 131-143 bpm) + 1km Z1 緩跑。重點在於「慢」，搭配 Garmin 心率警示，促進肌群毛細血管增生。  
    3. **週六 17km LSD 戶外跑 (07/25)**：1km Z1 暖身 + 15km Z2 有氧慢跑 (配速 7'01"-7'51"/km，心率 131-143 bpm) + 1km Z1 緩跑。必須戶外執行，模擬雪梨冬季海風與坡度，維持 170+ spm 步頻並執行定點補給。

*   **四、 預期成效：乳酸清除率優化、跑步經濟性提升與補給胃部適應**  
    1. **乳酸清除能力提高**：透過精準的 7km 乳酸門檻刺激，提升乳酸轉運蛋白活性，使身體在乳酸堆積與清除間達到更高動態平衡，提高高強度下的配速耐受力，為雪梨馬拉松 Sub 4:30（目標均速 6'23"/km）奠定配速基石。  
    2. **有氧引擎深化與力學效率提升**：週四 8K 基礎跑與週六 17K LSD 的低心率刺激，促進左心室離心肥大，增加每搏輸出量，並提高脂肪氧化比例。同時，透過 170+ spm 高步頻維持，將觸地時間控制在 260ms 以下，提升跑步經濟性。  
    3. **補給系統建立**：週六長跑的定點能量膠補給，能有效預防賽事後半程醣原耗竭（撞牆期），並訓練胃部在高運動量下的吸收與排空能力，避免雪梨馬拉松後半段發生抽筋或腸胃不適。

*   **五、 執行建議：手錶配速/心率警示雙硬約束、下肢活動度管理**  
    1. **手錶警示硬約束**：跑前必須在 Garmin 965 上設定「配速警示（E跑下限 7'51"/km，上限 7'01"/km）」與「心率警示（Z2 上限 143 bpm，T跑上限 167 bpm）」。警示觸發時立即放鬆摆臂並縮小步幅，將速度降回預定區間。  
    2. **上坡與逆風體感守恆**：週六戶外 LSD 遭遇上坡或強風時，不刻意維持配速，而以「心率守恆」為核心，放慢速度、縮短步長，將心率壓在 Z2 區間，下坡時順行放鬆，嚴防股四頭肌離心受衝擊。  
    3. **睡前瑜伽修復防線**：每日睡前瑜伽 (10-15 分鐘) 固定於 22:00 前執行，專注下犬式（拉伸小腿與足底）與鴿子式（釋放髖部與臀肌），保證 23:00 前入睡，為低充電的 Body Battery 提供充分修復。
