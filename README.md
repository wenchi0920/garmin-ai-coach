## 📅 本週訓練重點：W20 基礎期重建圓滿週

**1. 訓練重點：有氧效能固化與乳酸門檻喚醒**
本週是「基礎期重建」的最終章。核心任務是將 Z2 輕鬆跑的步頻穩定在 170 spm 以上，並透過週三的 T-Pace (乳酸門檻) 課表，重新喚醒身體代謝乳酸的能力。這不僅是體能的累積，更是為 6 月份進入「馬拉松配速期 (M-Phase)」做生理與心理的雙重銜接。

**2. 核心目標：頻率革命與耐力門檻**
*   **技術指標**：在所有 Z2 跑中維持 170+ spm，週三 T-Pace 挑戰 180 spm。縮短觸地時間至 260ms 以下。
*   **生理指標**：週六 12km 長跑中心率需穩定在 Z1/Z2 銜接地帶 (153 bpm 左右)，測試長距離下的心率穩定性。
*   **心理指標**：建立「掌控配速」的自信，特別是 T-Pace 時的呼吸節律控制 (建議 2-2 呼吸法)。

**3. 關鍵課表深度解析：**
*   **05/13 (三) 4km T-Pace 間歇 (4x1km)**：這是本週最關鍵的強度刺激。T-Pace 的目的是提升「乳酸門檻」。透過這種刺激，您的有氧區間上限會被往上推，這會讓未來雪梨馬 6'23" 的目標配速在體感上從「吃力」轉變為「舒適」。
*   **05/16 (六) 12km 長跑 (Long Run)**：本週期里程高峰。這不是一般的慢跑，而是一次「耐力轉化」。重點在於 9km 後的姿勢維持，檢查核心是否疲勞，步頻是否因疲勞而掉下。這是模擬全馬最後 10 公里心態的最佳時機。

**4. 預期成效：**
完成 W20 後，您的身體將產生「隱形跑力提升」。雖然 VDOT 數據可能短期維持在 33，但您的「配速/心率成本」將大幅下降。這代表同樣的配速下，您的心臟跳動次數減少了，節省下來的能量就是您在雪梨馬 35 公里後的保命錢。

**5. Execution 執行建議：**
*   **強制節拍器**：請將 Garmin 965 設定 175 bpm 節拍器，所有跑步行程強制對位。
*   **戶外優先原則**：為因應雪梨馬的真實地形（坡度、路感），週三強度課表與週六長跑請務必在戶外執行，避免跑步機對步態的簡化。
*   **肌力強化 (Single-leg Stability)**：週二與週五的徒手肌力請著重「單腳硬舉」，強化足踝與臀中肌，這是支撐高步頻、預防傷勢的技術地基。
