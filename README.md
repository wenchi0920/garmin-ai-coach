## 1. 系統核心定位
Garmin AI Coach 是一個專為跑者量身打造的科學化訓練計畫系統。系統的核心定位在於「科學訓練，精確校準」，透過自動化整合與分析 Garmin 手錶記錄的 `.fit` 運動數據，我們能夠深入洞察跑者的心率變異、跑步動態與疲勞負荷。結合教練的專業馬拉松週期化訓練理論，AI 系統會自動為您計算當前的 VDOT 跑力指標，並精準劃分各個心率與配速區間。我們不僅能自動規劃每週包含有氧基礎 (Z2)、乳酸門檻 (T)、長距離慢跑 (LSD) 以及專項肌力與恢復瑜伽的課表，更會根據您每日的恢復狀態與訓練達成率進行動態校準。透過這套自動化機制，跑者不再需要為繁瑣的排表與數據解讀煩心，只需百分之百專注於訓練的執行。在追求卓越成績的同時，AI Coach 嚴格監控您的健康指標，降低受傷風險，讓運動不僅是肉體的鍛鍊，更是一場體驗心靈成長與豐富的旅程，穩健邁向每一次的起跑線。

## 📅 4. 本週訓練重點 (W23)
本週 (W23) 我們已正式開啟為期數週的「進展期初期 (Strengthening Phase)」。距離雪梨馬拉松剩餘不到三個月（約 12 週），我們必須在持續擴張有氧底盤的同時，重新喚醒您的門檻速度，並透過特定肌力訓練增強下肢的抗疲勞剛性。以下是本週課表的五大核心剖析：

**一、 訓練重點：乳酸門檻刺激與基礎有氧擴張 (Threshold & Endurance)**
本週的訓練邏輯採取「雙軸並進」策略。第一軸著重於「T 門檻刺激」，目的在訓練您的身體於乳酸即將大量堆積的臨界點（約 6:07/km）提升「耐受度」與「清除效率」。這能實質拉高您在無氧邊緣的續航力。第二軸則是週末的重頭戲「16km LSD 長距離慢跑」。透過超過 90 分鐘以上的持續跑動，強迫身體動用脂肪作為主要能量來源，促進微血管網與粒線體的增生，這是全馬要達成 Sub 4:30 且避免撞牆期的最關鍵基礎。

**二、 核心目標：重建下肢抗疲勞能力與核心穩定 (Leg Strength & Core Stability)**
雪梨馬拉松賽道以其充滿挑戰的起伏丘陵地形著稱，頻繁的上下坡對肌肉的離心收縮要求極高。因此，本週的核心目標之一是透過高質量的重量訓練（如肩部穩定與階梯運動），直接強化您的下肢推蹬能力與上肢抗疲勞度。當比賽進入後半段（30km 後）體能大幅下滑時，強韌的軀幹與穩定的核心將是您維持標準跑姿、防止步幅縮水與避免抽筋受傷的最強防線。

**三、 關鍵課表：T 門檻跑與長距離耐力考驗 (Key Workouts)**
*   **W23-6km T 門檻跑**：本週四您在室內順利完成了 T 配速的嘗試，甚至在 6'15"/km 的配速下心率依然寬裕。這意味著您的心肺潛力極大。下次執行 T 區間課表時，請勇敢嘗試穩定鎖定在 6:07/km，讓身體感受真正的門檻壓力。
*   **W23-16km LSD 長跑**：本週日的終極考驗，也是您完成 ONE TOKYO VR 21.1K 挑戰的決定性戰役。請嚴格鎖定 Z2 區間 (7:12-8:02/km)，將專注力放在呼吸節律的順暢度與高步頻的維持，切勿盲目追快。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能貫徹執行本週的質量與有氧課表，我們預期能觀察到顯著的生理適應。您的心肺系統將更能適應較高的需氧量，未來在執行 Z2 輕鬆跑時，同等配速下的心率將會下降，體感疲勞度也會大幅減輕。此外，藉由肌力訓練與長跑的雙重刺激，您足弓、阿基里斯腱及膝關節韌帶的彈性將被強化，進一步提升跑步經濟性。這種全身性的綜合強化，將讓您在面對雪梨馬拉松的各種複雜路況時，都能游刃有餘。

**五、 執行建議：高步頻防線與夏日實戰補給演練 (Execution Advice)**
*   **嚴守高步頻防線**：請在所有跑步過程中時刻提醒自己維持 170-175 spm 以上的高步頻。這能有效縮短觸地時間，分散每一步的衝擊力，保護膝關節免受煞車效應的傷害。
*   **實戰補給演練**：對於耗時較長的 16km LSD，請務必攜帶能量膠，嘗試每 45-50 分鐘補充一次，並定時定量攝取水分。這是訓練腸胃在跑動中吸收能力的最佳時機，切勿等到口渴才喝水。
*   **環境熱適應**：夏季氣溫高，心率漂移是正常的生理現象。請以「心率區間」為最高指導原則，若體感過於悶熱或心率飆升，請果斷放慢配速。夏訓的目的是生理刺激與適應，安全永遠是第一要務。
