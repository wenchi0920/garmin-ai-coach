## 1. 系統核心定位

## 📅 4. 本週訓練重點 (W24)
本週已進入雪梨馬拉松備賽的「進展期初期」，這是決定您整體耐力天花板與完賽自信心的關鍵過渡階段。

**一、 訓練重點：修補耐力斷層與優化跑步經濟性 (Endurance Rebuilding)**
由於前兩週受氣候影響，里程數出現明顯斷層，特別是長距離訓練的缺席。本週首要任務是「有氧質量的補償」。不需要追求極速，而是要確保每一公里的 Z2 輕鬆跑與長跑都能精準落在設定區間內。透過穩定里程堆疊，喚醒身體脂肪供能的依賴，並透過高步頻優化跑步經濟性。

**二、 核心目標：力量與耐力的平穩轉化 (Strength-to-Speed Conversion)**
近期您在重訓上取得顯著進步，本週的目標是將「力量」轉化為「跑動力」。我們要在週末的 16km LSD 中測試雙腿承重能力，並維持 175 spm 以上的步頻。穩固的核心能確保雪梨馬賽道上的骨盆穩定，避免疲勞傾斜與抽筋。

**三、 關鍵課表：16km 韌性挑戰與週末有氧擴充 (Key Sessions)**
本週後半段的訓練將直接決定當週成效：
*   **06/13 (週六) 6km Z2 輕鬆跑**：輕量有氧，維持肌肉彈性。請嚴格守在 7:12-8:02 /km 配速，絕不超速，把能量保留給隔日。
*   **06/14 (週日) 16km LSD 長跑**：本週最核心的課表，雪梨馬模擬跑。目標「不間斷完成」，保持 Z2 高標心率。這是一場心理韌性建設，必須完成對長距離的征服。

**四、 預期成效：心肺飄移抑制與恢復彈性提升 (Expected Gains)**
若能順利完成週末計畫，預期您的心肺飄移現象將明顯改善，有氧效率獲得實質提升。同時，隨著充分睡眠與肌力轉化，基礎代謝與耐酸能力將獲得鞏固，使您未來面對更高強度訓練時能恢復得更快。

**五、 執行建議：精準補給與環境適應 (Execution Strategy)**
*   **熱適應與時段選擇**：台灣高溫下，請盡量於清晨或傍晚執行長跑。以「心率」為最高指導原則，不強求配速。
*   **補給演練**：週日 16km 長跑務必實戰演練補水與能量膠（每 7km/12km 嘗試），讓腸胃提早適應賽時狀態。
*   **步頻監控**：隨時提醒自己加快交換腳的速度，將步頻穩定在 175 spm，避免疲勞時掉速。
