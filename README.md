## 1. 系統核心定位
Garmin AI Coach 是一個結合數據分析與自動化課表的專屬科學化訓練計畫系統。系統透過自動解析 Garmin 的 `.fit` 檔案，深入了解跑者的跑步動態與生理負荷，從而實現精準的數據驅動訓練。系統將教練的專業知識與週期化訓練理論（涵蓋有氧、乳酸門檻、肌力與恢復）結合，能自動針對跑者的 VDOT 跑力指標、心率區間以及最新恢復狀態進行動態校準。不僅如此，自動化規劃與上傳每週課表的功能，大幅節省了跑者手動安排與紀錄的時間，讓您能百分之百專注於訓練執行。我們的核心理念是「科學訓練，精確校準」，在確保跑者遠離傷痛的同時，幫助您在每一次呼吸與步伐中超越自我，穩健地朝向賽事目標邁進。

## 📅 4. 本週訓練重點 (W23)
本週 (W23) 我們已正式開啟了為期數週的「進展期初期 (Strengthening Phase)」。距離雪梨馬拉松剩下不到三個月，我們必須在鞏固有氧底盤的同時，喚醒您的門檻速度，並強化下肢的剛性。以下是本週課表的五大核心剖析：

**一、 訓練重點：乳酸門檻刺激與基礎有氧擴張 (Threshold & Endurance)**
本週的訓練邏輯採取「雙軸並進」的策略。第一軸是透過週四的「T 門檻刺激跑」，訓練您的身體在乳酸即將大量堆積的邊緣（6:07/km）提升「耐受」與「清除」的能力。第二軸則是週末重頭戲「16km LSD 長距離慢跑」。這不僅是六月最重要的有氧引擎擴張練習，透過長達兩小時的持續跑動，將促進粒線體增生與毛細血管網發展，強化身體燃燒脂肪作為能量的效率，是全馬 sub 4:30 避免撞牆期的最關鍵訓練。

**二、 核心目標：重建下肢抗疲勞能力與核心穩定 (Leg Strength & Core Stability)**
雪梨馬拉松以其充滿挑戰的丘陵地形著稱，尤其是對肌肉離心收縮要求極高的下坡路段。因此，本週的核心目標之一是透過高質量的重量訓練（如肩部穩定與階梯運動），直接強化您的上肢穩定度與下肢推蹬能力。當賽事末段體能下滑時，強韌的軀幹與穩定的核心將是您維持標準跑姿、防止步幅縮減與避免受傷的最強防線。

**三、 關鍵課表：T 門檻測驗與長距離耐力考驗 (Key Workouts)**
*   **W23-6km T 門檻跑**：本週四的室內 T 配速（6'15"/km）表現優異，心率甚至仍處於較低區間，顯示心肺潛力極大。接下來的 T 區間課表，請勇敢嘗試穩定鎖定在 6:07/km 的目標配速。
*   **W23-16km LSD 長跑**：本週日的終極考驗，也是您完成 ONE TOKYO VR 21K 挑戰的決定性戰役。請嚴格鎖定 Z2 區間 (7:12-8:02/km)，將專注力放在呼吸節律與高步頻的維持。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能堅定地執行本週所有質量與有氧課表，我們將能看到顯著的生理適應。您的心肺系統將適應更高的需氧量，日後執行 Z2 輕鬆跑時的體感疲勞度將大幅下降。此外，肌力訓練與長跑的雙重刺激，將強化您的足弓、阿基里斯腱及各關節韌帶的彈性，進一步提升跑步經濟性。

**五、 執行建議：高步頻防線與夏日實戰補給演練 (Execution Advice)**
*   **嚴守高步頻防線**：請在跑步中時刻提醒自己維持 170-175 spm 以上的高步頻，以分散衝擊力並保護膝關節，避免步幅過大帶來的煞車效應。
*   **實戰補給演練**：16km LSD 將耗時約兩小時，請務必攜帶能量膠，嘗試每 45-60 分鐘補充一次，並定時攝取水分，訓練腸胃在跑動中的吸收適應力。
*   **環境熱適應**：面對夏季高溫，心率漂移是必然現象。請以「心率區間」為最高指導原則，若體感過於悶熱或心率飆升，請果斷放慢配速。夏訓的目的是生理刺激與耐熱適應，切勿死守配速數據而造成中暑。
