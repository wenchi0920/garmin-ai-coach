3.  **關鍵課表：10km Z2 基礎耐力 (04-04 Sat)**。
    這是本週最重要的「地基工程」。目標心率絕對不准超過 143 bpm。如果為了維持心率而配速掉到 8'00"/km，也請務必接受。這是在修理您的「冷卻系統」，沒有強大的冷卻系統，雪梨 Sub 4:30 將只是空中樓閣。這 10 公里將檢驗您在長時間運動下維持步頻與心率穩定的能力。

4.  **預期成效：降低「有氧脫鉤」現象**。
    透過嚴格控速，我們預期在長跑末段的心率漂移能控制在 5% 以內。這將提升脂肪利用率，為 4/18 的淡江大橋 10K 賽事儲備真正的「體力銀行」。若能達成此成效，代表您的有氧引擎已開始優化，能更有效率地使用能源。

5.  **執行建議**：
    *   **「步頻優先」戰術**：在跑步機上使用節拍器設定為 175 bpm，哪怕配速慢到像快走，也要維持輕快的步伐。想像腳下是滾動的球，而非沉重的鐵塊。這能強迫神經系統適應更高效的推進節奏。

    *   **動態補水訓練**：本週六長跑請模擬賽事，每 3km 飲用少量開水，進行「腸胃耐受度」演練。
    *   **心理建設**：忘記配速，專注呼吸與步頻。在馬拉松的訓練中，「慢得下來」比「衝得出去」更需要勇氣。告訴自己：今天的慢是為了明天的遠。
