**【專業恢復、防傷策略與旅遊過渡期前瞻建議】**
面對本週即將啟程的日本旅遊行程，這是一個絕佳的體能與心理的雙重放鬆期。為了確保前幾週辛苦累積的訓練成果不流失，並將本週的修復效果深度固化，為您制定以下專業恢復與防傷策略：
1. **髖部活動度解放與步頻革命的延續**：日本旅遊期間勢必會有大量的步行。請將這些每日的觀光徒步視為低強度的「主動恢復 (Active Recovery)」。每天回到飯店後，務必進行 10-15 分鐘的「蜥蜴式」與「鴿式」來舒緩緊繃的髖關節。這不僅能有效解除久走帶來的骨盆前傾與下背痛，同時也能維持關節靈活度，這是我們未來追求 175 spm 以上高步頻不可或缺的解剖學關鍵。
2. **足底與小腿筋膜的預防性保養**：長期的海外連續步行容易造成足底筋膜與小腿後側承受累積性壓力。請在睡前利用溫水泡腳，並適度使用按摩球或雙手深度揉捏腓腸肌與比目魚肌，確保筋膜與肌肉組織的彈性。腳趾受傷部位雖然已經痊癒，但下肢末端組織的防護與保養依然不能有任何懈怠，預防潛在的足底筋膜炎發生。
3. **營養攝取的黃金窗口與腸胃適應**：盡情享受日本的精緻美食！但在週末準備回歸 8km 質量跑與 15km 長跑的前一晚，請確保攝取充足的優質碳水化合物（如白飯、烏龍麵），為身體重新建立滿載的肝醣儲備。同時，減少酒精攝入以免干擾深層睡眠與隔日訓練品質。長距離訓練時，務必實踐「少量多次」的能量膠與水分補給，持續訓練腸胃吸收力。
4. **身心靈的全面重置與心態調整**：請將 W21 視為完美的過渡週。讓自己沉浸在異國風情中，不需要被嚴格的課表與配速數字綁架。旅途中的慢跑（如 5/20 的 5km Z2 有氧）只是為了維持肌肉記憶與享受街頭風景。徹底放鬆心靈，讓身體重置，為回國後更高強度的馬拉松專項訓練週期準備滿滿的動能。

本週 (W21) 恰逢您的日本旅遊，這是一個極佳的過渡期（Transition Phase）。我們的核心目標是「主動恢復與輕度維持」，讓身體在前四週基礎重建期累積的疲勞完全釋放，同時確保體能不流失。以下是本週的五大核心重點與執行方針：

**一、 訓練重點：旅遊動態恢復與週末質量銜接**
本週前半段（5/18-5/22）將完全以旅遊期間的動態恢復為主軸。我們不再盲目追求高跑量或高強度課表，而是專注於「維持肌肉基本張力與心肺活性」。這段期間海外的大量步行本身就是一種優質的低強度有氧刺激。訓練重點在於回國後的週末（5/23-5/24），我們將直接銜接 T 區間的質量跑與 15km 的有氧長跑。這需要您的身體在旅遊期間獲得充分的深度修復與營養補充，才能在週末展現出色的爆發力、肌耐力與心肺耐久度，完美銜接下一個訓練週期。

**二、 核心目標：主動修復身體能量，維持 Z2 有氧基礎**
本週的核心任務是嚴格保護好您的雙腿，並在適當時機進行輕量化的運動。我們設定的具體目標是完成旅遊中的 5km Z2 基礎跑，以及每天睡前的高品質修復瑜伽或飯店房內的徒手肌力。在身心放鬆的同時，我們依然要將「高步頻」的概念落實在每一次的跑步中。哪怕只是 5km 的異國街頭慢跑，也要有意識地維持 170+ spm 的步頻，想像腳步輕快彈起，這能確保技術層面的記憶不致中斷，並降低受傷風險。

**三、 關鍵課表：週末 T 區間質量跑與 15km 有氧長跑**
本週真正的考驗將在週末展開，這兩項課表是本月的重大里程碑：
1. **8km T 區質量跑 (5/23)**：這是一次針對乳酸門檻 (Lactate Threshold) 的精準刺激。課表包含 2k 熱身、4k 的 T 配速 (約 6:07/km) 與 2k 緩跑。其核心目的是喚醒快速收縮的肌纖維，提升身體在較高強度下清除乳酸的生化效率，為之後的 M 配速打下堅實基礎。
2. **15km 有氧長跑 (LSD) (5/24)**：這是本月距離最長的一次訓練，重點在於長時間維持穩定的 E 區配速 (7:12-8:02/km)。這不僅是對下肢肌耐力的終極淬鍊，更是對週末長跑腸胃適應與補給策略的實戰演練，確保您有足夠的耐力與能量儲備面對雪梨馬拉松的 30km 撞牆期考驗。

**四、 預期成效：深層疲勞歸零，體能滿載強勢回歸**
我們預期在經歷這趟日本旅途的身心雙重放鬆後，您在生理與心理上都能夠達到完美的「超量補償」狀態。透過每日規律的瑜伽拉伸與輕量肌力喚醒，您的自律神經系統將獲得徹底重置。週末的兩次高質量訓練，如果您能夠順利吃下並將心率精準控制在合理區間，那將證明您前期的有氧底盤已無比堅固，為接下來的強化週期注入一劑強心針。您將會感受到跑步時步伐的輕盈感，且心率的穩定度將達到前所未有的境界。

**五、 執行建議：傾聽身體真實反饋，不拘泥於配速數字**
在日本旅遊期間，所有的課表安排都具有極大的「彈性」。如果今天走了兩萬步，覺得腿部肌肉痠痛疲乏，請毫不猶豫地跳過當天的肌力或跑步課表，改為深度的瑜伽伸展。享受旅途是最高原則！但在週末的長跑中，請務必切換回「備戰模式」，提前準備好水袋與能量膠，確保 15km 長跑的補給無虞。訓練時請將專注力放在您的步頻與呼吸節奏上，將注意力拉回身體內部。心率是我們唯一的導航指標，請確保長距離維持在穩定的 Z2 區間。
