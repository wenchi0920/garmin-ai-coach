# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個整合運動生理學與大數據分析的頂尖科學化馬拉松訓練系統。本系統以「數據導航，無痛增長」為核心理念，每日自動讀取並解析跑者的 Garmin 穿戴裝置數據。透過對心率、步頻、配速、HRV（心率變異度）與身體能量 (BB) 等關鍵生理指標的長期追蹤，動態評估跑力值 (VDOT)，精準校準個人化的訓練區間（包括輕鬆跑 E、馬拉松配速 M、乳酸門檻 T 與間歇跑 I）。本系統在給予生理系統「適度超載刺激」與確保「超補償恢復」之間取得黃金平衡，協助跑者在不累積傷痛的前提下，穩步提升有氧耐力與跑步經濟性，系統化地實現賽事目標，達成個人體能天花板的突破。本系統更著重於防傷機制的動態調整，當生理指標出現疲勞警訊時，會即時啟動避險機制，引導跑者在安全且科學的框架下前行。

## 📅 4. 本週訓練重點 (W29)

本週（2026-07-13 ~ 2026-07-19）為「進展期後段 (W28-W30)」的第二週。本週的訓練邏輯在於**「穩步擴大有氧容量、拉長乳酸門檻耐受時間，並以高步頻防禦機制應對里程突破」**。本週預計總跑量將穩步爬升至 **39 公里**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：LSD 里程突破與乳酸門檻持續時間拉長**
    本週的長跑日（週六）將迎來關鍵的突破，LSD 里程由上週的 14km 提升至 **16km**。這 2 公里的增量，是對下肢肌肉耐力與心肺系統的一次重要超載刺激，旨在逐步適應雪梨馬拉松後半段的耐力需求。此外，週二質量跑的 T 配速（乳酸門檻跑）總長度將從 6km 提高至 **7km**（維持 6'07"/km），總里程達 11km。由於週二執行失控中斷，本週週四（7/16）基礎跑與週六 LSD 將作為核心，暫不對週二的門檻跑進行補課，避免疲勞二次疊加。後續的訓練重點在於拉長處於乳酸門檻下的持續時間，以提高身體清除乳酸的效率。
*   **二、 核心目標：步頻 170-175 spm 的穩態維持與補給窗口演練**
    1.  **步頻穩態控制**：全體跑步課表均搭配節拍器或手錶步頻警示，強制將雙邊步頻穩定在 **170-175 spm** 之間。透過「高步頻、小步幅」的跑姿，縮短觸地時間，控制垂直振幅在 75mm 以下，從而最大程度降低每次落地的煞車效應與關節衝擊力。
    2.  **配速與心率區間守恆**：週四有氧跑及週六 LSD 必須嚴格將心率壓制在 Zone 2 (153-166 bpm) 的有氧區間內。尤其週六 LSD 前 3 公里必須壓抑速度，以 **7'45" - 8'00"/km** 慢速起跑，防止心率過早漂移。
    3.  **腸胃補給窗口演練**：利用週六的 16km 長跑，進行雪梨馬拉松的補給模擬。分別於 **第 6 公里與第 12 公里** 定時補給一包能量果膠，每次補給均配合 150ml 的水分，演練腸胃在高運動強度下的吸收能力，防止賽事當天發生腸胃不適。
    4.  **力量打底防傷**：週四安排下肢單側重訓，以單腳硬舉與保加利亞分腿蹲為核心，強化臀大肌與大腿後側肌群，增強關節穩定度。
*   **三、 關鍵課表：7km Z2 基礎跑 + 下肢重訓與 16km LSD 戶外實戰**
    1.  **週四 7km Z2 基礎跑 + 下肢重訓**：1km Z1 暖身 + 5km Z2 基礎有氧 (HR: 153-166, 步頻目標 170+) + 1km Z1 緩跑。跑後進行下半身力量訓練（深蹲、保加利亞分腿蹲），每次重訓前需進行 10-15 分鐘的核心啟動（死蟲式、鳥狗式）。
    2.  **週六 16km LSD 長距離慢跑**：**強制戶外 GPS 跑**。1km Z1 暖身 + 14km Z2 有氧慢跑 (配速: 7'12"-8'02"/km) + 1km Z1 緩跑。本課表核心在於「穩」，中後段以 170+ spm 步頻防守，嚴格執行 6k / 12k 的補給紀律。
    3.  **週五/週日 完全休息與活動度保養**：不跑步，分別進行 20 分鐘與 15 分鐘的睡前瑜伽，針對髖部（鴿子式/蜥蜴式）與小腿後側進行深度伸展，確保肌肉與神經完全放鬆。
*   **四、 預期成效：提升跑步經濟性與延緩後半程心率漂移**
    透過將步頻穩定在 170-175 spm，學員的平均觸地時間預期能穩定控制在 **270ms 以下**，垂直振幅降至 **75mm 以下**。這將大幅降低股四頭肌的離心收縮負擔，減少肌肉微細撕裂。隨著 7km T 門檻跑的刺激，學員的有氧/無氧臨界點將得到拓寬，乳酸門檻配速下的心肺負擔會逐步減輕。在週六 16km LSD 中，若能落實前 3 公里壓抑配速與定時補給，將能有效延緩後半程的心率漂移，使心率曲線平緩，並確保在 12km 後腿部肌肉仍有充足的力量支持，不至於因肌肉崩潰而中途停步。
*   **五、 執行建議：節拍器輔助跑、熱應激防護與避險退場機制**
    1.  **節拍器輔助**：跑步時請播放 172 bpm 的背景音樂或開啟手錶的步頻節拍器，利用肌肉聽覺回饋強制跟上步頻，專注於「小步幅、輕落地、骨盆送髖」的感覺。
    2.  **熱應激與防曬防護**：7 月中旬高溫濕熱是心率漂移的最大干擾源。所有戶外跑步（特別是週六 LSD）務必安排在**清晨 05:30 - 07:30** 或**傍晚 18:00 以後**進行，隨身攜帶水袋或選擇有補水點的路線，嚴格防範中暑與嚴重脫水。
    3.  **避險與退場機制**：由於近期出現疲勞與違規加練，本週執行課表時，若在任何跑步中發現心率在配速未變的情況下飆升至 171 bpm 以上 (Zone 4)，或感覺下肢關節有針刺般的銳痛，應立即由跑步切換為快走，甚至中止訓練。科學訓練的第一信條是無傷，不強練疲勞課表。
