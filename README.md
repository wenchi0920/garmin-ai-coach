# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位
Garmin AI Coach 是一個結合運動生理學與大數據運算的核心教練系統。本系統秉持「科學訓練，精確校準」之精神，每天自動讀取並解析跑者的 Garmin `.fit` 運動紀錄檔案。透過對心率、步頻、配速、HRV 與壓力指數等關鍵生理指標的長期監控與深度分析，動態推算跑力值 (VDOT) 並修正個人化心率與配速訓練區間，自動產生並上傳每週科學化訓練課表。教練的終極目標是為跑者提供精準的數據導航，在「給予身體超載刺激」與「安全超補償修復」之間尋找黃金平衡，從而系統化、無痛地提升有氧天花板，穩健實現賽事目標，達成身心健康的全面成長。

在 W27 週期之中，學員進行了高密度的訓練，但也暴露出疲勞累積與恢復排程上的問題，以下是關鍵生理數據與訓練負荷的深度分析：
*   **07/01 (週三) 的「一日三練」與超載生理負荷：**
    學員在 7/1 執行了極具挑戰性但超出身體負荷的「一日三練」。中午 12:34 進行了 44 分鐘的肌力訓練（7 組深蹲及 8 組肩部穩定動作，平均心率 127 bpm，最大心率 152 bpm），旨在為雪梨馬起伏路段儲備力量。然而，在中午重訓對神經肌肉系統造成局部疲勞的情況下，學員於當晚 21:17 再次進行了 9.03 公里的跑步機 T 門檻跑，5 公里質量段平均配速達 5'47.6"/km，快於原設定 5'58"/km 的 T 配速目標，最大心率飆升至 168 bpm，雙腳步頻提升至 172-178 spm。隨後於 23:28 進行了 21 分鐘的深夜修復瑜伽。

    今日（7/02）晨間的生理指標進一步顯示疲勞累積的負面效應：**靜止心率攀升至 57 bpm**，**身體能量 (BB) 更是下滑至 34** 的警戒水位，壓力指數維持在 19。週四在 W27 課表中原訂為**完全休息日**，目的是讓身體進行生理超補償與自主神經修復。然而，學員卻於中午 12:32 強行進行了高強度的重訓（負重深蹲 4 組最高 45kg，槓鈴硬舉 4 組最高 35kg，核心/抬腿 6 組最高 25kg，耗時 45 分鐘）。這不僅中斷了前一日門檻跑的超補償恢復，更大幅增加了肌肉微創傷與受傷的機率。

    經過週四下午與夜間的休整，今日（7/03）清晨生理指標出現溫和改善：**身體能量 (BB) 回升至 48**，睡眠分數為 71（07/02 晚間睡眠），壓力指數下降至 **15**。這表明休息發揮了作用，自主神經系統正在逐步恢復。然而，**靜止心率 (RHR) 仍居高不下維持在 57 bpm**，這是一個關鍵信號，代表心血管系統與神經系統依然處於累積疲勞狀態，心臟在靜息狀態下仍需代償性地加快搏動以協助修復。

結合 [Workouts-2026-W27.md](logs/Workouts/2026/Workouts-2026-W27.md) 的「上週回顧」，我們對 W26（06/22 ~ 06/28）進行了結算。上週學員順利完成了 **36.37 km** 的跑步里程，總跑步時間為 **04:11:26**。若加上下半身與核心重量訓練（74 分鐘）及睡前修復瑜伽（133 分鐘），總運動時數達到了 **07:01:05**，展現了極高的課表紀律。

3.  **長跑因疲勞未達標**：週日原本規劃 18km LSD，但因連日累積疲勞，週日晨間 BB 僅 37，睡眠分數僅 48 分。最終僅完成了 10.01km，且前半程起跑過快（6'34"-6'35"/km）導致心率早衰，最後一公里配速掉至 8'34"/km，步頻崩潰至 142 spm。

### 【專業恢復建議、疲勞評估與防傷指引】
1.  **取消今日 (07/03) 重訓，落實主動恢復**：
    鑒於今日 RHR 仍高達 57 bpm 且 BB 僅 48，原定今日執行的「下半身肌力」必須**強制取消**。今日的 6km 輕鬆跑定位為「純主動恢復跑」，配速必須嚴格控制在 **7'30" - 8'00"/km** 之間，心率壓在 Z1-Z2 區間。跑步時嚴禁追求速度，以讓肌肉排酸、活化血管網為唯一目的。
2.  **建立 175+ spm 步頻防線**：
    學員左腳大拇指指甲撕裂傷雖已 100% 康復，但 W26 出現的 160 spm 超低步頻，是潛在的致傷隱憂。7/1 的門檻跑中步頻能提升至 172-178 spm，證明學員有能力做到。教練要求在接下來的輕鬆跑與 LSD 中，必須強制將步頻維持在 **175 spm 以上**，透過「縮小步幅、輕快收腿」的力學機制，降低關節受力。
3.  **優化睡眠與瑜伽排程**：
    學員近期瑜伽常於深夜（如 7/1 的 23:28）進行，會使核心體溫升高，推遲副交感神經的啟動。教練建議將每日修復瑜伽提前至 **22:00 前** 完成，確保 **23:00 前** 入睡，以利 BB 指數與 HRV 在週末 LSD 前順利回升。
4.  **週日 18km LSD 控速與補給實戰演練**：
    週日 18km 戶外 LSD 是本週的核心收官課表。學員必須嚴格遵守：
    *   **防禦性配速**：前 3 公里強制設定在 7'50" - 8'10"/km 熱身，後續配速嚴禁跑進 7'12"/km 以內，心率壓在 Z2 (131-143 bpm)。
    *   **腸胃補給**：攜帶至少 3 包能量膠，每 45 分鐘固定補給，小口補水，演練雪梨馬拉松實戰補給。

## 📅 4. 本週訓練重點 (W27)
本週為「進展期中段 (W25-W27)」的最後一週 (W27)。由於上週長跑出現里程赤字，且近期步頻普遍偏低、本週三出現一日三練與週四重訓違規後的疲勞超載，本週的關鍵任務是**「穩定刺激乳酸門檻、落實高步頻防禦機制、以及在起伏地形上完成長距離耐力與補給演練」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：鞏固乳酸門檻耐力與雪梨賽道地形適應**
    備戰全馬的黃金法則在於有氧耐力與乳酸門檻（Lactate Threshold）的雙重刺激。本週作為進展期中段的收官週，訓練重點是將門檻跑的刺激時間進一步拉長，週三（已執行）的 T 跑（乳酸門檻跑）質量段由上週的 4km 提升至 **5km**，這將穩定刺激學員的 T 區間，提升身體在高酸性狀態下的心肺代償與肌肉收縮效率。同時，本週日的 18km LSD 將正式移至有坡度起伏的戶外道路，模擬雪梨馬拉松的丘陵地形，為下肢肌肉注入耐衝擊的力量地基，並通過起伏路段來訓練股四頭肌的離心控制能力，以利在雪梨丘陵賽道上維持配速。

*   **二、 核心目標：T區配速穩定化與步頻突破 175 spm**
    1.  **乳酸門檻配速穩定 (T 配速 5'58"/km)**：在週三的門檻跑中，在 5'58"/km 配速下，將心率穩定控制在 Zone 4 (155-167 bpm)，擴大乳酸清除效率，避免心率提前漂移。週三實際執行均速 5'47"/km 偏快，雖然展示了極佳的心肺能力，但下肢負荷過重，後續需依課表配速執行。
    2.  **步頻防線建立 (175+ spm)**：徹底糾正 160-162 spm 的低步頻、大步幅跑姿。學員必須將「輕快著地、快速離地」作為最高指令，在所有輕鬆跑與 LSD 中將步頻強推至 175 spm 以上，藉此減少垂直振幅與每次著地的煞車效應。
    3.  **線上挑戰完賽**：於 7/5 前完成 ONE TOKYO GLOBAL Virtual Race 3rd 線上賽剩餘的 **13.88km** 里程，並藉由週末跑與長跑累積完成。

*   **三、 關鍵課表：9km T 門檻跑與模擬雪梨馬 18km 戶外 LSD**
    1.  **週三 9km T 門檻跑 (已於 7/1 執行)**：包含 2km Z1-Z2 暖身 + **5km T 區間主課表 (均速 5'47.6"/km, 心率 Zone 2-3)** + 2km Z1 緩跑。這將是本週強度最高、對心肺刺激最深的質量課表。雖然超前完成，但未來應嚴格跑重分流，防止一日多練的累積性損傷。
    2.  **週五 6km 與週六 8km Z2 輕鬆跑**：作為週末長跑前的有氧鋪墊。配速必須嚴格鎖定在 **7:01 - 7'51"/km** 之間，心率壓在 Z2 區間，全力為週日長跑留存能量。週五重訓取消。
    3.  **週日 18km 戶外 LSD**：包含 1km Z1 暖身 + 16km Z2/Z3 有氧慢跑 (配速: 7:01-7:51) + 1km Z1 緩跑。必須選擇有起伏的戶外地形，並隨身攜帶 3 包能量果膠與水，每 45 分鐘固定補給，演練腸胃吸收。

*   **四、 預期成效：提升跑步經濟性 (RE) 與控制後半程心率漂移**
    當步頻提升至 175+ spm時，將顯著降低跑步時的垂直振幅與關節受力。在 5km T 跑與 18km LSD 後半程，心率漂移現象將獲得緩解。隨著有氧引擎與下肢力量的轉化，學員在維持馬拉松配速跑動時，心肺呼吸將更為從容，肌肉能更效率地利用糖原，這將是 Sub 4:30 備賽最穩固的基石。

*   **五、 建議與警告：坡度設定、戶外實戰補給與提前黃金睡眠**
    1.  **跑步機坡度設定**：若因天候在跑步機上進行輕鬆跑，必須設定 1.0% - 1.5% 坡度，補足無風阻生理差額，使其更貼近戶外路感。
    2.  **防禦性配速控制**：週日 LSD 出發前 3 公里強制設定在 7'50" - 8'10"/km 的慢速熱身，避免起跑過快導致心率過早拉升。
    3.  **提前睡前伸展**：睡前瑜伽拉伸課表提前至 22:00 前完成，避免深夜高強度活動干擾睡眠，爭取每日 7.5 小時以上的睡眠，以利 HRV 與 BB 回升。
