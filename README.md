## 1. 系統核心定位：數據驅動的動態校準與精準訓練
Garmin AI Coach 是一個深度結合 Garmin 穿戴裝置數據（涵蓋心率、配速、步頻、壓力、睡眠與 HRV 等）的智能訓練分析系統。我們秉持「科學訓練，精確校準」的核心理念，透過自動抓取並深度解析您的 `.fit` 訓練檔案，動態追蹤並評估您的跑力（VDOT）、疲勞累積以及生理恢復狀態。系統不僅將產出每週客製化的週期性課表排程，更會依照每日健康數據給予即時性的回饋與戰略微調。我們的目標，不僅是確保您在最小化受傷風險的前提下穩步推升競技體能，更是期望陪伴您在追求馬拉松極限的旅程中，實現身、心、靈的全面成長與突破。

**【專業恢復、防傷策略與減量週前瞻建議】**
面對下週 (W21) 即將到來的日本旅遊行程與訓練減量週，為了確保體能不流失並將本週的成果深度固化，為您制定以下專業恢復與防傷策略：

2. **足底筋膜與阿基里斯腱的預防性保養**：傷勢雖已痊癒，但隨著基礎期跑量的攀升，下肢末端筋膜承受的累積壓力將以幾何倍數增長。請務必於睡前使用按摩球輕踩足底，並針對小腿後側（腓腸肌與比目魚肌）進行深層滾筒放鬆，保持組織的絕佳彈性以減少代償壓力，預防潛在的足底筋膜炎發生。
3. **營養攝取的黃金窗口與腸胃適應訓練**：在進行長距離訓練或 T 區間間歇時，肝醣消耗劇烈。請在訓練後 30 分鐘的「黃金窗口」內，以碳水化合物與蛋白質 3:1 的黃金比例進行補充，加速肌肉微損傷修復。同時，週末長跑必須實踐「少量多次」的水分與能量膠補給，訓練腸胃在運動中的吸收能力。

## 📅 4. 本週訓練重點 (W20 基礎期重建結算)

本週 (W20) 是本次「基礎期重建 (Base Phase Rebuilding)」的第四週，同時也是這一個關鍵週期的總結與收尾階段。在下週即將啟程前往日本並迎來期待已久的減量週之際，本週最神聖的任務就是將前三週累積的訓練紅利深度固化，並將技術瑕疵徹底修正。以下是本週的五大核心重點與執行方針：

**一、 訓練重點：基礎期完美收尾與動作經濟性的深度內化**
本週的訓練主軸不再是盲目地增加跑量，而在於「技術細節的嚴格檢核」與「有氧底盤的終極鞏固」。經過將近一個月的打磨，您的有氧基礎已然強韌，心率在 Z2 區間的穩定表現堪稱完美。現在，我們必須將專注力全面轉移到「跑步經濟性 (Running Economy)」的提升上。我們需要強制將高步頻與低觸地時間的技巧，深深地刻印在您的肌肉神經記憶中，使其成為您面對長期疲勞時的本能反應。這不僅是為了跑得更快，更是為了跑得更長久、更無痛。將技術動作內化，是這週最重要的心理與生理功課。

**二、 核心目標：確立堅不可摧的有氧底盤與高步頻技術革命**
所有跑步課表首要且絕對不可妥協的技術目標，就是將「步頻」穩定拉升並維持在 **175-185 spm** 的高效區間內。根據近期的訓練數據檢測，您的平均步頻仍多在 158-166 spm 之間徘徊，這意味著步幅偏大，潛藏著嚴重的過度跨步 (Overstriding) 風險，容易對膝關節造成「剎車效應」。請主動且刻意地縮小步幅，想像地表發燙，迫使腳掌在觸地的千分之一秒內迅速彈起，大幅縮短觸地時間。這項技術革命將從根本上降低下肢關節受傷的風險，同時大幅提升向前的推進效率。請務必在手錶畫面上將「步頻」設為首要關注指標，時刻提醒自己維持輕快的節奏。

**三、 關鍵課表：12km 週末戶外長跑 (LSD) 及 T-Pace 間歇**
為確保能量系統得到全面刺激，本週安排了兩項截然不同但相輔相成的關鍵課表：
1. **4km T-Pace 間歇 (乳酸門檻訓練)**：此課表旨在精準測試並提升您在乳酸門檻附近的代謝清除能力。透過適度的無氧刺激，鍛鍊心肺系統的韌性，讓身體學會在高配速下如何有效處理代謝廢物。請確實做好前後的動態熱身與緩和跑，確保在安全的前提下完成這項高強度刺激。
2. **12km 長距離慢跑 (LSD)**：這是對下肢肌肉耐力與有氧代謝系統的深度淬鍊。由於虛擬賽事的規範要求，我們強烈要求此課表必須在「戶外」執行，以模擬真實路面的反饋、坡度變化與風阻。請嚴格遵守「負分割 (Negative Split)」配速原則，前半程極力保留體力並控制心率，後半程再逐漸穩定輸出，這更是補給策略與心理韌性的最佳彩排。

**四、 預期成效：全面擴張有氧區間，實現代謝效率的根本質變**
若您能確實貫徹高步頻技術與嚴格的 Z2 心率控制，您將會在短時間內明顯感受到能量代謝效率的躍升。在相同的 Z2 輕鬆跑心率下，您的體感配速將會自然變得更加輕快且流暢；而在面對高強度的 T 區間訓練時，組間休息的心率下降速度也會變得更為迅速。這種生理指標的正向反饋，代表著心臟每一次泵血的效能正在提升，且肌肉組織間的微血管網正在快速擴張。這正是為雪梨馬拉松打造堅不可摧有氧防護罩的最佳證明，讓您在未來的 30km 長距離訓練中，擁有更充沛的體力儲備與強悍的抗疲勞能力。

**五、 執行建議：心率絕對優先、嚴格戶外適應與心靈深度放鬆**
在執行 Z2 輕鬆跑時，請務必秉持「心率絕對優先，配速僅供參考」的鐵血守則。絕對不要被手錶上的配速數字綁架，若當天受氣溫或身體狀況影響，主動降速以維持低心率才是智者之舉。此外，由於目前正積極參與線上虛擬賽（TATTA RUN 與 ONE TOKYO），所有累積里程皆需 GPS 軌跡，請務必將長跑與基礎跑轉移至戶外進行。最後，請持續並深化睡前主動修復瑜伽的習慣，透過深層的動態伸展與呼吸冥想，讓身心在每日訓練後都能達到最佳的重置狀態。保持耐心與極致的專注，我們正穩健地走在達成目標的軌道上！
