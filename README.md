3.  **關鍵課表：10K 體感測試 (週四) 與 12km LSD (週六)**

4.  **預期成效：跑步經濟性 (RE) 的質變**

5.  **執行建議 (詳盡導引)**
    *   **「節拍器」是本週的主角**：所有的跑步訓練，請強制開啟節拍器 175 bpm。這不僅是為了保護腳趾，更是為了優化您的跑步效能。
    *   **力量訓練的針對性**：週三的肌力訓練請專注於「核心穩定」與「單腳平衡」。強壯的核心能確保您在週六長跑的後半程，骨盆不會因疲勞而產生過度晃動，這對 10K 賽事的後半段衝刺至關重要。
    *   **環境模擬**：建議本週至少安排一次傍晚跑步，模擬 4/18 比賽當日的氣溫與光線。跑步機雖然穩定，但無法取代真實路面的衝擊力。
