# 🏃‍♂️ Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位
Garmin AI Coach 是一個結合尖端運動科學與大數據分析的專屬自動化課表系統，核心理念在於「科學訓練，精確校準」。系統透過每日自動解析 Garmin 裝置生成的 `.fit` 檔案，深度追蹤您的心率變異度、跑步動態與疲勞負荷。結合您的專屬跑力指標 (VDOT) 與個人化生理回饋，AI Coach 每週會動態生成包含長距離慢跑 (LSD)、基礎有氧 (Z2)、乳酸門檻 (T) 以及跑後修復與專項肌力的全面性訓練計畫。我們不僅專注於突破成績，更在乎降低受傷風險與身心靈的平衡，讓跑者只需百分之百專注於訓練的執行，穩健邁向每一次的起點與終點，協助您達成設定的卓越目標。

## 📅 4. 本週訓練重點 (W24)
本週 (W24) 我們正式進入備戰雪梨馬拉松的「進展期初期 (Strengthening Phase)」。由於上週在 LSD 的執行上出現了較大的斷層，且本週二的 T 跑未能足額完成，本週的核心要務是「確保後續長距離課表的完整與連續執行」，在鞏固有氧底盤的同時，開始強化乳酸門檻的耐受力。這是一個從純有氧基礎邁向專項耐力轉換的關鍵過渡週。以下是本週課表的五大核心剖析：

**一、 訓練重點：彌補耐力缺口與鞏固門檻跑穩定性 (Endurance & Threshold Focus)**
本週的訓練邏輯採取了「雙軸並進」的穩健強化策略，旨在修復近期流失的訓練效益。第一軸著重於「T 門檻刺激」，雖然週二執行不如預期，但我們必須在後續的訓練中補足。T 跑的目的在於訓練您的身體於乳酸即將大量堆積的臨界點（根據您 VDOT 33 的能力，約為 6:07/km）提升「耐受度」與「清除效率」。這能實質拉高您在無氧邊緣的續航力。第二軸則是週末的重頭戲「16km LSD 長距離慢跑」。透過超過 90 分鐘至 2 小時的持續有氧跑動，強迫身體動用脂肪作為主要能量來源，並促進微血管網的擴張與肌肉細胞內粒線體的增生。這是全馬要達成 Sub 4:30 且避免撞牆期最不可或缺的生理基石。

**二、 核心目標：模擬雪梨馬微起伏地形與下肢抗疲勞強化 (Terrain Simulation & Leg Strength)**
雪梨馬拉松的賽道以其充滿挑戰的起伏丘陵地形、跨越雪梨港灣大橋的爬坡，以及多變的海風著稱，這頻繁的上下坡對腿部肌肉（特別是股四頭肌與小腿）的離心收縮要求極高。因此，本週的核心目標之一，是透過週末的長跑刻意尋找具有微幅起伏的戶外路線（如河濱緩坡或公園丘陵），提早讓下肢適應坡度變化所帶來的不穩定負荷。同時，週二您已完美執行的「下肢基礎肌力」與「跑者核心穩定」課程（包含深蹲、硬舉與棒式），已經直接強化了您的股四頭肌、臀中肌與軀幹的抗疲勞度。當比賽進入 30 公里後的撞牆期，強韌的下肢剛性與穩固的核心，將是您維持標準跑姿、防止抽筋受傷的最強防線。

**三、 關鍵課表：彈性調整後的質量與長跑考驗 (Key Workouts)**
*   **W24-8km Z2 基礎有氧 (週四)**：以穩定擴張有氧引擎為唯一目標，請將心率嚴格控制在 153-166 bpm 之間，配速維持在 7:12-8:02/km。為了彌補週二 T 跑的不足，請在最後 2-3km 嘗試提升至 M 配速 (6:33/km) 以增加刺激。
*   **W24-6km Z2 輕鬆跑 (週六)**：輕量有氧，保持雙腿靈活，為週日重頭戲儲備體力。
*   **W24-16km LSD 長跑 (週日)**：這是本週的終極考驗，也是重建月跑量進度的戰役。請嚴格鎖定在 Z2 區間，將專注力放在呼吸節律的順暢度。務必提早於清晨 6 點前起跑，避開高溫，確保能「絕對不中斷」地完成目標里程，重新建立對長距離的生理適應與心理韌性。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能貫徹執行本週剩餘的質量與有氧課表，我們預期在週末能觀察到顯著的綜合生理適應。您的心肺系統將更能適應較高的需氧量，未來在執行 Z2 輕鬆跑時，同等配速下的心率將會下降，體感疲勞度也會大幅減輕。此外，藉由週二優異的專項肌力訓練與 16km 長跑的雙重肌肉破壞與重建機制，您足弓、阿基里斯腱及膝關節周圍韌帶的彈性與剛性將被顯著強化，進一步提升每一步的跑步經濟性。這種全身性的綜合強化，將讓您在面對雪梨馬拉松的起伏賽道時，保持游刃滑裕的體能儲備。

**五、 執行建議：高步頻防線、夏日實戰補給演練與雙練平衡 (Execution Advice)**
*   **嚴守高步頻防線**：請時刻提醒自己維持 175 spm 以上的高步頻，目前 81 spm 偏低。高步頻能有效縮短觸地時間，分散衝擊力。
*   **夏日環境熱適應與時段優化**：夏季氣溫高，心率向上漂移是正常現象。請務必以「心率區間」為最高指導原則。週末 LSD 務必移至清晨涼爽時段執行。
*   **重訓與跑步平衡**：一日雙練時，確保補給充足，且將高強度跑步安排在重訓之前，或錯開至少 24 小時。
*   **實戰腸胃補給演練**：16km LSD 期間，請嘗試在第 7km 與第 12km 進行水分與能量膠補充，訓練腸胃在跑動中的吸收能力。跑後請確實執行修復瑜伽，加速排酸。
