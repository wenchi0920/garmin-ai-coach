**1. 訓練重點：神經張力維持與精準減量 (Sharpness & Tapering)**

**2. 核心目標：傷後首戰——淡江大橋 10K 效能驗證**
*   **目標配速**：5:50 - 6:15/km (目標完賽時間 58-60 分鐘)。
*   **技術指標**：全程步頻需維持在 178 bpm 以上，觀察在戶外風阻環境下，心率與跑步經濟性的關聯。

**3. 關鍵課表：賽前喚醒的三重奏**
*   **04/14 下肢肌力維持**：40kg 深蹲/硬舉，最後一次肌肉募集刺激。
*   **04/15 5.37km 喚醒跑**：包含 Z5 短衝刺，確認 VDOT 34 的上限空間。
*   **04/18 淡江大橋 10K**：本週的靈魂課表。這不僅是一場比賽，更是雪梨馬拉松 20 週週期的正式起點。

**4. 預期成效：VDOT 的實戰校準**
*   **數據意義**：若能以 6'00" 左右配速無痛完賽，代表您的 VDOT 將從 34 實質往 36 邁進。
*   **恢復驗證**：賽後 24 小時內的 RHR 與 HRV 波動，將作為我們規劃下週 W17「有氧重建期」強度的重要依據。

**5. 執行建議：教練的賽前精準叮嚀**

*   **裝備定案**：請務必穿著近期跑步機訓練最習慣的鞋襪組合（推薦已磨合好的輕量訓練鞋）。
*   **補給節奏**：10K 賽事不需額外能量膠，但賽前 2 小時的早餐應以易消化的碳水（如白吐司、香蕉）為主。
*   **心理暗示**：當比賽末段感到呼吸急促時，請專注於「擺臂」與「核心穩定」，回想 04/14 重訓時核心繃緊的體感，那將是您對抗疲勞的最強後盾。
