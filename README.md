## 1. 系統核心定位
Garmin AI Coach 是一個深度結合運動科學與大數據分析的專屬自動化訓練生態系統。我們不僅提供靜態計畫，更透過每日自動解析 Garmin 裝置生成的 `.fit` 檔案，即時監控跑者的心率變異度 (HRV)、睡眠品質、身體能量 (Body Battery) 以及各項訓練動態指標。系統的核心定位在於「**科學訓練，精確校準**」，透過跑力指標 (VDOT) 的動態修正與個人化數據分析，將繁瑣的生理資訊轉化為具體執行策略。無論是面對高溫挑戰、傷後復健或調整日常疲勞，AI Coach 都能彈性調整課表強度，協助跑者在追求賽事目標的過程中，有效管理疲勞、降低受傷風險，確保達成目標的同時，也兼顧身體修復與心靈成長。

## 📅 4. 本週訓練重點 (W24)
本週處於雪梨馬拉松備賽的「進展期初期」，這是決定耐力天花板與完賽信心的關鍵階段。

**一、訓練重點：修補耐力斷層與重新聚焦 (Endurance Rebuilding)**
由於前兩週高溫與外務導致里程缺口，本週首要任務是「穩定補償有氧里程」。我們不追求極速，而是要求每公里的輕鬆跑與長跑都能精準落在心率區間內，重新喚醒脂肪供能效率，找回長距離的耐心。請將專注力放在完成預定時間與距離，而非急於提升配速。

**二、核心目標：力量與耐力的平穩轉化 (Strength-to-Speed Conversion)**
您近期在深蹲與硬舉上取得的肌力進步，需轉化為實質跑動力。本週目標是利用這些「靜態結構力量」來維持高步頻與骨盆穩定，確保在週末 LSD 中克服疲勞與起伏地形，為 Sub 4:30 建立穩健基礎。肌力與有氧的結合將是本週與下週訓練銜接的核心。

**三、關鍵課表：6km Z2 輕鬆跑與 16km LSD 長跑 (Key Sessions)**
本週成效將由週末這兩項課表決定，請務必嚴格遵守配速紀律：
*   **06/13 (週六) 6km Z2 輕鬆跑**：作為輕量有氧刺激，請將心率嚴格控制在 Z1-Z2，配速 7:12-8:02/km，為隔日保留糖原，讓雙腿維持活絡但不疲勞。
*   **06/14 (週日) 16km LSD 長跑**：本週核心期末考。目標為「不間斷完成」，維持 Z2 區間高標心率，是一場體能與心理韌性的雙重建設，若遇高溫可分早晚兩次完成，重點在於總量積累。

**四、預期成效：有氧效率提升與恢復力強化 (Expected Gains)**
順利完成本週連日課表，預期能大幅改善心肺飄移現象 (Cardiac Drift)，提升有氧效率。同時，基礎代謝與微血管密度的增加，能讓您在未來面對高強度 T 跑時恢復得更快。穩定累積里程將能顯著提升對長距離的適應力，降低 30 公里後的撞牆期風險。

**五、執行建議：高溫應對與補給實戰 (Execution Strategy)**
*   **熱適應與時段**：請避開烈日，安排在清晨 5:30 前或傍晚起跑，以「心率」為最高指導原則，心率飆升請果斷降速。
*   **實戰補給演練**：週日 16km LSD 請於第 7km 與第 12km 實地測試能量膠與水分攝取，讓腸胃適應負載。
*   **步頻即時監控**：在疲勞期（12km 後）有意識地縮短步幅、提升步頻至 175 spm 以上，維持核心收緊。在跑步機上更應主動調高律動以減少觸地時間。
