## 📅 本週訓練重點：W19 步頻革命與攝氧開發

**1. 訓練重點：步頻革命 (Cadence Revolution)**
本週是 5 月重建期的核心技術週。我們不追求配速的提升，而是要進行一場「步頻革命」。重點在於將手錶節拍器固定在 175-180 bpm，全程校準。這不是為了跑快，而是為了透過高頻率減少單步落地衝擊，提升跑步經濟性。對於 VDOT 33 的跑者，將步頻從 152 提升到 175，將能直接減少約 15% 的地面衝擊力，這對於達成雪梨馬 Sub 4:30 而言，是比體能進步更重要的技術支柱。

**2. 核心目標：技術校準與間歇耐受**
*   **技術指標**：在所有跑動課表中，確保平均步頻突破 170 spm。即便配速掉到 9 分速，步頻也不能掉。這是為了重新建立大腦對頻率的認知。
*   **生理指標**：週三進行 5x1km 的 I 區間歇，目的是開發攝氧天花板，提升 VDOT 潛力。I 區間訓練能強迫身體在高強度下維持技術動作，是測試「步頻革命」成效的壓力環境。
*   **耐力內化**：週六 TATTA RUN 15km，這將是 5 月份首場長距離挑戰，重點在於維持長時間的高步頻穩定度，模擬全馬中後段的機械運作。

**3. 關鍵課表解析：**
*   **05/06 (三) 8km I 間歇 (5:38)**：這是本週唯一的質量日。透過 I 區間訓練提升最大攝氧量 (VO2Max)。請在此課表中專注於「高步頻下的動作完整度」，而非死命推蹬。間歇跑是技術的放大鏡，若能在此強度下維持 175 步頻，E 跑將變得輕而易舉。
*   **05/09 (六) 15km LSD (TATTA RUN)**：5 月份最重要的長跑練習。維持 Z2 心率 (153-166 bpm)，模擬賽事後半段的心率穩定度，並測試補給策略。這 15 公里是步頻革命的「期中考」，必須忍受縮小步幅帶來的體感不適，堅持高轉速。

**4. 預期成效：**
完成本週課表後，您的神經肌肉系統將開始適應較高的步頻節奏。這會讓您在未來的輕鬆跑中，感覺體感更輕盈、觸地更短暫。同時，週三的間歇跑將為下週的 VDOT 重新評估提供關鍵的高強度數據，幫助我們修正雪梨馬的目標配速。長遠來看，這能大幅降低未來訓練中腳趾或膝蓋舊傷復發的可能性。

**5. Execution 執行建議：**
*   **節拍器介入**：本週跑步課表請全程開啟手錶節拍器（設定 175）。聲音引導能減少大腦思考負擔。
*   **縮短步幅**：如果覺得 175 步頻太喘，請縮短步幅，感覺像是在腳下「踩火堆」一樣快速抬起。記住，頻率優先於步幅。
*   **核心支撐**：週四的重訓日請加強單腳穩定度與登山者動作，這能直接輔助步頻的提升。強大的核心能防止在高步頻下身體出現不必要的晃動，確保能量精準輸出。
