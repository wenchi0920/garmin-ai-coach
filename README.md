本週您正處於日本旅遊期間。這是一個絕佳的「體能與心理雙重放鬆期」。為確保前期累積的有氧底盤不流失，同時深度固化修復效果，教練提供以下建議：

其次，**下肢深度保養**絕對不可輕忽：長期的連續步行容易導致足底筋膜與小腿後側（腓腸肌、比目魚肌）承受壓力。每天返回飯店後，強烈建議您務必進行 10-15 分鐘的瑜伽伸展。您可以重點執行「靠牆掛腿」與「下犬式」，解除下肢血液堆積，促進淋巴回流，減輕腫脹感；下犬式則能深層拉伸背側筋膜線，確保肌肉彈性。腳趾傷勢雖然 100% 痊癒，但末端的防護與穿著舒適的休閒鞋也是保護足部的重要一環。

最後，**營養與身心重置**：旅遊期間請盡情享受異國美食，作為對自己過去幾週認真訓練的最佳獎勵。心理放鬆能大幅降低體內的皮質醇。同時請注意基本的水分補充與優質蛋白質攝取，減少疲勞造成的輕微發炎反應。請將本週（W21）視為一個完美的過渡週，徹底讓心靈歸零。

## 📅 4. 本週訓練重點 (W21 基礎期與旅遊恢復)

本週 (W21) 恰逢您的日本旅遊，這是一個戰略上極為重要的過渡期（Transition Phase）。在經歷了過去幾週的基礎期重建後，身體已經累積了一定程度的深層疲勞。本週的核心指導原則是「主動恢復與輕度維持」，讓身體在異國的放鬆氛圍中，使這些深層疲勞得以完全釋放，同時確保好不容易建立起來的心肺體能與肌肉張力不至於大幅流失。以下是本週的五大核心重點與詳細執行方針：

**一、 訓練重點：出國前質量刺激與旅遊期動態恢復**
本週的訓練結構有著明顯的兩極化。本週前半段（5/18-5/19）專注於行前的高效刺激，包含深度的修復瑜伽，以及針對乳酸門檻的 7km T 門檻跑。這不僅是為了保持肌肉的收縮張力，更是讓心肺系統在休假前維持活躍。本週後半段（5/20-5/24）則完全切換至旅遊期間的動態恢復主軸。在這幾天內，我們不再追求高跑量，也不設定任何高強度的間歇課表。您的重心應專注於維持心肺活性與緩解大量步行帶來的疲勞，並將海外的大量步行視為優質的低強度有氧刺激。過渡期是非常寶貴的，有助於肌肉重建和心理建設，請好好把握。

**二、 核心目標：主動修復身體能量，維持 Z2 有氧基礎**
本週的核心任務是在身心放鬆的同時，維持基本的體能運轉。目標是透過旅遊中安排的 5km Z2 探索跑，以及每天睡前高品質的修復瑜伽（如下犬式、靠牆掛腿等）或簡短的徒手肌力（如深蹲、死蟲式等），來達成動態的生理平衡。在進行維持性訓練時，請完全捨棄對配速的執著。有意識地維持輕快的步頻，讓雙腿習慣高轉換率即可。體能維持是為了確保下週回國時能無縫接軌進入下一階段的強化期，只要讓身體記住流汗與運動的感覺，目標便已圓滿達成。不要讓課表成為旅遊的負擔。

**三、 關鍵課表：行前 T 門檻跑與旅遊探索跑**
本週的關鍵訓練集中在出發前與旅途期間的點綴，確保身體在放鬆中不失敏銳：
1. **7km T 門檻跑 (5/19)**：包含暖身與 T 配速 (約 6:07/km)。這是一次針對乳酸門檻的精準刺激，能在出國前喚醒您快速收縮的肌纖維，有效提升身體清除血乳酸的效率，對於長期的速度維持有極大的幫助。
2. **5km Z2 旅遊探索跑 (5/20 或視行程調整)**：我們安排了一次 5km 的 Z2 有氧跑。主要目的是讓身體快速適應當地的氣候與濕度。在毫無壓力的狀態下重新建立有氧基礎，喚醒長跑肌群。能以雙腳探索陌生的城市風景，將為您的旅程增添獨特的回憶。
3. **旅遊舒緩瑜伽與核心維持 (5/21-5/22)**：利用零碎時間進行深蹲、死蟲式維持肌肉張力；睡前執行靠牆掛腿促進下肢血液回流。

**四、 預期成效：深層疲勞歸零，身心靈滿載回歸**
經過這趟日本旅途，我們預期您能獲得身心雙重的極致放鬆。生理上，透過規律的瑜伽拉伸與輕量步行，自律神經系統將獲得徹底的重置，肌肉微小損傷也能在此期間完全修復；心理上，遠離日常工作與高壓課表的束縛，將重新點燃對跑步的熱情。只要在旅途中保持適度活動，您的有氧底盤不僅不會退步，甚至會因為獲得了充分休息，而展現出更強大的潛能。這次過渡期將為八月底雪梨馬拉松的長距離訓練打下堅如磐石的恢復基礎，讓下半年的訓練可以更有效率。

**五、 執行建議：傾聽身體真實反饋，絕對不拘泥於既定課表**
在日本旅遊期間，所有的課表安排都具有極大的「彈性」與「容錯率」。旅遊的品質與安全遠高於課表的完成度。如果您當天觀光行程滿檔，回到飯店時覺得腿部肌肉痠痛、全身疲乏，請毫不猶豫地跳過當天的跑步或肌力課表，直接改為深度的瑜伽伸展與泡熱水澡。享受旅途、不產生任何心理壓力，是本週的最高指導原則！若進行探索跑，請將專注力放在呼吸節奏與風景上，確保維持在穩定的 Z1 或 Z2 區間內，平安健康地完成每一次探索。聆聽身體最真實的聲音，才是科學訓練中最重要的一環。
