## 📅 4. 本週訓練重點 (W23)
本週 (W23) 是迎向 8 月雪梨馬拉松的倒數 13 週，我們正式開啟備賽週期的「進展期初期 (Strengthening Phase)」。這個階段的戰略核心是「鞏固有氧底盤，喚醒門檻速度，強化剛性」。六月是奠定基礎的關鍵月份，本週課表設計緊扣以下五大項目：

**一、 訓練重點：乳酸門檻刺激與基礎有氧擴張 (Threshold & Endurance)**
本週採取「雙軸並進」的策略。第一軸為週四的「6km T 門檻跑」，這項速度耐力指標要求以 6:07/km 巡航，強迫身體提升清除血乳酸的效率；第二軸是週日的「16km LSD 長距離慢跑」，旨在徹底擴展有氧引擎容量，讓身體在長時間運動下更有效率地燃燒脂肪，這對於全馬後半段的續航力至關重要。

**二、 核心目標：重建下肢抗疲勞能力與核心穩定 (Leg Strength & Core Stability)**
為了對抗雪梨馬拉松起伏多變的賽道，本週前段已安排高質量重訓。目前的重訓核心不僅是單純力量增長，更著重於大腿的「離心收縮」能力及骨盆周圍與核心的穩定性。這能大幅降低比賽後半段因肌肉疲乏而產生的代償風險，確保跑姿不崩潰。

**三、 關鍵課表：T 門檻測驗與長距離耐力考驗 (Key Workouts)**
*   **W23-6km T 門檻跑 (06/04 週四)**：本週的速度指標測驗。包含 2km 暖身、3km 穩定維持 T 配速 (6:07/km) 的主課表，以及 1km 緩跑冷卻。請密切關注心率，若遇高溫導致心率提前進入 Z5，請主動改以心率區間主導，避免過度耗損。
*   **W23-16km LSD 長跑 (06/07 週日)**：本週耐力與心理的雙重考驗。請嚴格鎖定 Z2 區間 (配速 7:00-7:30/km，心率 153-166 bpm)。這不僅是擴大有氧基礎的關鍵，更是完成 ONE TOKYO VR 21K 挑戰剩餘里程的最佳時機。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能紀律地完成本週課表，您的生理適應將獲得顯著提升。T 門檻跑將促進微血管增生，讓未來維持 M 配速 (6:33/km) 時體感更加游刃吸有餘；而重量訓練與 LSD 的結合則能深層鍛鍊慢縮肌的極限耐力，為雙腿帶來強大的下肢剛性，以應對長距離挑戰。

**五、 執行建議：高步頻防線與夏日補給實戰演練 (Execution Advice)**
*   **嚴守高步頻防線**：請在所有跑步課表中刻意維持 175 spm 以上的高步頻，利用小步幅降低觸地時間（降至 270ms 以下），大幅減少落地對膝蓋與腳趾的衝擊。
*   **實戰補給演練**：週日的 16km 長跑預計耗時兩小時左右，請務必在出發前及起跑後每 45 分鐘，進行能量膠與水分攝取的實戰測試，找出最適合腸胃的補給節奏。
*   **氣候與心率應對**：夏日濕熱氣候極易引發心率漂移 (Cardiac Drift)，請時刻「以心率為尊」，當發現心率逐漸超標時，請主動降速，切勿為了死守配速而過度透支體能。
