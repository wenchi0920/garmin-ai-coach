## 1. 系統核心定位
Garmin AI Coach 是一個結合尖端運動科學與大數據分析的專屬自動化課表系統，核心理念在於「科學訓練，精確校準」。系統透過每日自動解析 Garmin 裝置生成的 `.fit` 檔案，深度追蹤您的心率變異度、跑步動態與疲勞負荷。結合您的專屬跑力指標 (VDOT) 與個人化生理回饋，AI Coach 每週會動態生成包含長距離慢跑 (LSD)、基礎有氧 (Z2)、乳酸門檻 (T) 以及跑後修復與專項肌力的全面性訓練計畫。我們不僅專注於突破成績，更在乎降低受傷風險與身心靈的平衡，讓跑者只需百分之百專注於訓練的執行，將配速與疲勞監控交給系統，穩健邁向每一次的起點與終點，協助您達成設定的卓越目標。

## 📅 4. 本週訓練重點 (W24)
本週 (W24) 我們持續處於備戰雪梨馬拉松的「進展期初期 (Strengthening Phase)」。由於上週未能足額完成 LSD，本週的核心要務是「確保長距離課表的完整執行」，在鞏固有氧底盤的同時，強化乳酸門檻的耐受力。以下是本週課表的五大核心剖析：

**一、 訓練重點：彌補耐力缺口與鞏固門檻跑穩定性 (Endurance & Threshold Focus)**
本週的訓練邏輯採取了「雙軸並進」的穩健強化策略。第一軸著重於「T 門檻刺激」，目的在於訓練您的身體於乳酸即將大量堆積的臨界點（約 6:07/km）提升「耐受度」與「清除效率」。這能實質拉高您在無氧邊緣的續航力，讓未來的馬拉松配速 (M pace) 變得更加游刃有餘。第二軸則是週末的重頭戲「16km LSD 長距離慢跑」。透過超過 90 分鐘以上的持續有氧跑動，強迫身體動用脂肪作為主要能量來源，並促進微血管網與肌肉細胞內粒線體的增生。這是全馬要達成 Sub 4:30 且避免撞牆期最不可或缺的基石。本週必須確保這個雙軸訓練的核心精神被徹底且連貫地執行，以彌補先前產生的有氧耐力缺口。

**二、 核心目標：模擬雪梨馬微起伏地形與下肢抗疲勞強化 (Terrain Simulation & Leg Strength)**
雪梨馬拉松的賽道以其充滿挑戰的起伏丘陵地形著稱，頻繁的上下坡對腿部肌肉的離心收縮要求極高。因此，本週的核心目標之一，是透過週末的長跑刻意尋找具有微幅起伏的戶外路線，提早讓下肢適應坡度變化所帶來的不穩定負荷。同時，週二與週四所安排的「下肢基礎肌力」與「跑者核心穩定」課程，將透過深蹲、單腳硬舉與棒式等經典動作，直接強化您的股四頭肌、臀中肌與軀幹的抗疲勞度。當比賽進入後半段體能大幅下滑時，強韌的下肢剛性與穩固的核心，將是您維持標準跑姿、防止抽筋受傷的最強防線。

**三、 關鍵課表：7km T 門檻跑與 16km 長距離耐力考驗 (Key Workouts)**
*   **W24-7km T 門檻跑 (週二)**：本次課表結構為 2km 輕鬆暖身、4km 的穩定 T 配速 (目標 6:07/km) 以及 1km 緩跑修復。這是一次極高質量的無氧邊緣刺激。請勇敢嘗試在戶外穩定鎖定在 6:07/km，讓身體真實感受門檻壓力。這是提升 VDOT 數值與配速經濟性的關鍵鑰匙。
*   **W24-8km Z2 基礎有氧 (週四)**：以穩定擴張有氧引擎為目標，請將心率嚴格控制在 153-166 bpm 之間，配速維持在 7:12-8:02/km。這不僅是體能的堆疊，更是讓雙腿在 T 跑後進行積極的動態恢復。
*   **W24-16km LSD 長跑 (週日)**：這是本週的終極考驗。請嚴格鎖定在 Z2 區間，將專注力完全放在呼吸節律的順暢度。記取上週的教訓，本週的長跑務必提早於清晨起跑，確保能「不中斷」地完成目標里程，重新建立對長距離的生理適應與心理自信。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能貫徹執行本週進展期的質量與有氧課表，我們預期能觀察到顯著的綜合生理適應。您的心肺系統將更能適應較高的需氧量，未來在執行 Z2 輕鬆跑時，同等配速下的心率將會下降，體感疲勞度也會大幅減輕。此外，藉由專項肌力訓練與 16km 長跑的雙重肌肉破壞與重建機制，您足弓、阿基里斯腱及膝關節周圍韌帶的彈性與剛性將被顯著強化，進一步提升每一步的跑步經濟性。這種全身性的綜合強化，將讓您在面對雪梨馬拉松時，保持游刃有餘的體能。

**五、 執行建議：高步頻防線、夏日實戰補給演練與恢復策略 (Execution Advice)**
*   **嚴守高步頻防線**：無論是 T 跑還是 LSD，請時刻提醒自己維持 175 spm 以上的高步頻。高步頻能有效分散每一步的衝擊力，保護膝關節免受過大的煞車效應傷害。
*   **夏日環境熱適應與時段優化**：夏季氣溫高，心率向上漂移是正常的生理現象。請務必以「心率區間」為最高指導原則。若體感過於悶熱或心率飆升，請果斷放慢速度。強烈建議將週末 LSD 移至清晨涼爽時段執行。
*   **實戰補給演練**：對於耗時超過 90 分鐘的 16km LSD，請隨身攜帶能量膠，嘗試每 45-50 分鐘規律補充一次，並定時定量攝取水分與電解質。這能訓練腸胃在跑動中的吸收能力。跑後請確實執行修復瑜伽，加速排酸。
