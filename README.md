# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個結合運動生理學、穿戴裝置大數據分析與動態自動化課表生成的頂尖科學化馬拉松訓練系統。本系統以「數據驅動，精確校準，無痛增長」為核心理念，每日自動讀取與解析跑者的 Garmin 穿戴裝置數據（包括 .fit 檔、心率、步頻、HRV 心率變異度、身體能量 BB、靜息心率 RHR 與睡眠品質等）。透過對個人化動態跑力 (VDOT) 的動態校準，精準設定輕鬆跑 (E)、馬拉松配速 (M)、乳酸門檻 (T) 及間歇跑 (I) 之配速與心率區間。系統在給予生理超載刺激與確保超補償修復之間取得黃金平衡，協助跑者突破耐力天花板。同時，系統設有強大的防傷避險機制，當生理指標出現疲勞警訊時即時調整強度，陪伴跑者安全、系統化地實現賽事目標。

## 📅 4. 本週訓練重點 (W30)

本週（2026-07-20 ~ 2026-07-26）為「進展期後段 (W28-W30)」的第三週（高峰鞏固週）。本週訓練邏輯的核心在於**「穩步擴大有氧容量、拉長乳酸門檻耐受時間，並以高步頻防禦機制與補給演練應對高峰里程突破」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：高峰鞏固期的負荷控制與防禦控速**  
    本週為備戰雪梨馬拉松進展期後段的最高峰週（W30 高峰鞏固週），預計總跑量達到 41 公里，為備賽以來的最高峰。由於上週累積了顯著的自主神經疲勞，且多次出現配速失控教訓，本週的重點不是追求盲目速度突破，而是「在安全受控的配速與心率區間內，穩定完成里程累積」。特別是週六將迎來備賽以來最長的 17km LSD 挑戰，因此平日的非質量跑（輕鬆跑與恢復跑）必須執行「防禦性控速」，將強度鎖定在 Z2/Z1 區間（心率 < 153/166 bpm，配速 7'12"-8'02"/km），促使身體產生超補償修復。

*   **二、 核心目標：17km 實戰補給演練與配速守恆**  
    1. **ONE TOKYO Global Virtual Race 4th 虛擬賽與補給演練**：週六（07/25）安排的 17km LSD 戶外長跑為備賽期單次最長里程突破。要求全程戶外 GPS 記錄，學員必須演練雪梨馬拉松實戰補給策略，於 6km 與 12km 處定時補充能量膠與電解質水，測試胃部高運動量下的吸收力。  
    2. **重塑配速與心率紀律**：週二 11km T 門檻跑（含 7km 門檻區段）已順利以 6'07"/km 均速完成。週四 8km 輕鬆跑與週末慢跑心率需嚴格控制在 Z2 區間（153-166 bpm，134-153 bpm 尤佳）。  
    3. **維持下肢穩定力量**：週一（上半身）與週四（下半身）重訓重點放在單腳穩定性（單腳硬舉、保加利亞分腿蹲），強化臀中肌與後側鏈剛性，為週六 17km 長跑與雪梨馬丘陵地形建立關節抗衝擊能力。

*   **三、 關鍵課表：T門檻跑的紀律重建與LSD長跑實戰**  
    1. **週二 11km T 門檻跑 (07/21)**：已執行 5.64km 跑步機 T門檻跑，平均配速 6'07"/km（主區段 5'45"-5'52"/km），步頻 174 spm，順利建立高配速巡航感與乳酸耐受力。  
    2. **週四 8km Z2 基礎有氧跑 (07/23)**：已完成 8.00km 戶外基礎跑 (耗時 55m 21s，平均配速 6'55"/km，心率 160 bpm)。重點在於「慢」，搭配 Garmin 心率警示，促進肌群毛細血管增生。  
    3. **週六 17km LSD 戶外跑 (07/25)**：1km Z1 暖身 + 15km Z2 有氧慢跑 (配速 7'12"-8'02"/km，心率 153-166 bpm) + 1km Z1 緩跑。必須戶外執行，模擬雪梨冬季海風與坡度，維持 170+ spm 步頻並執行定點補給。

*   **四、 預期成效：乳酸清除率優化、跑步經濟性提升與補給胃部適應**  
    1. **乳酸清除能力提高**：透過精準的 7km 乳酸門檻刺激，提升乳酸轉運蛋白活性，使身體在乳酸堆積與清除間達到更高動態平衡，提高高強度下的配速耐受力，為雪梨馬拉松 Sub 4:30（目標均速 6'23"/km）奠定配速基石。
