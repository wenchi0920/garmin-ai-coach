## 📅 4. 本週訓練重點 (W21)

本週 (W21) 進入了訓練計畫中特殊的「日本旅遊過渡週」。這週的核心精神是「戰略性修復」，我們不再追求里程的堆疊，而是專注於深層恢復與維持基礎體能。以下是本週的五大核心項目與執行方針：

**一、 訓練重點：戰略性減量與適應 (Strategic Tapering & Adaptation)**
本週的訓練呈現「出國前強化、旅途中修復」的兩極化安排。在出國前完成高質量的乳酸門檻刺激，隨後在旅遊期間轉入極低強度的維持模式。這種安排能有效避免連續放假導致的體能滑落，同時透過適度的低強度期引發「超補償 (Supercompensation)」效應，為下週回歸常態訓練儲備能量。

**二、 核心目標：維持 Z2 有氧基礎與心肺感知**
旅遊期間，每天的徒步行程本身就是一種優質的低強度恆速運動 (LISS)。我們的目標是在不增加額外壓力的情況下，透過零星的輕量慢跑維持心肺系統的適度喚醒，防止毛細血管退化。請將專注力放在維持輕快的步頻，而非追求特定的配速。

**三、 關鍵課表：出國前 T 門檻與每日修復**
1. **7km T 門檻跑 (出國前)**：本週唯一的質量課表，透過 3km 的 T 配速刺激，強化身體清除血乳酸的能力，保持對速度的神經記憶。
2. **5km Z2 旅遊探索跑 (旅途中)**：一次無壓力的異國晨跑，目標將心率穩定在 Z2 區間，專注於 175+ spm 的步頻，享受環境。
3. **每日修復瑜伽與靠牆掛腿**：每晚必做的基礎保養，釋放當天步行累積的下肢壓力，維持髖關節與腳踝的靈活度。

**四、 預期成效：深層疲勞重置與受傷風險歸零**
透過本週的戰略性減量，預期您的身體能量 (Body Battery) 與 HRV 將會穩定回升，自律神經系統得到充分休息。更重要的是，這週的減壓能讓腳趾傷勢的復原狀況徹底固化。當您下週回國時，雙腿將重獲輕盈與力量，隨時準備好迎接下一階段的高強度基礎期訓練。

**五、 執行建議：傾聽身體，靈活變通 (Listen & Adapt)**
**「旅遊與放鬆為主，訓練課表為輔」** 是本週的最高指導原則。如果當天行程滿檔、雙腿沉重，請毫不猶豫地取消跑步課表，直接改為溫泉泡澡與深層拉伸。在陌生街道跑步時，請注意路面安全與水分補充。保持愉快的心情，這將是最好的修復劑。
