**【專業恢復、防傷策略與近期建議】**
面對本週即將啟程的日本旅遊，教練將此階段定位為絕佳的「體能與心理雙重放鬆期」。為確保前期辛苦累積的有氧底盤不流失，同時將修復效果深度固化，提供以下專業策略：
1.  **動態恢復視角**：日本旅遊勢必伴隨大量步行，請將這些日常觀光視為低強度的「主動恢復 (Active Recovery)」。大量的低強度有氧活動同樣能刺激微血管增生，對長跑者而言是極佳的隱性訓練，請放鬆心情享受。
2.  **下肢深度保養**：長期的連續步行易導致足底筋膜與小腿承受累積性壓力。每天返回飯店後，務必進行 10-15 分鐘的瑜伽伸展，強烈建議執行「靠牆掛腿」與「下犬式」。這能有效解除下肢腫脹，促進血液回流，確保筋膜彈性。腳趾傷勢雖已痊癒，但末端防護與放鬆依然不可懈怠。
3.  **營養與身心重置**：旅遊期間請盡情享受異國美食，作為對自己認真訓練的獎勵。同時注意水分與優質蛋白質的攝取，以減少疲勞造成的輕微發炎反應。將 W21 視為完美的過渡週，徹底讓心靈歸零重置，為回國後更高強度的馬拉松專項訓練週期準備滿滿的動能。

## 📅 4. 本週訓練重點 (W21 基礎期與旅遊恢復)

本週 (W21) 恰逢您的日本旅遊，這是一個戰略上極為重要的過渡期（Transition Phase）。我們的核心指導原則是「主動恢復與輕度維持」，讓身體在基礎重建期累積的深層疲勞得以完全釋放，同時確保心肺體能與肌肉張力不流失。以下是本週的五大核心重點與執行方針：

**一、 訓練重點：出國前質量刺激與旅遊期動態恢復**
本週前半段（5/18-5/19）專注於行前的高效刺激，包含深度的修復瑜伽與一次具備強度的 T 門檻跑，以保持肌肉的收縮張力。本週後半段（5/20-5/24）將完全切換至旅遊期間的動態恢復主軸。我們不再追求高跑量、不設定高強度課表，而是專注於「維持心肺活性與緩解步行疲勞」。將海外的大量步行視為優質的低強度有氧刺激。

**二、 核心目標：主動修復身體能量，維持 Z2 有氧基礎**
本週的核心任務是在身心放鬆的同時，維持基本體能。目標是透過旅遊中安排的 5km Z2 探索跑，以及每天睡前高品質的修復瑜伽（如下犬式、靠牆掛腿等）或徒手肌力（深蹲、死蟲式等），來達成動態平衡。在慢跑時不看配速，有意識地維持輕快的步頻，讓雙腿習慣高轉換率即可。

**三、 關鍵課表：行前 T 門檻跑與旅遊探索跑**
本週的關鍵訓練集中在出發前與旅途期間的點綴：
1.  **7km T 門檻跑 (5/19)**：包含 3km 的 T 配速 (約 6:07/km)。這是一次針對乳酸門檻的精準刺激，在出國前喚醒快速收縮的肌纖維，提升身體清除乳酸的效率，讓神經系統記住高配速的流暢體感。
2.  **5km Z2 旅遊探索跑 (5/20)**：抵達日本的首日或次日，安排一次 5km 的 Z2 有氧跑。主要目的是適應當地氣候，在放鬆的狀態下重新建立有氧基礎，喚醒長跑肌群，並以雙腳探索陌生的城市風景。

**四、 預期成效：深層疲勞歸零，身心靈滿載回歸**
我們預期在經歷這趟日本旅途的身心雙重放鬆後，您在生理與心理上都能夠達到滿血復活的極佳狀態。透過每日規律的瑜伽拉伸與輕量活動，您的自律神經系統將獲得徹底重置。只要順利完成行前的質量跑並在旅途中保持適度活動量，您的有氧底盤不僅不會退步，甚至會因為獲得充分休息而展現出更強的潛力，為接下來的強化週期注入嶄新能量。

**五、 執行建議：傾聽身體真實反饋，絕對不拘泥於既定課表**
在日本旅遊期間，所有的課表安排都具有極大的「彈性」與「容錯率」。如果您當天觀光已經走了兩萬步，覺得腿部肌肉痠痛疲乏，請毫不猶豫地跳過當天的跑步或肌力課表，改為深度的瑜伽伸展與泡澡。享受旅途、不產生任何壓力是本週的最高指導原則！若有安排探索跑，請將專注力放在呼吸節奏與沿途風景上，確保輕鬆跑維持在穩定的 Z2 區間內，平安健康地完成每一次訓練。
