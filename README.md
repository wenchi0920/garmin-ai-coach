# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個整合運動生理學與大數據分析的頂尖科學化馬拉松訓練系統。本系統以「數據導航，無痛增長」為核心理念，每日自動讀取並解析跑者的 Garmin 穿戴裝置數據。透過對心率、步頻、配速、HRV（心率變異度）與身體能量 (BB) 等關鍵生理指標的長期追蹤，動態評估跑力值 (VDOT)，精準校準個人化的訓練區間（包括輕鬆跑 E、馬拉松配速 M、乳酸門檻 T 與間歇跑 I）。本系統在給予生理系統「適度超載刺激」與確保「超補償恢復」之間取得黃金平衡，協助跑者在不累積傷痛的前提下，穩步提升有氧耐力與跑步經濟性，系統化地實現賽事目標，達成個人體能天花板的突破。

## 📅 4. 本週訓練重點 (W28)

本週為「進展期後段 (W28-W30)」的第一週。本週訓練的設計邏輯是**「降階長跑以重建信心、微調門檻跑以刺激心肺、並將步頻強制拉升至 170+ spm 以建立安全防禦機制」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：LSD 降階信心重建與有氧配速轉化**
    鑑於上週日 18km LSD 因心率漂移與低步頻在 8.86km 提前中止，本週的長跑日採取「主動防守與安全降階」策略，將 LSD 里程降至 **14km**。本週的訓練重點不在於堆疊里程，而在於「全程配速守恆」與「步頻控制」。透過降階 4 公里，我們為下肢關節與股四頭肌提供了一次寶貴的超補償（Supercompensation）恢復契機。此外，週二原訂的質量跑（T 門檻跑）將 T 區間段由 5km 微幅提升至 **6km**，以 5'58"/km 的乳酸門檻配速刺激心肺（雖週二因疲勞避險未執行，但在後續備賽中仍應作此規劃），旨在提升在高酸性狀態下的乳酸清除效率與心肺代償能力。

*   **二、 核心目標：步頻強制突破 170+ spm 與起伏地形控速**
    1.  **步頻底線建立**：全體跑步課表（包含週一恢復跑、週四有氧跑及週六 LSD）均須使用手機節拍器 App，將步頻強制維持在 **170+ spm**（理想目標 175 spm），藉由縮短步幅來降低垂直振幅與每次著地的煞車效應。
    2.  **乳酸門檻配速穩定 (5'58"/km)**：在質量跑中，將心率穩定控制在 Zone 4 (155-167 bpm) 內，避免配速過快（如上週跑出 5'47"/km 導致隔日疲勞累積）。
    3.  **雪梨賽道起伏適應**：週六的 14km 戶外跑必須選擇有坡度起伏的路段，練習「上坡縮短步幅維持步頻，下坡身體微前傾降低衝擊」的控速技巧，並模擬雪梨馬拉松的寒冷強風氣候，加強熱身與保暖。
    4.  **腸胃補給窗口演練**：利用週六的長距離跑，演練每 45 分鐘固定補給一包能量果膠與水的節奏，訓練腸胃在高強度活動下的吸收能力。

*   **三、 關鍵課表：10km T 門檻跑與 14km LSD 戶外實戰演練**
    1.  **週二 10km T 門檻跑**：2km Z1 暖身 + **6km T配速 (5'58"/km, 心率 155-167 bpm)** + 2km Z1 緩跑。此為本週強度最高的質量課表，必須嚴格控速，防止乳酸過度堆積。（*本週二已執行避險休息，故後續課表需專注於週末 LSD 的完成*）。
    2.  **週六 14km LSD 長距離慢跑**：**強制戶外執行**。1km Z1 暖身 + 12km Z2 有氧慢跑 (HR: 131-143 bpm, 配速: 7'01"-7'51"/km) + 1km Z1 緩跑。出發前 3 公里強制以 7'45" - 8'00"/km 熱身，中後段以 170+ spm 步頻防守，嚴禁超速。
    3.  **單側下肢力量訓練（週四）**：週四安排下半身重訓（重點在於**保加利亞分腿蹲**與**單腳硬舉**，每邊 3 組，每組 10-12 次），強化骨盆穩定度，提升單腳支撐力量以應對起伏賽道。（*今日週四因疲勞，重訓強度需調整為自重控制*）。

*   **四、 預期成效：降低垂直振幅與減緩後半程心率漂移**
    透過步頻從 152 spm 提升至 170+ spm，預期能將垂直振幅控制在 **75mm 以下**，觸地時間縮短至 **270ms 以下**。這將顯著提高跑步經濟性 (RE)，減少每次著地對股四頭肌的離心負荷。隨著步姿的改善與 LSD 控速的落實，週六 LSD 中後段的心率漂移現象將得到有效緩解，心率能穩定在 Zone 2 區間，使身體在長跑後半程仍能維持良好的能量代謝與穩定的跑姿，不再因肌肉提前力竭而中止。

*   **五、 執行建議與警告：節拍器應用、腸胃補給窗口與極限警訊防守**
    1.  **節拍器輔助**：跑步時請播放 170 bpm 的背景節拍器，或是利用手錶的步頻警示，主動透過「小步幅、輕落地」強迫自己跟上節奏。
    2.  **腸胃補給演練**：週六的 14km 長跑，請隨身攜帶 2 包能量果膠。在開跑前 30 分鐘補充水分與一片吐司，開跑後每 45 分鐘（約在第 6km 與第 11km 處）固定補充一包，搭配少量水分。
