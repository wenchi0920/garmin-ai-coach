## 1. 系統核心定位
Garmin AI Coach 是一個深度結合運動科學、數據分析與自動化課表的專屬教練系統。本系統以「**科學訓練，精確校準**」為核心價值，每日自動解析跑者的 Garmin `.fit` 紀錄，監測心率變異度 (HRV)、睡眠分數、壓力指數等關鍵生理數據，並即時評估疲勞與恢復狀態。透過動態修正跑力指標 (VDOT)，系統能將複雜的運動生理數據轉化為個人化的每週訓練課表。無論是備戰高強度賽事、傷後復健期，還是面對氣候挑戰，AI Coach 都能提供精準的訓練與修復建議，幫助跑者在追求卓越成績的同時，大幅降低受傷風險，享受運動與心靈成長的旅程。

## 📅 4. 本週訓練重點 (W24)
本週 (W24) 正處於雪梨馬拉松備賽的「進展期初期」，這是決定您耐力天花板高度與完賽信心的關鍵階段。所有的訓練都將圍繞著找回有氧耐力與適應賽道地形來展開。

**一、訓練重點：修補耐力斷層與重新聚焦 (Endurance Rebuilding)**
由於前兩週因氣候與外務導致里程缺口，本週首要任務是「穩定補償有氧里程」。我們絕不追求極速，而是嚴格要求每公里的輕鬆跑與長跑都能精準落在指定的心率區間內，重新喚醒脂肪供能效率，找回對長距離奔跑的耐心。請將專注力放在完成預定時間與距離，而非急於提升配速。夏季高溫容易使心率產生飄移，若發現心率超標，應立刻放慢腳步，維持在有氧區間內，藉由長時間的低強度運動促進微血管增生與粒線體發展。

**二、核心目標：力量與耐力的平穩轉化 (Strength-to-Speed Conversion)**
您近期在深蹲與硬舉等重訓上取得了傲人的肌力進步，現在必須將這些「靜態結構力量」轉化為實質的跑動力。本週目標是利用強大的核心與下肢來維持高步頻與骨盆穩定，確保在週末的 LSD 中能克服疲勞與起伏地形，為 Sub 4:30 建立穩健基礎。肌力與有氧的完美結合將是本週與下週訓練銜接的核心。您需要透過反覆的跨步與推蹬，讓肌肉適應長時間的收縮與放鬆，使神經肌肉的徵召更加流暢。

**三、關鍵課表：6km Z2 輕鬆跑與 16km LSD 長跑 (Key Sessions)**
本週成效將由週末這兩項關鍵課表決定，請務必嚴格遵守配速與心率紀律：
*   **06/13 (週六) 6km Z2 輕鬆跑**：作為輕量有氧刺激，請將心率嚴格控制在 Z1-Z2 區間，配速維持在 7:12-8:02/km。此課表目的是為隔日保留糖原，讓雙腿維持活絡但不產生任何疲勞累積。利用這段時間檢視跑姿，確保手臂擺動自然，步伐輕盈。
*   **06/14 (週日) 16km LSD 長跑**：這是本週的核心期末考，更是雪梨馬的模擬跑。目標為「不間斷完成」，維持 Z2 區間的高標心率。這是一場體能與心理韌性的雙重建設，若遇高溫可彈性分早晚兩次完成，重點在於總量的積累。將這次長跑視為與身體對話的過程，仔細體會疲勞的累積與代謝。

**四、預期成效：有氧效率提升與恢復力強化 (Expected Gains)**
順利完成本週連日的基礎課表後，預期能大幅改善心肺飄移現象 (Cardiac Drift)，顯著提升有氧效率。同時，基礎代謝與微血管密度的增加，能讓您在未來面對高強度乳酸門檻跑 (T跑) 時恢復得更快。穩定累積里程將能大幅提升對長距離的適應力，有效降低 30 公里後發生撞牆期的風險。扎實的有氧底子也能幫助您在訓練後更快速地排解乳酸，迎戰下週的進階挑戰。

**五、執行建議：高溫應對與補給實戰 (Execution Strategy)**
*   **熱適應與時段選擇**：請避開烈日，盡量將訓練安排在清晨 5:30 前或傍晚太陽下山後起跑。以「心率」為最高指導原則，若心率飆升請果斷降速，切勿硬撐配速。適當的熱適應有助於增強排汗效率，但安全永遠是第一考量。
*   **實戰補給演練**：週日的 16km LSD 請務必實地測試能量膠與水分攝取。建議在第 7km 與第 12km 進行補給，讓腸胃提早適應跑步中的消化負載。了解自己對不同品牌補給品的接受度，避免賽事當天發生腸胃不適。
*   **步頻即時監控**：在長跑疲勞期（12km 後），要有意識地縮短步幅、提升步頻至 175 spm 以上，並維持核心收緊。若在跑步機上訓練，更應主動調高律動以減少觸地時間，保護膝關節免受衝擊。
