## 📅 本週訓練重點：W15 基礎累積與賽前校準 (04-06 ~ 04-12)

2.  **核心目標：確立 10K 比賽體感配速 (RPE 6-7)**
    週四的「10K 賽前體感測試」是本週最重要的指標。我們要確認在 6:05-6:30/km 的目標區間內，您的心率是否能穩定在 Z3 (166-171 bpm) 以下。如果心率在第三組測試結束前突破 175 bpm，代表 4/18 的開賽配速必須保守下修。

3.  **關鍵課表：10K 體感測試 (週四) 與 12km LSD (週六)**
    *   **週四測試 (2km E + 3x1km T + 1km E)**：這是一個「喚醒最大攝氧量」的過程。短暫的門檻跑能增加肌肉對氧氣的攝取效率，同時不累積過多疲勞。
    *   **週六長跑 (12km Z2)**：這是最後一次長距離有氧累積。請務必嘗試模擬戶外環境（若天氣允許），感受真實地面的推進力與阻力。

4.  **預期成效：跑步經濟性 (RE) 的質變**
    *   **步頻提升**：我們希望看到步頻在 7'00" 配速下能穩定回升至 172+ spm。
    *   **心血管穩定**：透過本週的 Z2 累積，預期在週六的 12km 長跑中，後半段的心率漂移 (Cardiac Drift) 能控制在 3% 以內。

5.  **執行建議 (詳盡導引)**
    *   **「節拍器」是本週的主角**：所有的跑步訓練，請強制開啟節拍器 175 bpm。這不僅是為了保護腳趾，更是為了減少每一步的垂直振幅。
    *   **力量訓練的針對性**：週三的肌力訓練請專注於「單腳 RDL」與「保加利亞分腿蹲」，強化跑步時的側向穩定性，避免在長跑中後段因骨盆晃動造成效率下降。
    *   **環境模擬**：建議本週至少安排一次傍晚跑步，模擬 4/18 比賽當日的氣溫與光線，調整體感。
