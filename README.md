# 🏃‍♂️ Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位
Garmin AI Coach 是一個深度結合運動科學、數據分析與自動化課表的專屬教練系統。本系統以「**科學訓練，精確校準**」為核心價值，每日自動解析跑者的 Garmin `.fit` 紀錄，透過監測心率、步頻、配速等關鍵生理與運動數據，並即時評估疲勞與恢復狀態。藉由動態修正跑力指標 (VDOT)，系統能將複雜的運動生理數據轉化為個人化的每週訓練課表。無論是備戰高強度賽事、傷後復健期，還是面對氣候挑戰，AI Coach 都能提供精準的訓練與修復建議，自動規劃包含跑步、肌力與瑜伽放鬆的全方位課表，幫助跑者在追求卓越成績的同時，大幅降低受傷風險，享受運動與心靈成長的旅程。

### 【專業恢復與疲勞度評估建議】
綜合當前各項生理指標，您的身體狀況正處於一個非常適合承載訓練壓力的「生產中」狀態。傷勢已不再是阻礙，真正的挑戰在於如何將肌力轉化為耐力。教練提出以下恢復與防護建議：
1. **強化下肢肌群的動態修復**：上週密集的深蹲與硬舉訓練雖然增強了力量，但也可能在腿後肌群與臀大肌累積了深層緊繃。本週的瑜伽放鬆請特別著重在「鴿式」與「下犬式」，確保髖關節的活動度不受影響，這對於維持理想步幅至關重要。
2. **精準控制 T 區間疲勞**：本週三將迎來關鍵的 8km T 門檻跑（配速 6:07/km）。這項訓練對中樞神經系統的消耗較大，請務必確保週二晚上的睡眠品質。若週四晨間的靜止心率 (RHR) 異常上升超過 5 bpm，請主動降低週四與週五的基礎跑強度，以利超量恢復。
3. **高溫與水分管理**：隨著六月氣溫攀升，心率飄移現象會愈發明顯。請在長跑與質量課表前充分補水，並準備適當的電解質補充。高溫下的訓練疲勞會成倍增加，訓練後的冷水浴或冰敷小腿能有效降低發炎反應。
4. **傾聽身體的微小反饋**：雖然舊傷已癒，但在增加跑量的過程中，足底筋膜與阿基里斯腱的負荷會加重。每日睡前利用按摩球針對足底進行 3-5 分鐘的放鬆，能有效預防潛在的發炎風險。

## 📅 4. 本週訓練重點 (W25)
本週 (W25) 我們正式進入進展期的中段。這是一個充滿挑戰但也最能看見突破的階段。本週的訓練將不再只是純粹的累積里程，而是要在穩定的有氧基礎上，針對性地提升「速度耐力」與「賽道地形適應力」。

**一、訓練重點：精準刺激乳酸門檻與長距離地形模擬**
本週的重心將一分為二：平日透過高質量的 T 門檻跑來提升身體排除乳酸的效率，讓您在更高的配速下仍能維持有氧狀態；週末則透過長達 120 分鐘以上的 LSD 長跑，來強化身體對長時間運動的脂肪供能依賴。我們必須將上週在肌力訓練中獲得的「力量」，在長距離跑步中轉化為推蹬的「持久力」。所有的配速執行必須嚴格遵守紀律，不可隨意超速。

**二、核心目標：強化 T 區間耐受力與心肺轉換能力**
要在雪梨馬拉松達成 Sub 4:30 的目標，您必須能在微起伏的地形中從容應對心率的波動。因此，本週的核心目標在於提升「T 區間（6:07/km）的耐受力」。我們要在不過度累積疲勞的前提下，讓心肺系統習慣在 Z4 區間運作，這將顯著提高您的 VDOT 天花板。同時，透過週末刻意選擇有起伏路段的長跑，讓大腦與神經肌肉提前適應雪梨賽道頻繁的上下坡轉換。

**三、關鍵課表：8km T 門檻跑與 16km 起伏 LSD**
本週的成敗將由以下兩次關鍵訓練決定，請務必在精神與體力最佳的狀態下執行：
*   **週三 8km T 門檻跑 (質量日)**：這是本週強度的巔峰。在 2km Z2 充分暖身後，您需要連續 4km 嚴格鎖定在 6:07/km 的 T 配速，心率控制在 171-180 bpm。這考驗的不僅是體能，更是對配速的專注力與步頻的穩定控制（請維持 175 spm 以上）。
*   **週日 16km LSD 長跑日 (模擬賽)**：要求 120 分鐘以上的不間斷跑。**特別要求：請務必尋找有微幅上下坡的路線（如橋樑或河濱坡道）進行訓練。** 藉此模擬雪梨海港大橋與周邊丘陵的起伏，並測試下坡時對股四頭肌的離心收縮控制。

**四、預期成效：提升速度天花板與抗撞牆期能力**
順利吃下本週的 T 門檻跑後，您的微血管密度將進一步增加，身體在接近無氧邊緣時的乳酸代謝速率將獲得提升。這意味著您未來在面對 6:33/km 的馬拉松配速 (M配速) 時，體感會變得更加輕鬆。而 16km 的丘陵 LSD 將能有效增強您的慢肌纖維耐力，提升肝醣儲備的運用效率，大幅降低在 30 公里處遭遇「撞牆期」的機率。

**五、執行建議：專注配速紀律與實戰補給演練**
*   **紀律至上**：在執行 T 跑時，請克制想要衝刺的衝動；在執行 Z2 輕鬆跑時，即使感覺狀況極佳也請壓住速度。真正的進步來自於「在正確的區間給予正確的刺激」，而非盲目求快。
*   **步頻即時監控**：上週數據顯示您的平均步頻偏低（約 162 spm）。本週請在所有跑步課表中加入「步頻意識」，利用節拍器強迫自己小步快頻，降低觸地時間，這對減輕膝蓋負擔有立竿見影的效果。
*   **腸胃訓練與賽前模擬**：週日的 16km 長跑請視為一場小型賽事。請演練完整的賽前早餐計畫（賽前 2 小時進食），並在跑步的第 7km 與第 12km 實地測試能量果膠的吞嚥與腸胃吸收狀況，及早找出最適合自己的補給策略。
