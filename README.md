# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個整合運動生理學與大數據分析的頂尖科學化馬拉松訓練系統。本系統以「數據導航，無痛增長」為核心理念，每日自動讀取並解析跑者的 Garmin 穿戴裝置數據。透過對心率、步頻、配速、HRV（心率變異度）與身體能量 (BB) 等關鍵生理指標的長期追蹤，動態評估跑力值 (VDOT)，精準校準個人化的訓練區間（包括輕鬆跑 E、馬拉松配速 M、乳酸門檻 T 與間歇跑 I）。本系統在給予生理系統「適度超載刺激」與確保「超補償恢復」之間取得黃金平衡，協助跑者在不累積傷痛的前提下，穩步提升有氧耐力與跑步經濟性，系統化地實現賽事目標，達成個人體能天花板的突破。本系統更著重於防傷機制的動態調整，當生理指標出現疲勞警訊時，會即時啟動避險機制，引導跑者在安全且科學的框架下前行。

## 📅 4. 本週訓練重點 (W30)

本週（2026-07-20 ~ 2026-07-26）為「進展期後段 (W28-W30)」的第三週（高峰鞏固週）。本週的訓練邏輯在於**「穩步擴大有氧容量、拉長乳酸門檻耐受時間，並以高步頻防禦機制與補給演練應對高峰里程突破」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：高峰鞏固期的負荷控制與防禦控速**  
    本週為備戰雪梨馬拉松進展期後段的最高峰週（W30 高峰鞏固週）。本週跑量預計達到 41 公里，為整個備賽週期以來的最高峰。由於上週累積了顯著的深層自主神經疲勞，加上多次課表出現配速失控的教訓，本週的訓練重點不是盲目追求速度突破，而是「在安全且受控的配速與心率區間內，穩定完成里程累積」。特別是週六將迎來備賽以來最長的 17km LSD 挑戰，因此本週的週一至週五必須執行「防禦性控速」，將所有非質量跑（輕鬆跑與恢復跑）的強度牢牢限制在有氧低心率 Z2/Z1 區間（心率 < 155 bpm），以促使身體產生超補償修復，為週末長跑儲備體能。

*   **二、 核心目標：17km 實戰補給演練與配速守恆**  
    1. **ONE TOKYO Global Virtual Race 4th 虛擬賽與補給演練**：週六（07/25）安排的 17km LSD 戶外長跑是整個備賽週期中單次最長里程突破。要求全程在戶外以 GPS 記錄執行，學員必須藉由此課表，演練雪梨馬拉松的實戰補給策略，規畫在跑步的 6km 與 12km 處定時補充能量膠與水分，測試並建立高運動量下胃部的吸收力與耐受度。  
    2. **重塑配速與心率紀律**：週二的 11km T 門檻跑（含 7km 門檻區段）必須將配速鎖死在 6'07"/km，不准快於 6'00"/km。週一恢復跑與週四輕鬆跑心率嚴格控制在 Z2 下限（心率 < 155 bpm）。  
    3. **維持下肢穩定力量**：週一（上半身）與週四（下半身）的重量訓練將重點放在單腳穩定性（如單腳硬舉、保加利亞分腿蹲），藉此強化臀中肌與後側鏈剛性，為週六 17km 長跑與雪梨馬拉松的起伏丘陵地形建立足夠的關節抗衝擊與離心煞車能力。

*   **三、 關鍵課表：T門檻跑的紀律重建與LSD長跑實戰**  
    1. **週二 11km T 門檻跑 (07/21)**：2km Z1/Z2 暖身 + 7km T 配速區段 (鎖死在 6'07"/km，HR: 171-180 bpm) + 2km Z1 緩跑。此課表旨在提高乳酸門檻閾值。執行時必須以前段 6'10"-6'15"/km 穩定切入，嚴防前 2km 的爆衝，確保能高質量地完成完整的 7km 門檻刺激。  
    2. **週四 8km Z2 基礎有氧跑 (07/23)**：1km Z1 暖身 + 6km Z2 基礎有氧 (配速 7'12"-8'02"/km，心率控制在 153-166 bpm) + 1km Z1 緩跑。此課表的核心在於「慢」。必須搭配 Garmin 的心率警示，將心率壓在 Z2 中下限（<155 bpm 尤佳），讓肌肉在低衝擊下建立工作肌群的毛細血管密度。  
    3. **週六 17km LSD 戶外跑 (07/25)**：1km Z1 暖身 + 15km Z2 有氧慢跑 (配速: 7'12"-8'02"/km，心率 153-166 bpm) + 1km Z1 緩跑。必須在戶外執行，模擬雪梨冬季強海風與緩坡地形。必須全程維持 170+ spm 步頻，藉此縮短步幅以減輕關節衝擊，並執行定點補給。

*   **四、 預期成效：乳酸清除率優化、跑步經濟性提升與補給胃部適應**  
    1. **乳酸清除能力提高**：通過週二精準的 7km 乳酸門檻刺激，提高乳酸轉運蛋白的活性，使身體能在乳酸堆積與清除之間達到更高的動態平衡點，進而提高在高強度下的配速耐受力，為雪梨馬拉松 Sub 4:30（目標均速 6'23"/km）奠定紮實的配速基石。  
    2. **有氧引擎深化與力學效率提升**：週四 8K 基礎跑與週六 17K LSD 的低心率有氧刺激，將促進左心室離心肥大，增加每搏輸出量，並在工作肌肉中建立更密集的毛細血管網路，提高跑步時的脂肪氧化比例，節省醣原。同時，透過 170+ spm 的高步頻維持，能有效將平均觸地時間控制在 270ms 以下，降低垂直振幅，提升跑步經濟性。  
    3. **補給系統建立**：週六長跑的兩次能量膠補給，能有效預防賽事後半程的醣原耗竭（撞牆期），並訓練胃部在高運動量下的吸收與排空能力，避免雪梨馬拉松後半段發生肌肉抽筋或腸胃不適。

*   **五、 執行建議：手錶配速/心率警示雙硬約束、下肢活動度管理**  
    1. **手錶警示硬約束**：跑前必須在 Garmin 965 上設定「配速警示（E跑下限 8'02"/km，上限 7'12"/km）」與「心率警示（Z2 上限 166 bpm，T跑上限 180 bpm）」。只要手錶震動警示，必須立即放鬆摆臂並縮小步幅，將速度降回預定區間，克服「體感配速」的盲目性。  
    2. **上坡與逆風的體感守恆**：週六戶外 LSD 遭遇上坡或強風時，不要刻意維持配速，而應以「心率守恆」為核心，放慢配速、縮短步長，將心率牢牢壓在 166 bpm 以下，下坡時利用重力順行，嚴防股四頭肌離心收縮度過受衝擊。  
    3. **睡前瑜伽防線**：高峰週的肌肉緊繃度極高，每日睡前瑜伽 (10-15 分鐘) 必須雷打不動地執行。瑜伽時間應提早至 22:00 前，專注於下犬式（拉伸小腿與足底）與鴿子式（釋放髖部與臀肌），並保證 23:00 前入睡，為低起點的 BB (32) 提供充足的充電時間。
