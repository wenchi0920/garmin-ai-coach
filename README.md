# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位
Garmin AI Coach 是一個結合運動生理學與大數據運算的核心教練系統。本系統秉持「科學訓練，精確校準」之精神，每天自動讀取並解析跑者的 Garmin `.fit` 運動紀錄檔案。透過對心率、步頻、配速、HRV 與壓力指數等關鍵生理指標的長期監控與深度分析，動態推算跑力值 (VDOT) 並修正個人化心率與配速訓練區間，自動產生並上傳每週科學化訓練課表。教練的終極目標是為跑者提供精準的數據導航，在「給予身體超載刺激」與「安全超補償修復」之間尋找黃金平衡，從而系統化、無痛地提升有氧天花板，穩健實現賽事目標，達成身心健康的全面成長。

## 📅 4. 本週訓練重點 (W27)
本週為「進展期中段 (W25-W27)」的最後一週 (W27)。本週的關鍵任務是**「穩定刺激乳酸門檻、落實高步頻防禦機制、以及在起伏地形上完成長距離耐力與補給演練」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：鞏固乳酸門檻耐力與雪梨賽道地形適應**
    備戰全馬的黃金法則在於有氧耐力與乳酸門檻（Lactate Threshold）的雙重刺激。本週作為進展期中段的收官週，訓練重點是將門檻跑的刺激時間進一步拉長，週三的 T 跑（乳酸門檻跑）質量段由上週的 4km 提升至 **5km**，這將穩定刺激學員的 T 區間，提升身體在高酸性狀態下的心肺代償與肌肉收縮效率。同時，本週日的 18km LSD 將正式移至有坡度起伏的戶外道路，模擬雪梨馬拉松的丘陵地形，為下肢肌肉注入耐衝擊的力量地基，並通過起伏路段來訓練股四頭肌的離心控制能力。

*   **二、 核心目標：T區配速穩定化與步頻突破 175 spm**
    1.  **乳酸門檻配速穩定 (T 配速 6'07"/km)**：在週三的門檻跑中，在 6'07"/km 配速下，將心率穩定控制在 Zone 4 (171-180 bpm)，擴大乳酸清除效率。週三實際執行均速 5'47"/km 偏快，雖然展示了極佳的心肺能力，但下肢負荷過重，後續需依課表配速執行。
    2.  **步頻防線建立 (175+ spm)**：徹底糾正 160-162 spm 的低步頻、大步幅跑姿。學員必須將「輕快著地、快速離地」作為最高指令，在所有輕鬆跑與 LSD 中將步頻強推至 175 spm 以上，藉此減少垂直振幅與每次著地的煞車效應。
    3.  **線上挑戰完賽**：於 7/5 前完成 ONE TOKYO GLOBAL Virtual Race 3rd 線上賽剩餘的 **13.88km** 里程，並藉由週末跑與長跑累積完成。

*   **三、 關鍵課表：9km T 門檻跑與模擬雪梨馬 18km 戶外 LSD**
    1.  **週三 9km T 門檻跑 (已於 7/1 執行)**：包含 2km Z1-Z2 暖身 + **5km T 區間主課表 (均速 5'47.6"/km, 心率 Zone 2-3)** + 2km Z1 緩跑。這將是本週強度最高、對心肺刺激最深的質量課表。雖然超前完成，但未來應嚴格跑重分流，防止一日多練的累積性損傷。
    2.  **週五 6km (已執行) 與週六 8km Z2 輕鬆跑**：作為週末長跑前的有氧鋪墊。配速必須嚴格鎖定在 **7'12" - 8'02"/km** 之間，心率壓在 Z2 區間，全力為週日長跑留存能量。週五下肢重訓取消。
    3.  **週日 18km 戶外 LSD**：包含 1km Z1 暖身 + 16km Z2/Z3 有氧慢跑 (配速: 7'12"-8'02"/km) + 1km Z1 緩跑。必須選擇有起伏的戶外地形，並隨身攜帶 3 包能量果膠與水，每 45 分鐘固定補給，演練腸胃吸收。

*   **四、 預期成效：提升跑步經濟性 (RE) 與控制後半程心率漂移**
    當步頻提升至 175+ spm時，將顯著降低跑步時的垂直振幅與關節受力。在 5km T 跑與 18km LSD 後半程，心率漂移現象將獲得緩解。隨著有氧引擎與下肢力量的轉化，學員在維持馬拉松配速跑動時，心肺呼吸將更為從容，肌肉能更效率地利用糖原，這將是 Sub 4:30 備賽最穩固的基石。

*   **五、 建議與警告：坡度設定、戶外實戰補給與提前黃金睡眠**
    1.  **跑步機坡度設定**：若因天候在跑步機上進行輕鬆跑，必須設定 1.0% - 1.5% 坡度，補足無風阻生理差額，使其更貼近戶外路感。
    2.  **防禦性配速控制**：週日 LSD 出發前 3 公里強制設定在 7'50" - 8'10"/km 的慢速熱身，避免起跑過快導致心率過早拉升。
    3.  **提前睡前伸展**：睡前瑜伽拉伸課表提前至 22:00 前完成，避免深夜高強度活動干擾睡眠，爭取每日 7.5 小時以上的睡眠，以利 HRV 與 BB 回升。
