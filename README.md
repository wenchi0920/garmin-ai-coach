3.  **關鍵課表：10K 賽前體感測試 (週四) 與 12km LSD (週六)**
    *   **週四測試**：這是一個「喚醒最大攝氧量 (VO2Max)」的過程。短暫的門檻跑能增加肌肉對氧氣的攝取效率，同時不累積過多疲勞。
    *   **週六長跑**：這是最後一次 10 公里以上的有氧累積。請務必嘗試「模擬戶外」，感受自然風對體溫與心率的影響。這是對您 4/18 賽事策略的最後預演。

4.  **預期成效：跑步經濟性與技術校準**
    *   **跑步經濟性 (RE)**：我們希望透過 175 spm 的步頻，讓您在 6:30 配速下的心率比以往降低 3-5 bpm。
    *   **技術定型**：習慣較小的步幅與較高的頻率，減少在 10K 比賽中後段因肌肉疲勞導致的步姿崩壞。

5.  **執行建議 (詳盡導引)**
    *   **「節拍器」的肌肉記憶**：所有的跑步訓練，請強制開啟節拍器 175 bpm。這不僅是為了保護腳趾，更是為了減少每一步的能量損耗。
    *   **環境適應**：本週至少安排一次戶外跑步。淡江大橋的海風會增加心血管壓力，您需要讓身體習慣這種環境擾動。
    *   **心態調整**：進入賽前減量期，身體可能會覺得沉重。請記住，這不是退步，而是身體正在儲備肝醣。保持自信，數據顯示您的底層引擎正在進化。
