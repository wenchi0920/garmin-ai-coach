## 1. 系統核心定位
Garmin AI Coach 是一個專為馬拉松跑者打造的智慧化訓練中樞。系統緊密結合 Garmin 穿戴裝置的龐大健康與運動數據，並透過 AI 深度學習技術自動分析每一筆訓練紀錄（.fit 檔案）與生理指標。本系統奉行「科學訓練，精確校準」的核心原則，精準掌握跑者的體能變化與疲勞趨勢，自動產生涵蓋跑步、專項肌力與修復瑜伽的個人化週期課表。我們的目標不僅是協助您無傷、高效地達成賽事成績，更期望在每一次的運動中，讓您體驗身心靈的成長與淬鍊，享受科學化訓練帶來的深遠影響。

**【專業恢復與營養銜接建議】**
面對今日與昨日正在進行的 ONE TOKYO VR 21.1km 虛擬半馬挑戰，這是對您有氧耐力、肌耐力以及心理韌性的一次綜合考驗。為了確保您能無痛完賽並在賽後迅速恢復，教練為您擬定以下專業的恢復與營養策略：
1. **排解隱形疲勞與筋膜深層照護**：本週密集進行的深蹲、硬舉及臥推等肌力訓練，加上週末的長距離跑，勢必會帶來局部的延遲性肌肉痠痛 (DOMS)。強烈建議在長跑結束後，立即進行「靠牆掛腿」10-15 分鐘，利用重力協助下肢靜脈血液回流。隨後，務必使用滾筒 (Foam Roller) 針對大腿前側的股四頭肌、臀大肌與小腿比目魚肌進行深層放鬆，加速乳酸排除與激痛點 (Trigger Points) 的舒緩。
2. **抗發炎營養與超補償飲食策略**：長時間的有氧與重訓後，身體組織會產生微小的發炎反應與氧化壓力。建議在未來幾天內，大幅增加深海魚類（富含 Omega-3）、深綠色蔬菜、藍莓與薑黃等抗氧化食材的攝取。此外，長跑後的「黃金 30 分鐘」內，請適度補充比例約 3:1 或 4:1 的碳水化合物與優質蛋白質（如巧克力牛奶或地瓜配茶葉蛋），為下週的訓練重新補滿肝醣庫，啟動肌肉修復的合成代謝。
3. **電解質重置與水分精密補充**：隨著初夏氣溫逐漸升高，每日的基礎飲水量應嚴格達到 2800cc-3000cc 以上。長跑挑戰中與挑戰後，請務必補充含有適當鈉、鉀等電解質的運動飲料或鹽錠，防止因大量排汗導致的神經傳導失衡或肌肉痙攣。這對於模擬 8 月雪梨馬拉松的實戰補給，具有絕佳的演練價值，請務必認真看待每一次的補給時機。
4. **體感導向與高步頻防線控制**：在長距離挑戰的後半段，當體力逐漸下滑時，請將專注力放在「穩定體感」與「高步頻」上。請強制要求自己將步頻維持在 175-180 spm 之間，避免因疲勞而跨大步幅。維持高步頻是縮短觸地時間、減少每一步衝擊力、保護大拇指舊傷、減輕膝蓋與阿基里斯腱負擔的最強防線。

## 📅 4. 本週訓練重點 (W22)
本週 (W22) 正處於雪梨馬拉松 14 週週期中的「基礎期 (Base Building)」，這是一個承先啟後的關鍵階段。歷經了一週的日本之旅，大腦與神經系統需要溫和且漸進的適應期。本週課表設計完美融合了「有氧基礎重建」、「神經肌肉喚醒」以及「週末長距離達標」的三大任務。透過精心編排的跑步與肌力雙重刺激，確保您在無痛無傷的前提下，迅速找回最佳狀態。以下為本週訓練的五大核心項目與詳盡解說：

**一、 訓練重點：平穩過渡與規律重塑 (Smooth Transition)**
從休閒旅遊模式切換回高紀律備賽狀態，本週首要重點在於「無痛回歸與平穩過渡」。我們不急於立刻堆疊極限跑量，而是透過深度的瑜伽修復（如 5/25），輕量的基礎有氧（如 5/26 的 6km 漸進跑），以及漸進式的肌力訓練（深蹲、硬舉），溫和地喚醒沉睡一週的肌肉群與心肺系統。馬拉松訓練是一場漫長的心理與生理戰，找回按表操課的「儀式感」，重建規律的生活作息，是為未來進階期打下磐石基礎的關鍵。

**二、 核心目標：有氧耐力重建與神經喚醒 (Aerobic Rebuilding)**
本週設定了雙重且明確的核心目標：第一，透過週末的 ONE TOKYO VR 21.1km 半馬分段挑戰，重新啟動強大的有氧引擎，擴張微血管網絡，增進粒線體密度，這對於應付雪梨馬拉松的長效續航力至關重要。第二，在週四 (5/28) 安插了 7km 的 T 門檻跑變速演練，透過成功突破 5'50" 的配速，有效喚醒快縮肌群，提升乳酸清除率，讓中樞神經系統維持對較快配速的敏銳度，確保速度感不因休息而流失。

**三、 關鍵課表：T 門檻跑、專項肌力啟動與週末實戰 (Key Workouts)**
本週的課表結構經過精心設計，旨在全面刺激不同的能量系統與肌肉群：
*   **過渡漸進跑與下肢重訓 (5/26)**：透過 Z2 穩定過渡至 M 配速，讓心率平穩上升。同時搭配深蹲與硬舉，強力徵召下肢肌群，強化股四頭肌與臀大肌，這是面對雪梨起伏坡度最穩固的力量來源。
*   **T 門檻跑與核心上肢重訓 (5/28)**：在跑步機上成功進行門檻配速的刺激，挑戰無氧區間；結合核心負重抬腿與臥推，確保軀幹在長跑後半段具備強大的抗旋轉與穩定度，減少無謂的動能耗損。
*   **短時綜合肌力與神經激活 (5/29)**：以輕量肩部、核心與深蹲訓練取代純跑步，為神經系統完成賽前最後的預熱與激活，不僅沒有造成肌肉過度耗損，反而讓雙腿在週末賽前充滿彈性。
*   **週末 VR 21.1km 挑戰 (5/30-5/31)**：這是一次期中考，鎖定在「Z2 穩定體感輸出」與「步頻嚴格控制」，在無痛的狀態下安全且高效地累積長距離訓練紅利。

**四、 預期成效：月跑量達標與核心肌力全面穩定 (Expected Outcomes)**
若能確實貫徹週末的長跑課表，您將成功突破 5 月份 100K 的月跑量基礎目標（目前執行前已達約 72.8 km，完賽後將穩健達標）。這不僅是數字上的勝利，更是有氧能力持續累積的證明。同時，本週密集加入的負重深蹲、硬舉與核心抬腿，已顯著強化了您的骨盆穩定度並提升了下肢剛性，這對於預防大拇指傷勢復發、應對雪梨賽道複雜路段，具有決定性的保護作用。順利完賽半馬虛擬賽，將是對近期有氧基礎重建的完美期中驗收。

**五、 執行建議：傾聽身體，嚴控步頻與模擬補給 (Execution Advice)**
身為經驗豐富的跑者，「謹慎觀察，絕不硬撐」是長跑的最高指導原則。我們追求的是長期的訓練品質與連貫性，而非單次的極限壓榨：
*   **警覺心率漂移 (Cardiac Drift)**：執行 VR 賽事時，若在目標 Z2 配速下發現心率異常升高 (例如持續 >166 bpm)，請立刻降速，給予心肺適應與散熱的空間，切勿強求配速。
*   **貫徹高步頻鐵律**：強制要求步頻維持在 175-180 spm。較快的節奏與較小的步幅，是縮短觸地時間、保護大拇指舊傷與減輕膝關節負擔的最有效策略。
*   **彈性動態調整**：若完成第一段 (5/30) 後極度疲憊，第二段 (5/31) 允許完全降速至 Z1 恢復跑甚至快走。這場挑戰的最終目的是「無痛完賽」與「累積里程」，請放下配速包袱。
*   **實戰補給演練**：請將這 21.1km 視為雪梨馬拉松的微型模擬考。請定時（例如每 5km 或每 30 分鐘）補充水分與碳水化合物，訓練腸胃在運動中的吸收能力，找出最適合自己的補給節奏。
