## 1. 系統核心定位
Garmin AI Coach 是一個深度結合運動科學與大數據分析的專屬自動化訓練生態系統。我們不僅僅是提供一份靜態的計畫，而是透過每日自動解析 Garmin 裝置生成的 `.fit` 檔案，即時監控跑者的心率變異度 (HRV)、睡眠品質、身體能量 (Body Battery) 以及每一趟訓練的動態指標。系統的核心定位在於「**科學訓練，精確校準**」，透過跑力指標 (VDOT) 的動態修正與個人化數據分析，將繁瑣的生理資訊轉化為具體的執行策略。無論是面對高溫挑戰、傷後復健，亦或是調整日常疲勞，AI Coach 都能彈性給予課表強度調整，協助您在追求目標賽事的過程中，有效管理疲勞、降低受傷風險，確保在達成目標的同時，兼顧身體修復與心靈成長，穩健朝向顛峰邁進。

進入本週中後段，您的健康數據與訓練重心呈現出修復與重啟的動態變化，顯示出身體在接受肌力訓練後的積極適應：

*   **6/12 (週五)：補償性有氧加速跑與心率管理**
    在週五深夜，雖然原定課表為修復瑜伽，您選擇進行了一次 2.78km 的補償性跑步機訓練。這次練習採取了遞增速跑策略，展現了極佳的心率管理能力：在末段加速至 5'56"/km (進入 Z5 配速) 的情況下，平均心率仍能維持在 159 bpm (Z2 區間內)。這證明您的基礎有氧能力 (Aerobic Base) 正在穩步提升。然而，必須注意的是，平均步頻僅 81 spm (換算約 162 spm) 偏低。此外，深夜進行高強度衝速容易過度刺激交感神經，可能會影響神經系統放鬆與睡眠深度，不利於迎接週末的長距離挑戰。這是一種生理上的「雙面刃」，未來在賽前備戰期應盡量避免在恢復日進行高張力衝刺。

### 【專業恢復與疲勞度評估建議】
綜合分析當前各項生理指標，您的身體正從肌力訓練後的「高張力」狀態，逐步過渡到「積極修復、準備承載長跑」的階段。舊傷已完全康復且狀態穩定，但面對高溫與進展期的訓練量，生理機能需要更謹慎的平衡：

1.  **堅守恢復日原則（Rest is Training）**：若當天安排為恢復日（如修復瑜伽），請盡量避免高強度的加速跑。身體的超量恢復發生在休息時，累積不必要的神經系統疲勞將直接影響週末關鍵課表（LSD）的執行品質與肌肉儲水能力。
2.  **提升步頻意識與跑姿經濟性**：目前的低步頻（約 162 spm）會增加每一次落地的觸地時間（GCT）與膝蓋、足底的衝擊負荷。無論是輕鬆跑或跑步機訓練，請刻意縮短步幅、增加步頻，目標穩定在 175-185 spm。這對於應對雪梨馬拉松的起伏賽道，減少後半程抽筋風險至關重要。
3.  **精準營養、睡眠與補水策略**：隨著夏日氣溫持續升高，脫水與電解質流失風險劇增。請確保全天候的電解質與水分補充，並在訓練後 30 分鐘內攝取優質蛋白質與碳水化合物。持續維持良好的睡眠品質（分數保持在 75 以上），以促進生長激素分泌與深層肌肉纖維修復。

## 📅 4. 本週訓練重點 (W24)
本週已正式進入雪梨馬拉松備賽的「進展期初期」，這是決定您整體耐力天花板與完賽自信心的關鍵過渡階段。面對高溫挑戰與上週的里程落後，本週的課表設計著重於重新建立長距離奔跑的耐心與身體的結構強度。

**一、 訓練重點：修補耐力斷層與優化跑步經濟性 (Endurance Rebuilding)**
由於前兩週受氣候與外務影響，里程數出現明顯斷層，特別是長距離訓練的缺席。本週首要任務是「有氧質量的補償」。我們不需要追求極速或打破個人紀錄，而是要確保每一公里的 Z2 輕鬆跑與長跑都能精準落在設定的心率與配速區間內。透過穩定的里程堆疊，喚醒身體對脂肪供能的依賴，並透過高步頻訓練優化跑步經濟性，讓心肺系統適應長時間的運作。

**二、 核心目標：力量與耐力的平穩轉化 (Strength-to-Speed Conversion)**
近期您在重量訓練（如負重深蹲 45kg 與硬舉 35kg）以及核心穩定上取得了顯著的進步，本週的核心目標是將這些「靜態力量」轉化為賽道上的「跑動力」。我們要在週末的 16km LSD 中測試雙腿承重能力，並維持 175 spm 以上的步頻。穩固的核心能確保雪梨馬賽道上的骨盆穩定，避免疲勞時的代償性傾斜與抽筋，這是達成 Sub 4:30 的結構性保障。

**三、 關鍵課表：16km 韌性挑戰與週末有氧擴充 (Key Sessions)**
本週後半段的訓練將直接決定當週的整體成效，請務必排除萬難執行：
*   **06/13 (週六) 6km Z2 輕鬆跑**：這是一次輕量級的有氧刺激，目的在於維持肌肉彈性與血液循環。請嚴格守在 7:12-8:02 /km 配速，心率控制在 Z1-Z2，絕不可超速，必須把寶貴的糖原能量保留給隔日。
*   **06/14 (週日) 16km LSD 長跑**：本週最核心的課表，也是雪梨馬的模擬跑。目標是「不間斷完成」，保持 Z2 高標心率。這不僅是一場體能考驗，更是一場心理韌性建設，必須完成對長距離的征服，彌補上週的缺口。

**四、 預期成效：心肺飄移抑制與恢復彈性提升 (Expected Gains)**
若能順利完成週末的連續計畫，預期您的心肺飄移現象（Cardiac Drift，即在相同配速下心率隨時間逐漸升高的現象）將獲得明顯改善，有氧效率獲得實質提升。同時，隨著充分的睡眠與肌力轉化，基礎代謝與微血管密度將獲得鞏固，使您未來面對更高強度的 T 跑或 I 跑訓練時，能恢復得更快，減少肌肉痠痛的滯留時間。

**五、 執行建議：精準補給與環境適應 (Execution Strategy)**
*   **熱適應與時段選擇**：台灣夏季高溫高濕，對心率影響極大。請務必將長跑安排於清晨（5:30前起跑）或傍晚執行。以「心率」為最高指導原則，若心率飆升，請果斷降速，不強求配速。
*   **補給實戰演練**：週日 16km 長跑務必實戰演練補水與能量膠。請在第 7km 與第 12km 嘗試攝取，讓腸胃提早適應賽事高強度下的吸收狀態，預防腸胃不適。
*   **步頻即時監控**：隨時提醒自己加快交換腳的速度，將步頻穩定在 175 spm。在疲勞期（12km 後），跑姿最容易走樣，此時更需專注於核心收緊與雙手擺臂的帶動。
