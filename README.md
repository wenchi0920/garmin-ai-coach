## 📅 4. 本週訓練重點 (W23)
本週 (W23) 是雪梨馬拉松倒數 13 週，我們正式開啟「進展期初期 (Strengthening Phase)」。六月的戰略核心是「鞏固有氧底盤，喚醒門檻速度，強化下肢剛性」。以下是針對本週目標的詳盡解析：

**一、 訓練重點：乳酸門檻刺激與基礎有氧擴張 (Threshold & Endurance)**
本週採取「雙軸並進」策略。第一軸是週四的「T 門檻刺激」。乳酸門檻跑的目的是訓練身體在乳酸堆積邊緣的「耐受」與「清除」能力。6/4 的跑步機訓練雖然里程較短，但配速已逼近目標且心率表現優異。這為我們在夏日維持強度提供了信心。第二軸是週日的「16km LSD 長距離慢跑」，這是六月最重要的有氧引擎擴張練習。透過長達兩小時的持續跑步，我們能促進粒線體增生，強化毛細血管網，讓身體在長距離後期仍能高效利用脂肪作為能源，這對於全馬 sub 4:30 來說是核心基本功。

**二、 核心目標：重建下肢抗疲勞能力與核心穩定 (Leg Strength & Core Stability)**
為了對抗雪梨馬拉松起伏多變的賽道，本週已安排高品質的負重深蹲與硬舉。這不僅是為了絕對力量的增長，更是為了建立肌肉的「剛性 (Stiffness)」與「離心收縮 (Eccentric Contraction)」耐力。跑步機上的跑步雖然效率高，但缺乏路面的衝擊變化。因此，我們透過 45kg 深蹲來模擬下坡時大腿承受的衝擊。強大的核心與下肢刚性是防止 30km 後姿勢崩潰、步幅縮減的最強防線。

**三、 關鍵課表：T 門檻測驗與長距離耐力考驗 (Key Workouts)**
*   **W23-6km T 門檻跑 (06/04)**：數據顯示您的心肺天花板很高。在 6'15" 配速下心率僅 153 bpm，這說明您的潛力巨大。下次執行時，請嘗試將這份心率寬裕度轉化為配速的穩定性。
*   **W23-16km LSD 長跑 (06/07)**：本週的終極考驗。這不只是里程的累積，更是心理層面的耐力訓練。請嚴格鎖定 Z2 區間 (7:12-8:02/km)，關注呼吸的節律感，這也是完成 ONE TOKYO VR 21K 挑戰剩餘里程的最佳契機。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能紀律執行本週課表，您的生理適應將獲得顯著提升：
1. **心肺連動優化**：T 門檻跑後，您會發現在執行輕鬆跑時心率會更低，同樣配速下的「體感疲勞度 (RPE)」會明顯下降。
2. **下肢彈性提升**：重量訓練與長距離跑步的結合，能強化足弓與阿基里斯腱的彈性。這意味著您在每一步推進時都能節省更多的能量，提升「跑步經濟性 (Running Economy)」。

**五、 執行建議：高步頻防線與夏日補給演練 (Execution Advice)**
*   **嚴守高步頻防線**：請在所有跑步課表中刻意維持 175 spm 以上的高步頻。高步頻能降低觸地時間，減少落地時身體重心垂直位移產生的能量浪費，更重要的是能大幅降低對關節與骨骼的衝擊負荷，確保無傷備賽。
*   **實戰補給演練**：週日 16km 長跑預計耗時兩小時左右，這是一個練習「腸胃適應力」的好機會。請務必模擬賽事節奏進行補給，找出最適合您的能量膠品牌與攝取頻率。
*   **環境應對策略**：夏日濕熱易引發心率漂移。若戶外氣溫超過 28 度，請主動將目標配速放慢 10-15 秒，改以「心率區間」為導向。記住，夏日訓練的目的是「生理刺激」，而非死守數據。
