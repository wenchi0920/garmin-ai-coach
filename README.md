## 1. 系統核心定位
Garmin AI Coach 是一個深度結合運動科學與大數據分析的專屬自動化訓練生態系統。我們不僅僅是提供一份靜態的計畫，而是透過每日自動解析 Garmin 裝置生成的 `.fit` 檔案，即時監控跑者的心率變異度 (HRV)、睡眠品質、身體能量 (Body Battery) 以及每一趟訓練的動態指標。系統的核心定位在於「科學訓練，精確校準」，透過跑力指標 (VDOT) 的動態修正，將繁瑣的數據轉化為具體的執行策略。無論是面對高溫挑戰還是傷後復健，AI Coach 都能彈性調整課表強度，確保跑者在追求目標賽事卓越表現的同時，能有效管理疲勞、降低受傷風險，最終達成身心靈平衡的巔峰狀態。

## 📅 4. 本週訓練重點 (W24)
本週 (W24) 已正式進入雪梨馬拉松備賽的「進展期初期」，這是決定您整體耐力天花板與完賽自信心的關鍵階段。

**一、 訓練重點：修補耐力斷層與優化跑步經濟性 (Endurance Rebuilding)**
由於前兩週受氣候與傷後初癒影響，里程數出現了明顯的斷層，特別是上週末長距離訓練的缺席。本週的首要核心工作是「有氧質量的補償」。我們不需要追求極速或瘋狂加量，而是需要確保每一公里的 Z2 輕鬆跑與 T 門檻跑都能精準地落在設定的配速與心率區間內。透過穩定的里程堆疊，重新喚醒身體對脂肪供能的依賴，並透過高步頻的維持，優化跑步經濟性，減少多餘的垂直振幅能量損耗。

**二、 核心目標：力量與耐力的平穩轉化 (Strength-to-Speed Conversion)**
您近期在重量訓練上取得了顯著的進步，這是一把雙面刃。本週的核心目標是將這份「死力量」轉化為「跑動力」。我們要在週末的 16km LSD 中，測試雙腿在長時間承重下，是否能維持住 175 spm 以上的步頻。穩固的核心能確保在雪梨馬 313m 爬升的賽道上，您的骨盆不會因為疲勞而傾斜，這是避免抽筋與維持末段配速的關鍵。這兩天的核心修復瑜伽已經為此打下良好基礎。

**三、 關鍵課表：Z2 有氧擴充與 16km 韌性挑戰 (Key Sessions)**
本週後半段的訓練將直接決定當週的訓練成效，請務必將以下課表視為第一優先：
*   **06/11 (週四) 8km Z2 基礎有氧 (需補足)**：穩定心率於 153-166 bpm。這是修復神經疲勞、擴大有氧引擎規模的最好方式。請專注於「放鬆」，想像自己是在雲端奔跑，減輕落地衝擊。
*   **06/13 (週六) 6km Z2 輕鬆跑**：作為長跑前的熱身，目的在於維持肌肉彈性。配速請嚴格守在 7:12-8:02 /km，絕不超速，將能量完全保留給隔日的挑戰。
*   **06/14 (週日) 16km LSD 長跑**：這是本週最神聖的課表。若氣溫過高，請務必在清晨 5:30 前出發。目標是「不間斷完成」，這是一場針對雪梨馬拉松的模擬跑。保持 Z2 高標心率，哪怕最後 2km 掉速，也要在心理上完成對長距離的征服，建立不可動搖的自信心。

**四、 預期成效：心肺飄移抑制與恢復彈性提升 (Expected Gains)**
若能堅定完成本週計畫，預期您的心肺飄移 (Heart Rate Drift) 現象將在下週明顯好轉。在固定的 Z2 配速下，後半程心率上升的斜率會變得更為平緩，這意味著心肺效率的實質提升。同時，藉由充足的睡眠與肌力訓練後的轉化，您的基礎代謝能力與耐酸能力 (乳酸門檻) 將獲得實質性的鞏固，使您在面對高強度挑戰時能恢復得更快。

**五、 執行建議：精準補給與環境適應 (Execution Strategy)**
*   **補給演練**：週日 16km 長跑時，請務必實戰演練「每 45-60 分鐘補水、每 8km 補充能量膠」的節奏。雪梨馬賽道起伏大，腸胃負擔會比平路大，提早適應賽事的補給節奏對實戰至關重要。
*   **步頻監控**：請將 Garmin 錶面設定「步頻」為大欄位。當疲勞產生時，步頻往往會不自覺地下降，請隨時提醒自己加快交換腳的速度，維持在 170-175 spm 以上。
*   **心態調適**：不要被先前 T 跑或 LSD 嘗試的挫敗所困擾。進展期初期的起伏是非常正常的，重點在於「持續出現在跑道上，執行計畫」。保持正向心態，每一次的努力都在拉近你與終點的距離。
