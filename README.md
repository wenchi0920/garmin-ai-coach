## 📅 4. 本週訓練重點 (W23)
本週 (W23) 是迎向 8 月雪梨馬拉松的倒數 13 週，我們正式開啟備賽週期的「進展期初期 (Strengthening Phase)」。我們的戰略核心為「鞏固有氧底盤，喚醒門檻速度，強化剛性」。本週課表設計緊扣以下五大項目：

**一、 訓練重點：乳酸門檻刺激與基礎有氧擴張 (Threshold & Endurance)**
本週採取「雙軸並進」的策略。第一軸為週四的「6km T 門檻跑」，這項速度耐力指標要求以 6:07/km 巡航，強迫身體提升清除血乳酸效率；第二軸是週日的「16km LSD 長距離慢跑」，旨在徹底擴展有氧引擎容量，讓身體在長時間運動下更有效率地燃燒脂肪。

**二、 核心目標：重建下肢抗疲勞能力與核心穩定 (Leg Strength & Core Stability)**
為了對抗雪梨馬拉松起伏多變的賽道，本週前段已安排高質量重訓。未來的重訓核心不僅是單純力量增長，更著重於大腿「離心收縮」能力及骨盆周圍穩定性，能大幅降低比賽後半段因肌肉疲乏產生的代償風險。

**三、 關鍵課表：T 門檻測驗與長距離耐力考驗 (Key Workouts)**
*   **W23-6km T 門檻跑 (06/04 週四)**：本週速度指標。包含 2km 暖身、3km 穩定維持 T 配速 (6:07/km) 的主課表，以及 1km 緩跑冷卻。若遇高溫導致心率進入 Z5，請主動改以心率區間主導。
*   **W23-16km LSD 長跑 (06/07 週日)**：本週耐力與心理指標。請嚴格鎖定 Z2 區間 (配速 7:00-7:30/km，心率 153-166 bpm)。這也是完成 ONE TOKYO VR 21K 挑戰剩餘里程的最佳時機。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若紀律地完成本週課表，生理適應將顯著提升。T 門檻跑將促進微血管增生，讓維持 M 配速 (6:33/km) 時體感更加游刃有餘；重量訓練與 LSD 的結合則能鍛鍊慢縮肌的極限耐力，帶來強大的下肢剛性。

**五、 執行建議：高步頻防線與夏日補給實戰演練 (Execution Advice)**
*   **嚴守高步頻防線**：請在跑步課表中刻意維持 175 spm 以上高步頻，以小步幅降低觸地時間（降至 270ms 以下），大幅減少落地衝擊。
*   **實戰補給演練**：週日的 16km 長跑預計耗時兩小時，請務必在出發前及起跑後每 45 分鐘進行能量膠與水分攝取測試。
*   **氣候與心率應對**：夏日濕熱氣候極易引發心率漂移，請「以心率為尊」，心率超標時主動降速。
