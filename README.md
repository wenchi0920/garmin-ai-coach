**近兩日數據動態與專業診斷**：

1.  **4/1 質量日 (T) 的強度失控診斷**：
    在 4/1 的 2x10' 門檻跑中，您展現了令人驚嘆的鬥志，第二組跑出了 5'11"/km 的極速。但從科學監控的角度看，這是一次「失敗的質量跑」。您的平均心率達 153 bpm，最大心率飆升至 173 bpm (Z5)。這已經不是在訓練乳酸代謝（T），而是在進行無氧間歇（I）。在高強度下，身體會產生大量的乳酸堆積與皮質醇，這會抑制免疫系統並延緩組織修復。您的「引擎」轉速已經過載，而「散熱系統」顯然跟不上。這種「灰區訓練」的誘惑很大，因為它讓你感覺很快，但對於馬拉松所需的長效耐力提升效果有限，反而增加了疲勞累積。

2.  **4/2 「主動恢復」的教科書級執行**：
    令人欣慰的是，您在 4/2 展現了高度的紀律。中午的肌力訓練避開了下肢負擔，改以核心為主；傍晚的 5.62km 恢復跑更是關鍵，平均心率僅 127 bpm。這種「超低強度跑」的作用不在於訓練心肺，而在於透過輕微的肌肉泵動加速血液循環，帶走 4/1 殘留的代謝廢物。這次執行成功抵銷了昨日的冒進，讓身體重新回到平衡狀態。

## 📅 本週訓練重點：W14 基礎累積期 (03-30 ~ 04-05)

1.  **訓練重點：嚴格強度區分與心率控管**。
    本週的核心是「定義邊界」。4/1 的數據證明了您具備速度，但缺乏對強度的自律。本週剩餘課表必須嚴格遵守「輕鬆跑不超過 143 bpm」的鐵律。我們要的是有氧效能的累積（線粒體密度提升），而非速度的證明。

2.  **核心目標：建立有氧底層與提升步頻**。
    在不超過心率 143 bpm (Z2) 的前提下完成所有輕鬆跑，並透過技術練習將步頻提升至 175 spm 以上。這不僅是為了節能，更是為了透過減少觸地時間來保護脆弱的腳趾。

3.  **關鍵課表：10km Z2 基礎耐力 (04-04 Sat)**。
    這是本週最重要的「地基課表」。目標是心率絕對不准超過 143 bpm。如果為了維持心率而配速掉到 8'00"/km，也請務必接受。這是在「修理引擎的冷卻系統」，沒有強大的冷卻系統，您的雪梨 Sub 4:30 夢想將在 30 公里處破滅。

4.  **預期成效：降低「有氧脫鉤」比率**。
    透過嚴格控速，我們預期在長跑末段的心率漂移能控制在 5% 以內。這將提升您利用脂肪供能的效率，為 4/18 的淡江大橋賽事儲備能量，而非在那之前耗盡意志力。

5.  **執行建議**：
    *   **「步頻優先」戰術**：在跑步機上使用節拍器，設定為 175 bpm。哪怕配速掉到 8'30"/km，也要維持輕快的腳步節奏。想像腳下踩著薄冰，要以最輕盈的觸地感完成每一次邁步。這不僅是為了速度，更是為了透過優化跑步經濟性（Running Economy）來節省 10% 以上的體能，這在全馬後半段是生死關鍵。
    *   **心理建設：慢得下來才是真強者**。忘記配速，只看心率。在馬拉松的國度裡，「慢得下來」比「衝得出去」更需要勇氣。本週六 10km 請模擬賽事，每 3km 補水一次，進行「腸胃訓練」。
    *   **散熱環境優化**：室內跑步務必開啟大風扇。數據顯示您的心率飆升可能與體溫調節壓力有關，充足的散熱能有效降低 5-10 bpm 的心率漂移。
    *   **跑後黃金 30 分鐘修復**：每次訓練後，請立即進行 5 分鐘的腳踝與足底放鬆。針對剛痊癒的腳趾，建議進行「腳趾抓毛巾」的復健動作，強化足底小肌肉。
