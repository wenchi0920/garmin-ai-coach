# Running Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位

Garmin AI Coach 是一個整合運動生理學與大數據分析的頂尖科學化馬拉松訓練系統。本系統以「數據導航，無痛增長」為核心理念，每日自動讀取並解析跑者的 Garmin 穿戴裝置數據。透過對心率、步頻、配速、HRV（心率變異度）與身體能量 (BB) 等關鍵生理指標的長期追蹤，動態評估跑力值 (VDOT)，精準校準個人化的訓練區間（包括輕鬆跑 E、馬拉松配速 M、乳酸門檻 T 與間歇跑 I）。本系統在給予生理系統「適度超載刺激」與確保「超補償恢復」之間取得黃金平衡，協助跑者在不累積傷痛的前提下，穩步提升有氧耐力與跑步經濟性，系統化地實現賽事目標，達成個人體能天花板的突破。本系統更著重於防傷機制的動態調整，當生理指標出現疲勞警訊時，會即時啟動避險機制，引導跑者在安全且科學的框架下前行。

## 📅 4. 本週訓練重點 (W29)

本週（2026-07-13 ~ 2026-07-19）為「進展期後段 (W28-W30)」的第二週。本週的訓練邏輯在於**「穩步擴大有氧容量、拉長乳酸門檻耐受時間，並以高步頻防禦機制應對里程突破」**。以下為本週訓練的五大核心項目：

*   **一、 訓練重點：疲勞累積期的應變與長跑減量避險**
    本週是雪梨馬拉松備賽進展期後段的關鍵週。核心的訓練重點在於透過長跑 LSD 課表擴大心室容積，並以 T 心率區間的乳酸門檻跑強化肌肉對乳酸的清除能力。然而，由於週一至週四在實際執行過程中出現嚴重的配速失控（恢復跑與基礎跑均跑成 T 配速、門檻跑跑成 I 配速爆衝），導致身體迅速累積疲勞赤字，HRV 指標跌破平衡基線。因此，週六的訓練重點從原定的「盲目追求 16km 里程突破」主動調整為「防傷與控速下的減量防禦」，最終以 7.01km 的 Zone 2 慢跑完成，成功給予身體有氧刺激，又規避了深層疲勞下的運動受傷風險。
*   **二、 核心目標：以高步頻防禦與精準配速重建有氧引擎**
    1.  **高步頻防守（170-175 spm）**：全體跑步課表均搭配手錶步頻警示，強制將雙邊步頻穩定在 170-175 spm 之間。透過「高步頻、小步幅」的跑姿，縮短觸地時間，控制垂直振幅在 75mm 以下，從而最大程度降低每次落地的煞車效應與關節衝擊力。
    2.  **有氧心率守恆（Z2: 153-166 bpm）**：在有氧跑（週四、週六）中，嚴格將心率限制在 Zone 2 以內。週六的減量跑中，心率成功控制在平均 154 bpm，糾正了前幾次爆心率的現象，重新回歸科學化有氧基礎建構。
    3.  **肌肉力量打底**：週四中午順利完成了 34 分鐘的下肢重訓（深蹲最高 50kg、硬舉最高 40kg、腿推 30kg），累計搬運重量 950kg，旨在強化臀大肌與大腿後側肌群，為雪梨馬拉松起伏多坡的地形做下肢離心力量與抗拉伸的準備。
*   **三、 關鍵課表：T門檻跑的教訓與LSD調整實戰**
    1.  **週二 11km T 門檻跑（實際完成 5.62km）**：原定 7km 的 T 配速主課表，因學員以前段 5'31"-5'44"/km 的 I 配速爆衝，導致乳酸迅速堆積而被迫提前中止。這提醒我們，質量跑的核心是「穩定在目標強度區間（6'07"/km）」，嚴禁超速。
    2.  **週四 7km Z2 基礎跑（實際完成 7.00km）**：當日晚上跑步機執行時配速嚴重超標，Lap 2-6 配速達 6'04"-6'09"/km（T配速），心率飆升至 165+ bpm，造成心血管系統過度負荷，導致後半段被迫步行。
    3.  **週六 16km LSD 戶外跑（實際減量為 7.01km 戶外跑）**：考量週五 Readiness 僅 37 且 HRV 持續不平衡，今日實戰調整為 7.01km。學員全程控制配速，最大心率 172 bpm（均心率 154 bpm）守護有氧 Z2 區間，步頻穩定，成功達成有氧超補償的防禦性調整。
*   **四、 預期成效：擴張心室容積、提高毛細血管密度與優化跑步經濟性**
    1.  **有氧引擎深化**：儘管本週課表在量與質上均有主動與被動的調整，但週四與週六的 Zone 2 心率刺激，能有效促進心室心肌進行離心肥大，增加每搏輸出量，並在工作肌肉中建立更密集的毛細血管網與線粒體，提高脂肪氧化能力。
    2.  **跑步技術優化**：透過在日常跑步中刻意練習 170+ spm 步頻，學員的平均觸地時間預計能控制在 270ms 以下，垂直振幅降至 75mm 以下，降低股四頭肌的離心收縮壓力，提高跑步經濟性。
*   **五、 執行建議：精準配速警示、熱適應管理與睡前修復紀律**
    1.  **手錶警示設定**：在 Garmin 965 上設定配速警示（E跑：7'12"-8'02"/km）與心率上限警示（Z2：166 bpm），一旦超速或心率超標手錶會即時震動，強制跑者降速。
    2.  **防範熱壓力（熱應激）**：7 月中旬氣候悶熱，高溫是心率漂移的主因。跑步機跑時必須開啟風扇與冷氣，戶外跑則務必安排在清晨 05:30-07:30 或傍晚 18:00 以後，並隨身攜帶水袋，隨時補水。
    3.  **調整睡前伸展時間**：為避免深夜運動干擾睡眠，睡前放鬆瑜伽應提前至 21:30 - 22:00 之間進行，確保 23:00 前關燈入睡，拉長黃金睡眠充電窗口。
