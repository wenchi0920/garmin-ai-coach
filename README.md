3.  **關鍵課表：10K 體感測試 (週四) 與 12km LSD (週六)**

4.  **預期成效：跑步經濟性 (RE) 的質變**

5.  **執行建議 (詳盡導引)**

    *   **環境模擬與裝備測試**：週六的戶外長跑請穿著 4/18 預計參賽的跑鞋。若在戶外遭遇逆風，請專注於「降低重心、增加步頻」，而非與強風硬碰硬，這是節省體力的關鍵戰術。
