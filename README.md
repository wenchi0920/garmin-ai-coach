## 1. 系統核心定位
Garmin AI Coach 是一個結合 Garmin 穿戴裝置數據與 AI 智慧分析的專屬科學化訓練計畫書。我們秉持「科學訓練，精確校準」的核心理念，透過自動化提取並深度解析您的每一筆訓練檔案（.fit），動態監控跑力指標（VDOT）、疲勞累積及恢復狀態。本系統根據個人生理數據與賽事目標，自動規劃涵蓋跑步、專項肌力與瑜伽修復的週期化課表，並提供即時訓練回饋與每週調整。我們致力於在最小化受傷風險的前提下，幫助跑者精準推升體能，達成個人最佳表現，同時在運動過程中體驗身心靈的豐富成長。

### 【旅遊後過渡與專業恢復建議】
身為您的專屬教練，我必須提醒您，旅遊中的「靜態與低強度累積疲勞」與訓練中的「動態高強度疲勞」在生理機制上截然不同。每日長時間的穿梭會讓小腿腓腸肌與足底筋膜長時間處於半收縮狀態，容易導致回流不暢。針對您即將結束旅程並重返賽道，我提供以下恢復指南：

1. **下肢淋巴與靜脈回流管理**：強烈建議在今明兩晚回到飯店後，執行「靠牆掛腿」15-20 分鐘。透過改變重力方向強迫神經系統放鬆下肢張力。
2. **冷熱刺激的血管體操**：請善用日本浴場文化。透過 40-42 度的熱水浸泡 5 分鐘擴張血管，隨後以冷水沖洗小腿 1 分鐘，重複 3 個循環，加速深層肌肉廢物排除。
3. **筋膜張力重置**：利用飯店提供的按摩球或是礦泉水瓶，針對腳底進行深層滾動按壓；並透過弓箭步或蜥蜴式，拉長大腿前側與髖關節前方。
4. **睡眠與營養的協同效應**：確保攝取足夠的優質蛋白質與維生素 C。睡前進行「4-7-8 呼吸法」，幫助神經系統從白天的興奮狀態切換到深層修復狀態。
5. **心理韌性的超補償**：旅遊帶來的多巴胺分泌能有效重置大腦的疲勞閾值。當您結束旅程回到賽道時，您對長跑的渴望將會成為極高明的心理戰略。

## 📅 4. 本週訓練重點 (W21)：戰略性減量與身心重置
本週 W21 為計畫中的「過渡週」與「減載週 (Unloading Week)」。結合您的日本旅遊行程，訓練的核心不在於跑量的累積，而在於透過主動恢復與適度刺激，保持有氧感知，為回國後的進階訓練積蓄動能。以下是本週的五大核心項目：

**一、 訓練重點：戰略修復與主動恢復 (Strategic Recovery)**
經過連續四週密集的基礎期重建訓練，肌肉與中樞神經系統都已承受了不小的壓力。本週我們透過日本旅遊的環境轉換，刻意執行「戰略性減載」。主動恢復不僅是停止高強度的跑步，更是利用旅遊期間的大量低強度有氧（步行）來促進血液循環。此階段的重點是讓交感神經從備戰狀態平穩切換至副交感神經主導的修復狀態，有效預防因長期累積疲勞而誘發的過度訓練症候群。

**二、 核心目標：維持心肺感知與肌肉記憶 (Maintain Muscle Memory)**
即便在減載週，核心目標也是在旅遊期間透過極短距離的輕鬆跑或徒手肌力，喚醒肌肉神經連結。我們追求的是「保持神經對路面的敏銳度」，特別是 175+ spm 高步頻意識。在偶爾的晨跑中，不需要在乎配速，而是專注在腳步的輕盈感、落地位置及呼吸順暢度。這些記憶將確保回國後能夠第一時間接軌常規進階課表。

**三、 關鍵課表：T門檻刺激與深度修復瑜伽 (Key Workouts)**
本週的課表設計呈現「一頭一尾」的特徵：
1. **7km T 門檻跑（出國前執行）**：這是本週唯一的強度指標，利用 T 配速（6:07/km，Z4）刺激快縮肌群，同時檢驗乳酸清除能力。
2. **5km Z2 旅遊探索跑**：抵達日本後，以 Z2 輕鬆心率探索異國街道。這是無壓力的感知跑，讓跑步回歸純粹的樂趣。
3. **每日徒手核心與修復瑜伽**：睡前必須執行下犬式、蜥蜴式以及靠牆掛腿，拉伸緊繃的腿後腱與小腿肌群；搭配深蹲與死蟲式等核心動作，確保肌群不因旅遊鬆懈。

**四、 預期成效：體能超補償與傷勢穩固 (Expected Outcomes)**
預期這一週的彈性調整能收穫顯著的「超補償 (Supercompensation)」。當下週一重啟訓練時，您的身體能量指數將觸及高點，HRV 展現極佳的自律神經平衡。左腳大拇指的受傷部位在低壓環境下完成更深層的細胞重建；心理上，對長距離跑步的渴望會達到頂峰。這些改變都將成為接下來雪梨馬拉松高強度課表的堅實後盾。

**五、 執行建議：傾聽身體，靈活彈性 (Flexible Execution)**
在異國旅遊情境下，務必堅守「不勉強、不補課」的原則。旅遊行程就是對體力的消耗，請將「傾聽身體的聲音」放在首位。如果當天步行讓雙腿感到腫脹，請果斷取消探索跑，改為在飯店內進行 15 分鐘的瑜伽伸展或享受溫泉。一週的彈性變通非但不會讓體能倒退，反而是前進三步前必須的戰略後退。享受美食與美景，快樂的心態就是最高效的恢復劑。
