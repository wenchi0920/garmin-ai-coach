## 📅 本週訓練重點：W20 基礎期重建圓滿週

**1. 訓練重點：有氧效能固化與乳酸門檻突破**
本週是進入下週日本旅遊「減量期」前的關鍵高峰。核心任務是將 Z2 輕鬆跑的頻率穩定在 170 spm 以上，並透過週三的 T-Pace 課表重新喚醒身體代謝乳酸的能力。這將為 6 月份的「馬拉松配速期 (M-Phase)」打下穩定的生理基礎。

**2. 核心目標：穩定與頻率的平衡**
*   **技術指標**：在週三 4km T-Pace 中維持 180 spm 的步頻，且觸地時間目標壓在 260ms 以下。
*   **生理指標**：週六 12km 長跑心率不得超過 Z2 上限 (166 bpm)，模擬雪梨馬後半程的耐力穩定性。
*   **進度目標**：完成本週後，5 月累計跑量應達 60km 以上，達成月目標的 60%。

**3. 關鍵課表解析：**
*   **05/13 (三) 4km T-Pace 間歇**：這是本週最高強度的課表。目標配速 6'07"/km。這堂課的意義在於「提升速耐力」，讓 6'23" 的目標賽速在未來體感上變得更輕鬆。
*   **05/16 (六) 12km 長跑 (Long Run)**：本週期里程最高峰。重點在於 10km 後的節奏維持，測試在基礎期重建後，身體對超過 90 分鐘運動的能量供應效率。

**4. 預期成效：**
完成 W20 後，您的 VDOT 預計會因心肺與技術的優化而產生「隱形提升」。即使數據上仍維持 33，但「配速/心率比」將變得更優質。這將使您下週在日本旅遊時，即使減少跑量，身體也能維持極佳的代謝彈性。

**5. Execution 執行建議：**
*   **強制節拍器**：本週所有跑步，請手錶設定 175 bpm 節拍器。
*   **戶外優先**：為因應雪梨馬的真實地形，週三與週六課表請務必在戶外執行，避免跑步機對步態的誤導。
*   **肌力強化**：週二與週五的徒手肌力請著重「單腳穩定性」，這對預防長距離跑步後的技術變形至關重要。
