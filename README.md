# 🏃‍♂️ Garmin AI Coach - 專屬科學化訓練計畫書 (Wenchi)

## 1. 系統核心定位
Garmin AI Coach 是一個深度結合運動科學、大數據分析與自動化課表生成的專屬教練系統。本系統以「**科學訓練，精確校準**」為絕對核心價值，每日自動解析跑者的 Garmin `.fit` 運動紀錄，透過監測心率、步頻、配速、HRV 與壓力指數等關鍵生理數據，即時評估跑者的疲勞指數、恢復狀態與適應程度。

藉由動態修正跑力指標 (VDOT)，系統能將複雜的運動生理數據轉化為個人化的週期化訓練課表。AI Coach 的目標不僅是提升成績，更在於透過數據導航，平衡「訓練刺激」與「身體修復」，確保跑者在穩步邁向目標的同時，最大限度地降低運動傷害風險，達成身心健康的全面成長。

**【當前跑力分析與備賽狀態】**
目前系統校準後的跑力指標為 **VDOT 33**。基於此基準，我們的訓練區間定義如下：
*   **輕鬆跑 (E/Z2)**：7:12 - 8:02 /km (建立強大的有氧地基)
*   **馬拉松配速 (M)**：6:33 /km (賽事目標配速基準)
*   **乳酸門檻 (T)**：6:07 /km (提升抗乳酸能力)
*   **間歇跑 (I)**：5:38 /km (擴張最大攝氧量)

目前您的 **VO2 Max 穩定在 45**，狀態標示為「**生產中 (Productive)**」。這是一個非常正面的訊號，代表您的身體正在適應目前的訓練負荷。然而，目前的挑戰在於：Sub 4:30 所需的配速 (6'23"/km) 實際上略高於您目前的 M 配速 (6'33"/km)。這意味著在接下來的 74 天中，我們必須透過結構化的「進展期」訓練，將您的 VDOT 從 33 推升至 35 以上。雪梨馬拉松的賽道特色在於其起伏的地形，總爬升超過 300 公尺，這對下肢的肌耐力與心肺系統的穩定性提出了極高要求。目前您已平穩度過「基礎期」，正式轉入「進展期中段」。這個階段的核心任務是將先前建立的基礎力量轉化為高效率的「跑步經濟性」，確保在面對雪梨大橋與市區丘陵時，仍能維持穩定的步頻與配速，最終實現 Sub 4:30 的壯舉。

### 【近兩日 (6/16-6/17) 生理數據與重訓表現分析】
進入 W25，您的身體數據表現出極強的潛力，但也暴露出「強度執行力」的斷層：
*   **06/16 (週二)：** 這是近期訓練的高標。中午完成兩階段肌力訓練，臥推 35kg 與核心抬腿 50kg 的表現極佳，顯示神經肌肉系統已準備好迎接高負載。晚間 5km Z2 跑步更是驚艷，在 6'53"/km 的配速下，心率僅 154 bpm（Z2 底端）。這顯示您的「有氧天花板」正在上移，低心率高配速的效率初步顯現。

### 【專業恢復建議與傷勢監控】
您的身體「硬體」數據（RHR 49, HRV 78）目前處於年度巔峰，代表超量恢復效果顯著，但「軟體」修復仍有細節需要校準：

2.  **傷勢追蹤與防傷提醒：** 目前左腳大拇指傷勢已 100% 消失。但在步頻僅 160-162 spm 的情況下，每一步的觸地時間 (GCT) 偏長，這對關節與曾經受傷的趾部仍有潛在衝擊。步頻每提升 5%，觸地衝擊可減少約 15%。
3.  **疲勞度評估：** 您的身體能量 (BB) 持續在高檔 (92+)，這代表您並非因為「身體疲勞」而縮減 06/17 的課表，更多可能是由於「心理壓力」或「時間安排」導致的執行偏差。教練建議：當 BB 高於 80 時，您的身體絕對有能力承擔 T 跑的壓力，請勇敢走出舒適區。

## 📅 4. 本週訓練重點 (W25)
本週進入「進展期中段」的關鍵過渡期，我們的工作核心是擊碎 W24 出現的有氧脫鉤，並建立在疲勞下維持 175 spm 步頻的「身體紀律」。

**一、訓練重點：建立「耐速力」與「抗疲勞結構」**
本週的核心任務是訓練身體在乳酸閾值邊緣的續航力。這不僅是體能的提升，更是心理韌性的建立。我們不追求跑量的激增，而是追求每一公里的「質量」。週三未完成的 T 跑是用來拉高你的速度天花板，而週日的 LSD 則是用來擴張你的有氧地盤。兩者結合，才能形成 Sub 4:30 所需的「耐速力」。

**二、核心目標：M 配速穩定化與步頻紀律定型**
1.  **門檻耐受力補強：** 鑑於 06/17 強度不足，週五的課表將強制修正為「5km T 門檻跑」。必須鎖定在 6'07"/km，心率進入 Z4。我們要教導身體在乳酸產生的動態平衡中，仍能維持穩定的心率。
2.  **步頻革命 (175+ spm)：** 過去 162 spm 是效率的殺手。本週所有的跑步活動（包含熱身），心法只有一個：**頻率高一點、腳步輕一點**。目標是讓 175 spm 成為你的身體本能，這能減少垂直振幅，節省每一分瓦數。

**三、關鍵課表：16km 戶外起伏 LSD 與補償性 T 跑**
*   **06/19 (五) 強度補強：** 5km 門檻跑。這是對週三失利的補償。這 30 分鐘的努力，將決定你雪梨馬拉松後半程是否會失速。
*   **06/21 (日) 長跑日：** 戶外 16km LSD。這不是一場輕鬆的散步。請尋找有坡度的路線，模擬雪梨大橋的起伏。這 120 分鐘的體能堆疊，是達成 Sub 4:30 的唯一基石。

**四、預期成效：跑步經濟性 (RE) 的實質優化**
完成本週訓練後，您應感受到以 6'33" 的配速跑動時，呼吸比上週更為從容。透過提高步頻（175 spm），垂直振幅將會減少，這意味著你的能量被更多地用來「前進」而非「跳躍」。這種能量利用率的提升，是全馬運動員最強大的武器。

**五、執行建議：環境適應與步頻監控**
1.  **走出健身房：** 跑步機無法模擬雪梨的海風與路感。本週末兩次跑步請務必在「戶外」進行，這對於平衡穩定肌群的喚醒至關重要。
2.  **步頻節拍器：** 下次跑步請佩戴耳機，設定節拍器為 175 spm。忘記配速，先對準節拍。
3.  **補給演練：** 週日 LSD 請每 45 分鐘攝取一包能量膠。我們要測試你的腸胃在肌肉疲勞下，是否仍能高效吸收。
