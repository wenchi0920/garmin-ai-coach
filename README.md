## 1. 系統核心定位
Garmin AI Coach 是一個結合 Garmin 穿戴裝置數據與 AI 智慧分析的專屬科學化訓練計畫書。我們秉持「科學訓練，精確校準」的核心價值，系統化地從 Garmin 下載您的訓練紀錄（.fit 檔案），並透過自動化分析模組對每一筆資料進行深度解析。透過動態監控您的跑力指標（VDOT）、疲勞累積程度以及整體恢復狀態，系統會為您量身打造並自動規劃涵蓋跑步、專項肌力與瑜伽放鬆的週期化課表。系統不僅會自動上傳這些精心設計的課表供您執行，更提供即時的專業建議與每週訓練回饋。我們致力於在最小化受傷風險的前提下，精準推升您的體能，協助您達成設定的賽事目標。我們深信，追求卓越表現的同時，不僅能獲得身體的健康，更能讓您在運動的過程中體驗心靈的成長與豐富。

**【旅遊後過渡與專業恢復建議】**
身為您的專屬教練，我必須提醒您，旅遊中的「靜態與低強度累積疲勞」與訓練中的「動態高強度疲勞」在生理機制上截然不同。每日長時間的穿梭會讓小腿腓腸肌與足底筋膜長時間處於半收縮狀態，容易導致回流不暢與深層緊繃。針對您即將結束旅程並重返賽道，我提供以下專業的恢復指南：
1. **下肢淋巴與靜脈回流管理**：強烈建議在今明兩晚回到飯店後，執行「靠牆掛腿」15-20 分鐘。透過改變重力方向強迫神經系統放鬆下肢張力，加速血液與淋巴的回流。
2. **冷熱刺激的血管體操**：請善用日本的浴場文化。透過 40-42 度的熱水浸泡 5 分鐘擴張血管，隨後以冷水沖洗小腿 1 分鐘，重複 3 個循環，這能極大化地加速深層肌肉廢物的排除。
3. **筋膜張力重置**：利用飯店提供的按摩球或是礦泉水瓶，針對腳底進行深層滾動按壓；並透過弓箭步或蜥蜴式，拉長大腿前側與髖關節前方的筋膜，恢復下肢活動度。
4. **睡眠與營養的協同效應**：確保攝取足夠的優質蛋白質與維生素 C，以利細胞修復。睡前進行「4-7-8 呼吸法」，幫助神經系統從白天的興奮狀態切換到深層修復狀態，提升深度睡眠的比例。
5. **心理韌性的超補償**：旅遊帶來的多巴胺分泌能有效重置大腦的疲勞閾值。當您結束旅程回到賽道時，您對長跑的渴望將會成為極高明的心理戰略，成為後續訓練的強大推力。

## 📅 4. 本週訓練重點 (W21)：戰略性減量與身心重置
本週 W21 為計畫中的「過渡週」與「減載週 (Unloading Week)」。結合您的日本旅遊行程，本週訓練的核心不在於跑量的累積，而在於透過主動恢復與適度刺激，保持有氧感知，為回國後的進階訓練積蓄強大的動能。以下是本週訓練的五大核心項目與詳盡解說：

**一、 訓練重點：戰略修復與主動恢復 (Strategic Recovery)**
經過連續四週密集的基礎期重建訓練，您的肌肉與中樞神經系統都已承受了不小的壓力。本週我們巧妙地透過日本旅遊的環境轉換，刻意執行「戰略性減載」。主動恢復不僅僅是停止高強度的跑步，更是利用旅遊期間的大量低強度有氧（步行）來促進全身的血液循環。此階段的重點是讓交感神經從高度緊繃的備戰狀態，平穩切換至副交感神經主導的修復狀態，這能有效預防因長期累積疲勞而誘發的過度訓練症候群。

**二、 核心目標：維持心肺感知與肌肉記憶 (Maintain Muscle Memory)**
即便身處減載週，我們的核心目標依然明確：在旅遊期間透過極短距離的輕鬆跑或徒手肌力訓練，持續喚醒並維持肌肉與神經的連結。我們追求的是「保持神經對路面的敏銳度」，特別是針對 175+ spm 高步頻的肌肉記憶。在旅遊期間偶爾的晨跑中，您完全不需要在乎配速，而是專注在腳步的輕盈感、落地位置的精準度，以及呼吸的順暢度。這些神經記憶的維持將確保您回國後，能夠在第一時間無縫接軌常規的進階長跑課表。

**三、 關鍵課表：T門檻刺激與深度修復瑜伽 (Key Workouts)**
本週的課表設計呈現「一頭一尾」的巧妙特徵：
1. **7km T 門檻跑（出國前執行）**：這是本週唯一的質量指標與強度刺激，利用 T 配速（6:07/km，Z4）刺激快縮肌群，同時檢驗您的乳酸清除能力，確保出國前仍保有足夠的速度感與肌肉張力。
2. **5km Z2 旅遊探索跑**：抵達日本後，安排以 Z2 輕鬆心率探索異國街道。這是完全無壓力的感知跑，旨在讓跑步回歸純粹的樂趣，並透過慢跑加速下肢血液循環。
3. **每日徒手核心與修復瑜伽**：睡前必須執行下犬式、蜥蜴式以及靠牆掛腿，深度拉伸緊繃的腿後腱與小腿肌群；同時搭配深蹲、死蟲式等核心動作，確保核心肌群不會因為旅遊而過度鬆懈。

**四、 預期成效：體能超補償與傷勢穩固 (Expected Outcomes)**
我們預期這一週的彈性調整能為您帶來顯著的「超補償 (Supercompensation)」效應。當下週一您重啟常規訓練時，您的身體能量指數將會觸及近期高點，HRV 也能展現出極佳的自律神經平衡。此外，左腳大拇指曾經受傷的部位，能藉由低壓的環境完成更深層的細胞重建與穩固；在心理層面，幾天不進行長距離高強度訓練，對長跑的渴望將會達到頂峰。這些生理與心理的正向改變，都將成為接下來應對雪梨馬拉松高強度課表的堅實後盾。

**五、 執行建議：傾聽身體，靈活彈性 (Flexible Execution)**
在異國旅遊的情境下，請務必堅守「不勉強、不補課」的最高原則。旅遊行程本身就是對體力與精神的消耗，請將「傾聽身體的聲音」放在首位。如果當天的步行行程已經讓雙腿感到明顯腫脹或疲憊，請果斷取消探索跑的計畫，將時間改為在飯店內進行 15 分鐘的瑜伽伸展，或是好好享受一次溫泉。一週的彈性變通非但不會讓您的體能倒退，反而是為了未來前進三步前，必須採取的戰略性後退。請盡情享受美食與美景，保持快樂與放鬆的心態，這就是本週最高效的恢復劑。
