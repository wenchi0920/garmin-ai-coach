## 📅 4. 本週訓練重點 (W20 基礎期重建)

本週 (W20) 是本次「基礎期重建 (Base Phase Rebuilding)」的第四週，同時也是這一個關鍵週期的總結與收尾階段。在下週即將啟程前往日本並迎來減量週之際，本週的任務是將前三週累積的訓練效果深度固化。以下是五大核心重點：

**一、 訓練重點：基礎期完美收尾與動作經濟性的深度內化**
本週訓練主軸在於「技術細節的嚴格檢核」與「有氧底盤的終極鞏固」。有氧基礎已然穩固，必須將專注力轉移到「跑步經濟性」的提升，強制將高步頻與低觸地時間的技巧刻印在肌肉神經記憶中，成為面對疲勞時的本能反應。

**二、 核心目標：確立堅不可摧的有氧底盤與高步頻技術革命**
所有跑步課表首要且絕對不可妥協的技術目標，就是將「步頻」穩定拉升至 **175-185 spm** 區間內。請主動縮小步幅，想像地面發燙，迫使腳掌迅速彈起，縮短觸地時間，從根本上降低下肢受傷風險。

**三、 關鍵課表：12km 週末戶外長跑 (LSD) 及 T-Pace 間歇**
1. **4km T-Pace 間歇**：精準測試並提升乳酸門檻附近的代謝清除能力，鍛鍊心肺系統的韌性。
2. **12km 長距離慢跑 (LSD)**：對下肢肌肉耐力與有氧系統的深度淬鍊，要求在戶外執行以模擬真實路面與風阻，並嚴格遵守「負分割」配速原則。

**四、 預期成效：全面擴張有氧區間，實現代謝效率根本質變**
若確實貫徹高步頻技術與 Z2 心率控制，您將明顯感受到能量代謝效率躍升。在相同的 Z2 輕鬆跑心率下，配速能自然變得輕快流暢；面對高強度訓練時，組間心率下降也會更迅速，為雪梨馬拉松打造堅不可摧的有氧防護罩。

**五、 執行建議：心率絕對優先、嚴格戶外適應與心靈深度放鬆**
執行 Z2 輕鬆跑請秉持「心率絕對優先，配速僅供參考」原則。因參與線上虛擬賽，務必將長跑與基礎跑轉移至戶外進行。此外，請持續睡前瑜伽修復習慣，進行動態伸展與心靈層面冥想，讓身心皆達到最佳狀態。
