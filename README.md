**3. 關鍵課表解析：**
*   **04/27 & 04/30 修復瑜伽**：這是本週最重要的課表。沒有好的恢復，就沒有好的適應。重點在於髖關節與阿基里斯腱的深度放鬆。

*   **有氧天花板平移**：在 Z2 心率下，預期配速能穩定維持在 6'50"-7'10" 之間。

*   **環境模擬**：若在室內執行虛擬賽，請務必開啟電風扇並設定 1% 坡度，模擬戶外跑步的體感與核心負荷。
