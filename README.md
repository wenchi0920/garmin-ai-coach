## 📅 本週訓練重點：W20 基礎期重建圓滿週

**1. 訓練重點：有氧效能固化與乳酸耐受力喚醒**

**2. 核心目標：頻率革命與代謝閾值提升**
*   **技術指標**：全週所有跑步課表強制對位節拍器。Z2 輕鬆跑目標 170+ spm，週三 T-Pace 挑戰穩定在 178-185 spm。關鍵在於縮短觸地時間（目標 <260ms），利用肌腱的彈性勢能而非肌肉的蠻力推蹬。
*   **生理指標**：提升乳酸清除效率。透過 T-Pace 的刺激，預期將乳酸門檻配速往 6'00"/km 以內推動。
*   **心理指標**：建立對「強度」的耐受力。在週三間歇跑與週六長跑的後半段，學習在高心率下保持大腦清醒，這是雪梨馬多坡賽道必備的心理素質。

**3. 關鍵課表深度解析：**
*   **05/13 (三) 4km T-Pace 間歇 (4x1km)**：這是本週的靈魂課表。T-Pace (Threshold) 代表的是一種「快而舒坦」的節奏。1km 的間歇長度足以讓血乳酸濃度達到臨界點，這能訓練身體在比賽中更有效地處理乳酸堆積。對於目標 Sub 4:30 的您，這是在 6'23" 目標配速下感到「從容」的保證。
*   **05/16 (六) 12km 長跑 (Long Run)**：本週期里程最高峰。重點在於 9km 之後的「體態檢測」。當疲勞產生時，核心是否崩塌？步頻是否掉下？請在最後 3km 刻意將步頻提升 2-3 spm，模擬全馬最後衝刺時的神經支配能力。

**4. 預期成效：**
完成 W20 後，您的 VDOT 雖然可能短期因疲勞而在數據上維持 33，但您的「生理經濟性 (Running Economy)」將發生質變。同樣 7'30" 的配速，心率將比 W17 時降低約 3-5 bpm。這省下的心跳次數，就是您 8 月份在雪梨馬拉松後半段對抗地形挑戰的籌碼。

**5. Execution 執行建議：**
*   **戶外實戰切換**：週三 T-Pace 與週六 12km 長跑請**絕對禁止**在跑步機上執行。您需要真實路面的反饋力與風阻來訓練足底的小肌肉群。
*   **呼吸與步頻聯動**：建議採用 2-2 呼吸法（兩步吸、兩步吐），並與 175 bpm 的節拍器形成穩定的韻律感。
*   **肌力補強 (Eccentric Loading)**：週二與週五的徒手肌力請著重「單腳穩定」與「離心控制」，特別是單腳硬舉，這能確保您在日本旅遊期間的高強度步行中不會產生關節負擔。
