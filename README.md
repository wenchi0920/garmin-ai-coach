**一、 訓練重點：乳酸門檻刺激與基礎有氧擴張 (Threshold & Endurance)**
本週採取「雙軸並進」策略，旨在同時提升您的速度耐力與基礎有氧容量。第一軸是週四的「T 門檻刺激」。乳酸門檻跑的目的是訓練身體在乳酸堆積邊緣的「耐受」與「清除」能力。6/4 的跑步機訓練雖然里程較短，但配速已逼近目標且心率表現優異。這為我們在夏日維持強度提供了信心，代表您的有氧引擎已經足夠強大，能支撐更高強度的運轉。第二軸是週日的「16km LSD 長距離慢跑」，這是六月最重要的有氧引擎擴張練習。透過長達兩小時的持續跑步，我們能促進粒線體增生，強化毛細血管網，讓身體在長距離後期仍能高效利用脂肪作為能源。脂肪代謝的效率，對於全馬 sub 4:30 來說是無可取代的核心基本功，也是避免撞牆期的關鍵。

**二、 核心目標：重建下肢抗疲勞能力與核心穩定 (Leg Strength & Core Stability)**
為了對抗雪梨馬拉松起伏多變的賽道，特別是那些對肌肉離心收縮要求極高的下坡路段，本週已安排高品質的負重深蹲與硬舉。這不僅是為了絕對力量的增長，更是為了建立肌肉的「剛性 (Stiffness)」與「離心收縮 (Eccentric Contraction)」耐力。跑步機上的跑步雖然效率高，但缺乏路面的衝擊變化，容易讓微小穩定肌群處於休眠狀態。因此，我們透過 45kg 深蹲來模擬下坡時大腿承受的衝擊。強大的核心與下肢剛性是防止 30km 後姿勢崩潰、步幅縮減的最強防線。當您在賽事末段感到疲勞時，這些扎實的肌力將會是支撐您維持跑姿、穩定向前的強大後盾。

**三、 關鍵課表：T 門檻測驗與長距離耐力考驗 (Key Workouts)**
本週有兩項不可妥協的關鍵課表，必須全神貫注執行：

*   **W23-16km LSD 長跑 (06/07)**：本週的終極考驗。這不只是里程的累積，更是心理層面的耐力訓練。請嚴格鎖定 Z2 區間 (7:12-8:02/km)，專注於呼吸的節律感與腳步的輕盈度。這同時也是完成 ONE TOKYO VR 21K 挑戰剩餘里程的最佳契機。切記，LSD 的重點是「時間與持續性」，而非速度。

**四、 預期成效：心肺效率升級與下肢剛性強化 (Expected Outcomes)**
若能紀律執行本週課表，我們將能觀察到以下顯著的生理適應與成效提升：
1. **心肺連動優化與體感降級**：經歷 T 門檻跑的刺激後，您的心肺系統將適應更高強度的需氧量。這會導致您在日後執行輕鬆跑 (Z2) 時，心率會進一步降低，同樣配速下的「體感疲勞度 (RPE)」會明顯下降，讓您跑得更輕鬆、更持久。
2. **下肢彈性提升與步幅經濟性**：重量訓練與長距離跑步的結合，能有效強化足弓、阿基里斯腱與下肢筋膜的彈性。這意味著您在每一步推進時，都能更有效地回收並利用觸地時的彈性能量，節省更多的肌肉主動發力，從而全面提升「跑步經濟性 (Running Economy)」。

**五、 執行建議：高步頻防線與夏日實戰補給演練 (Execution Advice)**
面對逐漸炎熱的天氣與增加的訓練量，以下執行細節將是成功與否的關鍵：
*   **嚴守高步頻防線**：請在所有跑步課表中刻意維持 175 spm 以上的高步頻。高步頻能降低觸地時間，減少落地時身體重心垂直位移產生的能量浪費。更重要的是，高步頻能將每一步的衝擊力分散，大幅降低對膝關節與腳踝的負擔，這是無傷備賽的最高指導原則。
*   **實戰補給演練與腸胃適應**：週日 16km 長跑預計耗時兩小時左右，這是一個絕佳的「腸胃適應力」練習機會。請務必模擬雪梨馬拉松賽事當天的節奏進行補給。嘗試在出發前及每 45-60 分鐘補充一次能量膠，並定時飲水，找出最適合您的品牌與攝取頻率，避免賽事當天發生腸胃不適。
*   **環境應對策略與心率導向**：夏日濕熱易引發心率漂移 (Cardiac Drift)。若戶外氣溫超過 28 度，請主動將目標配速放慢 10-15 秒，改以「心率區間」為唯一導向。記住，夏日訓練的目的是「生理刺激與耐熱適應」，而非死守配速數據。傾聽身體的聲音，安全至上。
