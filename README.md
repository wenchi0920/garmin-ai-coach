## 📅 本週訓練重點：W17 重啟基礎期 (Base Phase) (約 1000 字詳盡描述)

**1. 訓練重點：強化有氧基礎與步頻校準 (Aerobic Foundation & Cadence Alignment)**
本週的核心任務是將「傷癒後的身體」轉換為「競技用的引擎」。重點不在於速度的數字，而在於「步頻的神經反射建立」。我們要建立一套低能耗、高效能的跑步模式，為雪梨馬那起伏不定的 42.195 公里儲備節能空間。

**2. 核心目標：175-180 spm 的金律節奏**
*   **神經校準**：無論配速快慢，跑步時步頻目標均設定為 **175-180 spm**。高步頻能減少每一步的垂直振幅（Vertical Oscillation），減少對腳趾傷處的衝擊，並提升跑步經濟性。
*   **有氧容量堆疊**：透過 Z2 區間的累積，提升毛細血管密度與脂肪氧化率。
*   **負荷適應**：完成 40kg 等級的下肢強化，建立對抗雪梨下坡所需的肌肉力量（離心收縮力）。

**3. 關鍵課表解析：**
*   **04/22 Z2 基礎有氧**：表現極佳（6.14km），步頻 178spm。這是本週最完美的數據標竿，證明了您具備高效能運作的潛力。
*   **04/23 下肢力量強化**：40kg 深蹲與硬舉。這雖然造成了後續疲勞，但為 8 月雪梨馬的坡度挑戰提供了必要的底盤支撐。
*   **04/24 T 區間耐力啟動**：7.58km。配速（5'16"-5'38"）展現了極高的天花板，但也提醒我們需注意訓練區間的精確度，避免過度消耗。

**4. 預期成效：**
*   **動作輕盈化**：預期觸地時間能從 280ms 穩定降至 260ms 以下，這代表彈性回收能量的效率提升。
*   **心率平穩化**：消除在相同配速下的「心率漂移」，代表有氧穩定度提升。
*   **動作自動化**：將 175+ spm 的步頻內化為潛意識，在疲勞時仍能維持正確跑姿。

**5. 執行建議：**
*   **步頻警報**：手錶請務必開啟步頻警報（170-185）。在下週的 Z2 練習中，若步頻低於 170，請立即縮小步幅，而非增加推蹬力。
*   **慢即是快**：在 Z2 跑中，若配速太快導致心率衝破 166bpm，請果斷降速。基礎期的成功在於「時間」的累積，而非「速度」的突破。
*   **跑步機坡度**：在室內練習時，建議設定 1.0% 的坡度，以模擬戶外路面的空氣阻力與物理回饋。
