**【專業恢復與防傷建議：迎戰長跑與減量週】**
面對下週 (W21) 即將到來的日本旅遊減量週，為您制定以下專業恢復與防傷策略：
1. **強化髖部活動度以支撐步頻革命**：雪梨馬拉松起伏多變，對髖關節活動度是極大考驗。請繼續維持「蜥蜴式」與「鴿式」的高頻率操作，打開髖關節不僅能減輕下背痛，更是拉升步頻至 175 spm 以上的解剖學關鍵。唯有靈活的髖關節，才能讓每一次的抬腿與推蹬流暢無礙。
2. **足底筋膜與阿基里斯腱的預防性保養**：傷勢雖已痊癒，但在跑量逐漸攀升的階段，下肢末端的壓力不容忽視。請務必在睡前使用按摩球輕踩足底，針對小腿肚深層進行放鬆，保持絕佳彈性以減少代償壓力。這對於預防足底筋膜炎與阿基里斯腱發炎至關重要。
3. **賽事級的營養攝取與腸胃適應訓練**：長距離訓練達肝醣耗盡臨界點，請在訓練前補充低 GI 碳水，並在訓練中實踐「少量多次」補給。訓練後 30 分鐘內以碳水與蛋白質 3:1 比例補充，加速微損傷修復。
4. **準備迎戰減量週，身心全面重置**：下週前往日本的旅遊是絕佳的心理與生理雙重放鬆期。每日觀光徒步可視為低強度的有氧恢復，請帶著愉悅心情享受，暫時放下課表的壓力。每晚回到飯店後進行 10-15 分鐘下肢伸展，讓身體徹底重置，為回國後更高強度的訓練週期儲備滿滿的能量。

## 📅 4. 本週訓練重點 (W20 基礎期重建)

本週 (W20) 是本次「基礎期重建 (Base Phase Rebuilding)」的第四週，同時也是這一個關鍵週期的總結與收尾階段。在下週即將啟程前往日本並迎來期待已久的減量週之際，本週的任務是將前三週累積的訓練效果深度固化。以下是五大核心重點：

**一、 訓練重點：基礎期完美收尾與動作經濟性的深度內化**
本週訓練主軸在於「技術細節的嚴格檢核」與「有氧底盤的終極鞏固」。經過近一個月的打磨，您的有氧基礎已然穩固，心率在 Z2 區間的表現堪稱完美。現在，我們必須將專注力全面轉移到「跑步經濟性」的提升上。我們需要強制將高步頻與低觸地時間的技巧，深深地刻印在肌肉神經記憶中，使其成為您面對疲勞時的本能反應。這不僅是為了跑得更快，更是為了跑得更長久、更無痛。將技術動作內化，是這週最重要的心理與生理功課，請在每一次的訓練中，不斷與自己的身體對話，尋找最省力的律動。

**二、 核心目標：確立堅不可摧的有氧底盤與高步頻技術革命**
所有跑步課表首要且絕對不可妥協的技術目標，就是將「步頻」穩定拉升至 **175-185 spm** 的高效區間內。近期的訓練數據顯示，您的步頻仍多在 158-166 spm 徘徊，這意味著步幅偏大，潛藏著過度跨步 (Overstriding) 的風險。請主動縮小步幅，想像地面發燙，迫使腳掌在觸地瞬間迅速彈起，大幅縮短觸地時間。這項技術革命將從根本上降低下肢關節（尤其是膝蓋與腳踝）受傷的風險，同時大幅提升向前的推進效率。請務必在手錶畫面上將步頻設為首要關注指標，時刻提醒自己維持輕快的節奏。

**三、 關鍵課表：12km 週末戶外長跑 (LSD) 及 T-Pace 間歇**
本週安排了兩項截然不同但相輔相成的關鍵課表，旨在全面刺激不同的能量系統：
1. **4km T-Pace 間歇**：此課表旨在精準測試並提升乳酸門檻附近的代謝清除能力。透過適度的無氧刺激，鍛鍊心肺系統的韌性，讓身體適應高配速下的代謝壓力。請確實做好前後的熱身與緩和，確保在安全的前提下完成強度刺激。
2. **12km 長距離慢跑 (LSD)**：這是對下肢肌肉耐力與有氧系統的深度淬鍊。我們強烈要求在戶外執行，以模擬真實路面的反饋與風阻。請嚴格遵守「負分割」配速原則，前半程保留體力，控制心率，後半程再逐漸穩定輸出。這 12km 不只是體能的考驗，更是補給策略與心理韌性的彩排。

**四、 預期成效：全面擴張有氧區間，實現代謝效率根本質變**
若能確實貫徹高步頻技術與嚴格的 Z2 心率控制，您將會明顯感受到能量代謝效率的躍升。在相同的 Z2 輕鬆跑心率下，您的配速將會自然變得更加輕快且流暢；而在面對高強度的 T 區間訓練時，組間休息的心率下降速度也會更為迅速。這種生理指標的正向反饋，代表著心臟泵血能力的提升與微血管網的擴張。這正是為雪梨馬拉松打造堅不可摧有氧防護罩的最佳證明，讓您在未來的長距離賽事中，擁有更充沛的體力儲備與更好的抗疲勞能力。

**五、 執行建議：心率絕對優先、嚴格戶外適應與心靈深度放鬆**
在執行 Z2 輕鬆跑時，請務必秉持「心率絕對優先，配速僅供參考」的鐵則。不要被手錶上的配速數字綁架，若當天狀況不佳，降速以維持低心率才是明智之舉。此外，由於目前正參與線上虛擬賽（TATTA RUN 與 ONE TOKYO），請務必將長跑與基礎跑轉移至戶外進行，以符合賽事規範，同時也能更好地適應真實賽道的路感。最後，請持續睡前主動修復瑜伽的習慣，透過動態伸展與呼吸冥想，讓身心在每日訓練後都能達到最佳的重置狀態，迎接次日的挑戰。保持耐心與專注，穩健邁向目標！
