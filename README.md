## 📅 本週訓練重點：W20 基礎期重建圓滿週

本週 (W20) 是「基礎期重建 (Base Phase Rebuilding)」的第四週，也是該週期的最後一戰，我們將透過以下五大核心項目來鞏固您的有氧底蘊：

**1. 訓練重點：基礎期收官與步頻技術內化**
本週的核心任務是將前三週累積的體能紅利進行深度的「固化」。在您下週啟程前往日本（預期跑量將大幅降低）之前，我們必須給予身體一次紮實的有氧刺激。同時，持續專注於「技術動作的內化」，強制要求將高步頻與低觸地時間的技巧，深深刻印在肌肉記憶中。

**2. 核心目標：有氧引擎極大化與關節衝擊極小化**
全週所有跑步課表，無論配速快慢，首要目標是將步頻穩定拉升至 175-185 spm。您的平均步幅偏大（約 0.88m），這在疲勞時容易造成過度跨步 (Overstriding)。請想像地面是發燙的，練習「腳掌觸地即刻彈起」的體感，從根本上降低對膝蓋與足踝的垂直衝擊力。同時，透過大量的 Z2 輕鬆跑，穩固有氧區間，讓身體成為更有效率的燃脂機器。

**3. 關鍵課表：12km 週末長跑 (Long Run)**
本週六安排的 12km 長跑是本週期最具指標性的一練，旨在模擬雪梨馬拉松中後段的體能消耗。請將其視為一場小型實戰，不僅要累積距離，更要演練補給策略。請嚴格執行「負分割 (Negative Split)」，前 3 公里刻意壓抑速度，並在最後 3 公里體力下滑時，專注於核心穩定與步頻不掉速。

**4. 預期成效：代謝效率的實質躍升**
順利完成本週的訓練後，您將會感受到身體產生的質變。在相同的 Z2 心率負荷下，您的配速將會比數週前更為輕快；而面臨相同配速時，您的呼吸將更為從容。這象徵著您的有氧底盤已經成功拓寬，為未來的高強度專項訓練打下了堅不可摧的基礎。

**5. 執行建議：心率絕對優先與戶外實戰切換**
在執行 Z2 輕鬆跑時，請「心率絕對優先」，若體能狀況導致心率飆升，請毫不猶豫地降速。此外，為了應對兩大線上虛擬賽（TATTA RUN 與 ONE TOKYO）的 GPS 軌跡要求，並適應真實路面反饋，請盡可能將本週的長跑與恢復跑移步至戶外進行。動態修復不間斷，瑜伽課表是這份計畫的隱形功臣，請帶著覺察進行每一個伸展。
