## 📅 本週訓練重點：W19 步頻革命與攝氧開發

**1. 訓練重點：步頻革命 (Cadence Revolution)**
本週的核心任務是將「步頻」從低效率的 152 spm 拉升至 175-180 spm 的科學區間。對於 40+ 跑者，這不只是速度問題，更是**長壽跑者 (Longevity Runner)** 的必經之路。高步頻能顯著縮短觸地時間，將衝擊力從關節轉移至肌肉彈性能量回彈，是 Sub 4:30 目標下「無傷完賽」的核心技術保險。透過週三間歇跑的實證，您已經證明自己具備高頻跑動的能力，現在我們需要的是將其「內化」為本能。

**2. 核心目標：技術校準與有氧引擎擴張**
*   **技術指標**：在所有配速下（包含 Z2 輕鬆跑），強制維持 170+ spm 的步頻，將垂直振幅控制在 8cm 以內。
*   **生理指標**：透過 I 間歇拉高最大攝氧量 (VO2Max)，在週三的訓練中您已成功觸及 173 bpm 的最大心率，這對開發心肺天花板至關重要。
*   **賽事對接**：完成 15km LSD (TATTA RUN)，這將是 5 月份最重要的耐力基石，也是對高步頻耐受度的終極考驗。

**3. 關鍵課表解析：**
*   **05/06 (三) 8km I 間歇訓練 (5:38/km)**：**已圓滿達成**。衝刺配速達 5'17"，步頻 176+。這堂課的意義在於「打破神經鎖定」。它告訴您的身體：你不再是一個 152 步頻的跑者，你具備高頻流動的潛力。
*   **05/09 (六) 15km LSD (Long Slow Distance)**：**本週期末考**。目標是在戶外真實路面、Z2 心率下，挑戰連續 120 分鐘維持 175 spm 的步頻。這是技術（頻率）與耐力（長距離）的結合，也是雪梨馬拉松後半段不崩盤的關鍵模擬。

**4. 預期成效：**
完成 W19 訓練後，您將會感受到跑步時的「體感」變輕。雖然心率相同，但因為步頻增加，單次觸地的垂直衝擊會降低。預計本週末完成後，您的神經肌肉連結將更趨穩定，下週的 Z2 配速有望在相同心率下自然提升 5-10 秒。

**5. Execution 執行建議：**
*   **「節拍器」依賴**：明日長跑請務必開啟手錶節拍器（設定 175）。在疲勞的後半程，不要聽音樂，改聽節拍，讓節奏帶動腳步。
*   **「輕快回收」意識**：想像腳底不是在推蹬地面，而是在觸地瞬間迅速縮回。強調「提腳」而非「用力蹬地」。
*   **補給模擬**：明日 LSD 請每 5km 補水一次，10km 處演練一次能量膠補給。在高步頻下，腸胃的震動會不同，這也是補給訓練的一部分。
*   **核心參與度**：利用 05/05 重訓建立的骨盆穩定感，在跑步中保持核心微收，防止高頻跑步時上半身產生不必要的晃動。
