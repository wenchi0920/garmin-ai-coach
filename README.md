## 1. 系統核心定位
Garmin AI Coach 是一個深度結合穿戴裝置數據的專屬智能訓練分析系統。我們秉持「科學訓練，精確校準」的核心理念，透過自動化抓取並深度解析跑者的 Garmin 訓練檔案（.fit），動態追蹤您的跑力指標（VDOT）、疲勞累積程度以及生理恢復狀態。系統不僅自動規劃涵蓋跑步、肌力與瑜伽修復的客製化週期課表，更依照每日最新數據提供即時的戰略回饋。我們致力於在最小化受傷風險的前提下，穩步推升您的競技體能，陪伴您達成賽事目標，並在運動旅程中實現身心靈的全面進化。

**【專業恢復、防傷策略與深度建議】**
作為教練，我必須強調本週在日本旅遊期間的「戰略性修復」價值。目前您正處於基礎期與旅遊過渡期的交界點，恢復不僅是單純的休息，更是一場生理系統的「重新校準」。

1.  **旅遊步行的「低強度恆速運動 (LISS)」效應**：
    在日本旅遊期間，每日動輒 1.5 萬至 2 萬步的行走，從運動科學角度看，這是一種絕佳的主動恢復。長時間的低強度運動能增加微血管密度，促進血液循環，加速基礎期訓練累積的代謝廢物（如乳酸、發炎因子）排除。這對於受傷後的筋膜重塑具有極高價值。請將這些步伐視為「無形的訓練」，它們正在強化您的耐力底層。

2.  **針對性下肢筋膜管理**：
    大量的步行雖然有利於有氧，但也容易導致足底筋膜、小腿比目魚肌與髂脛束（IT Band）的過度張力。建議每晚返回飯店後，執行 **「掛牆抬腿 (Viparita Karani)」15 分鐘**。此姿勢能利用重力引導下肢靜脈血與淋巴回流，減緩因長時間站立引起的腫脹感。此外，建議利用日本隨處可見的溫泉或飯店浴缸進行「冷熱交替浴」：熱水浸泡 3 分鐘配合冷水沖淋 1 分鐘，重複 3 次。這能引發強烈的血管舒縮運動，是目前頂尖運動員最常用的恢復手段。

3.  **神經系統的「皮質醇調節」策略**：
    高強度的馬拉松備賽往往會讓交感神經長期處於亢奮狀態。本週的旅遊是一次難得的神經系統重置 (Neural Reset)。享受異國美食與美景能顯著降低體內的皮質醇 (Cortisol) 水準。皮質醇是肌肉分解的元兇，透過這週的心理放鬆，我們能優化體內的荷爾蒙環境，讓受損的肌肉纖維獲得深層修補。請記住，一個快樂的腦袋能帶出一雙有力的腿。

4.  **營養補給的「抗發炎模式」**：
    旅遊期間的飲食是修復的關鍵。日本飲食中豐富的 Omega-3（如生魚片）與綠茶抗氧化劑，能有效對抗運動誘發的氧化壓力。建議額外補充水分，特別是在大量行走後，確保每日飲水量達到體重 x 40ml，以維持筋膜組織的水合度。充足的蛋白質攝取（如和牛、海鮮）將提供修補受傷後軟組織所需的胺基酸。

## 📅 4. 本週訓練重點 (W21)

本週 (W21) 是您訓練計畫中非常特殊的「日本旅遊過渡週」。這不是一個追求跑量（Volume）的階段，而是一個追求「品質與適應（Quality & Adaptation）」的轉場期。以下是本週的五大核心項目與詳盡執行方針：

**一、 訓練重點：兩極化的結構安排 (Polarized Structure)**
本週的訓練呈現明顯的「出國前強化、旅途中修復」特徵。在週一至週二，我們安排了高品質的深度修復瑜伽與 T 門檻跑，目的是在進入旅遊放鬆模式前，最後一次刺激乳酸門檻與核心穩定度，讓肌肉記住高強度的收縮體感。這種兩極化的安排，能有效避免因連續長假導致的體能「滑落（Detraining）」。當身體在接受強刺激後進入適度的低強度期，會引發「超補償 (Supercompensation)」效應，讓您的體能在下週回國時達到新的小巔峰。

**二、 核心目標：維持 Z2 有氧基礎與心肺感知**
在旅遊期間，我們絕對不強求里程的硬性累積，但要求維持「心肺系統的適度喚醒」。核心目標是透過 5km 的旅遊探索跑，將心率穩定控制在 Z2 (153-166 bpm) 區間。這種刺激能防止毛細血管的退化，並維持線粒體在燃脂模式下的運作效率。此外，本週的大量「非跑步」活動（如市區徒步探索）將成為支撐基礎期有氧底蘊的重要推手，目標是讓身體在低心率下持續運作。

**三、 關鍵課表：T 門檻刺激與旅遊探索跑**
1.  **7km T 門檻跑 (5/19)**：這是本週唯一且至關重要的質量課表。透過 3km 的 T 配速 (6:07/km) 刺激，強化身體清除血乳酸的能力。這對雪梨馬拉松後半段的抗疲勞能力至關重要，也是檢視基礎期速度耐力的試金石。
2.  **5km Z2 旅遊探索跑 (5/20 或 5/21)**：這是一次純粹放鬆的跑步。目標是享受異國的清晨街景，讓雙腿在陌生的環境中保持運轉。建議將專注力放在「步頻」而非配速，嘗試維持 175+ spm 的輕快節奏。
3.  **睡前修復瑜伽與靠牆掛腿 (每日)**：這是旅遊期間的「每日必修課」，目的是對沖每日上萬步步行帶來的下肢高張力。

**四、 預期成效：深層疲勞重置與受傷風險歸零**
經過 W21 整週的戰略性調整，我們預期您的 HRV 指標能重新站穩 65+ 的高點，且每日清晨的 Body Battery 回升率能達到 80 分以上。更重要的是，這種戰略性的「減量（Tapering）」能讓您原本剛痊癒的腳趾傷勢獲得最後且最關鍵的固化期。當您下週結束假期回國時，您的雙腿應感受到一種「對速度的飢餓感」，這代表您的神經與肌肉系統已完全重置，隨時準備好迎接六月更高強度的基礎期中段訓練。

**五、 執行建議：傾聽身體，靈活變通 (Listen & Adapt)**
在日本旅遊期間，我給您的最高指導原則是：**「旅遊與放鬆為主，訓練課表為輔」**。
*   **彈性調整**：若當天觀光步行超過 2 萬步，且感覺到腳踝、小腿異常痠痛或沉重，請毫不猶豫地取消當日的跑步課表，直接改為 20 分鐘的溫泉泡澡與深層足底伸展。絕不帶傷或帶著過度疲勞起跑。
*   **安全與體驗**：在陌生的街道跑步，請隨時注意交通與路面平整度。將配速錶面關掉，僅保留心率區間警示。
*   **水分與電解質**：日本春季氣候乾燥，且步行量大，容易產生隱性脫水。請隨身攜帶水瓶，並在飲食中適度增加鹽分攝取，以維持神經傳導的穩定。
