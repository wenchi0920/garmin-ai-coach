2. **睡前髖部與下肢的深度保養**：長期的連續步行容易造成足底與小腿承受累積性壓力。每天回到飯店後，務必進行 10-15 分鐘的瑜伽伸展，特別是「蜥蜴式」與「靠牆掛腿」，這能有效解除久走帶來的下肢腫脹與髖部痠痛，確保筋膜彈性。腳趾傷勢雖已痊癒，但末端防護依然不可懈怠。
3. **營養補充與身心重置**：旅遊期間請盡情享受美食，但同時注意水分與優質蛋白質的補充，以減少身體發炎反應。將 W21 視為完美的過渡週，徹底放鬆心靈，讓身體重置，為回國後更高強度的馬拉松專項訓練週期準備滿滿的動能。

## 📅 4. 本週訓練重點 (W21 基礎期與旅遊恢復)

本週 (W21) 恰逢您的日本旅遊，這是一個極佳的過渡期（Transition Phase）。我們的核心目標是「主動恢復與輕度維持」，讓身體在基礎重建期累積的疲勞完全釋放，同時確保體能不流失。以下是本週的五大核心重點與執行方針：

**一、 訓練重點：出國前質量刺激與旅遊動態恢復**
本週前半段（5/18-5/19）將專注於行前的高效刺激，包含深度的修復瑜伽與一次 T 門檻跑，以保持速度感與肌肉張力。隨後（5/20-5/24）將完全以旅遊期間的動態恢復為主軸。我們不再追求高跑量或高強度課表，而是專注於「維持心肺活性與緩解步行疲勞」。海外的大量步行本身就是一種優質的低強度有氧刺激。

**二、 核心目標：主動修復身體能量，維持 Z2 有氧基礎**
本週的核心任務是在出國前完成最後的質量訓練，並在旅遊期間保護好雙腿。目標是透過旅遊中的 5km Z2 探索跑，以及每天睡前的高品質修復瑜伽（下犬式、靠牆掛腿等）或徒手肌力（深蹲、死蟲式等），維持基本體能。在身心放鬆的同時，依然要有意識地在慢跑中維持輕快的步頻，讓雙腿習慣高轉換率。

**三、 關鍵課表：行前 T 門檻跑與旅遊輕量長跑**
本週的關鍵訓練集中在出發前與旅途尾聲：
1. **7km T 門檻跑 (5/19)**：包含 3km 的 T 配速 (約 6:07/km)。這是一次針對乳酸門檻的精準刺激，在出國前喚醒快速收縮的肌纖維，提升身體清除乳酸的效率，讓神經系統記住高配速的體感。
2. **10km 輕量長跑 (5/24)**：在旅遊的尾聲，安排一次 8km 的 Z2 有氧跑。主要目的是在經歷幾天的放鬆與大量步行後，重新建立有氧基礎，喚醒長跑肌群，並為下週回歸正常課表做好心肺適應與銜接。

**四、 預期成效：深層疲勞歸零，身心靈滿載回歸**
我們預期在經歷這趟日本旅途的身心雙重放鬆後，您在生理與心理上都能夠達到極佳的狀態。透過每日規律的瑜伽拉伸與輕量肌力喚醒，您的自律神經系統將獲得徹底重置。只要順利完成行前的質量跑並在旅途中保持適度活動量，前期的有氧底盤將依舊堅固，甚至會因為獲得充分休息而展現出更強的爆發力，為接下來的強化週期注入滿滿的能量。

**五、 執行建議：傾聽身體真實反饋，不拘泥於既定課表**
在日本旅遊期間，所有的課表安排都具有極大的「彈性」。如果當天觀光走了兩萬步，覺得腿部肌肉痠痛疲乏，請毫不猶豫地跳過當天的肌力或跑步課表，改為深度的瑜伽伸展。享受旅途、不產生壓力是最高原則！訓練時請將專注力放在放鬆與呼吸節奏上，心率是我們最好的導航，請確保輕鬆跑維持在穩定的 Z2 區間內，平安健康地完成每一趟異國探索。
