### 博帕爾馬拉松 (Run Bhopal Run)

1. **歷史背景**: 博帕爾馬拉松 (Run Bhopal Run, RBR) 是印度中央邦首府博帕爾市的年度盛事，以推廣「健康博帕爾」與環境保護為核心理念。賽事環繞著標誌性的博傑塔爾湖 (Bhojtal) 展開，結合了這座「湖泊之城」的自然風光與歷史建築，是印度中部規模最大且最具社會影響力的馬拉松賽事之一。

2. **賽道技術分析**: 賽道地形呈現典型的**起伏 (Rolling Hills)** 特徵，全馬總爬升約 200 公尺。這對跑者的**離心收縮**控制與核心穩定性要求較高。建議採取保守的配速策略，並在後半程實施**負分割 (Negative Split)** 以應對起伏不平的城市道路。博帕爾 12 月的氣候涼爽 (12°C-25°C)，但日照感強烈，需注意水分管理。

3. **補給特色**: 每 5 公里設有完善的補給站，提供電解質、香蕉及能量膠。終點處設有豐富的恢復區。民間私補熱情，常出現具地方特色的素食補給，建議跑者在賽前進行腸胃演練以確保適應。

4. **教練專業評論**: 此賽事定位為高品質的**心智淬煉**與 **LSD** 訓練場。雖然賽道起伏挑戰了 **PB** 的可能性，但對於追求 **BQ** 或備戰頂尖賽事的跑者來說，是絕佳的抗疲勞能力檢測。建議加強山徑模擬訓練，並根據 **RPE** (自覺運動強度) 動態調整節奏。
