# 伊豆大島地質公園馬拉松 (Izu Oshima Geopark Marathon)


### 伊豆大島地質公園馬拉松 (Izu Oshima Geopark Marathon)

1. **歷史背景**:
伊豆大島馬拉松（現常稱為伊豆大島地質公園馬拉松）是東京離島大島最具代表性的體育賽事。賽事深度結合「地質公園 (Geopark)」概念，跑者穿梭於三原山火山噴發形成的獨特地貌中。這不僅是一場競賽，更是一場穿越活火山遺跡、熔岩流與壯闊太平洋海岸線的自然探索之旅，展現了大自然數萬年演化的震撼力量。

2. **賽道技術分析**:
這是一場極具挑戰性的「硬核」山道馬拉松。全馬路線累計爬升約 785 公尺，高低落差達 365 公尺。賽道關鍵在於長距離的連續上坡與下坡，對股四頭肌的**離心收縮**能力要求極高。建議採取的**負分割 (Negative Split)** 策略應極度保守，前半程需嚴格控制 **RPE** 感知運動量，避免在早期的連續爬升中耗盡糖原，並在後半程應對可能強勁的海風挑戰。

3. **補給特色**:
賽事補給充滿濃厚的海島風情，完賽後提供的熱騰騰「あら汁 (魚湯)」是跑者的最大慰藉。賽道補給點除了基本的運動飲料外，也常提供大島特產食材。由於穿越「裏砂漠」路段時地景荒涼且起伏大，跑者需注意補給站間的節奏，確保電解質及水分的及時補充以維持體內環境穩定。

4. **教練專業評論**:
這絕對不是一場能挑戰 **PB** 的平緩賽道，而是檢驗意志力與山坡跑技巧的聖殿。賽事定位更接近於高強度的 **LSD** 實戰訓練，能極大化強化跑者的心肺耐力與下肢力量。對於追求 **BQ** 或極限挑戰的跑者，這場賽事能提供極佳的抗乳酸堆積訓練。若能在此艱難地形維持穩定跑姿完成，回到平原賽道時將會感受到體能階梯式的跨越。
