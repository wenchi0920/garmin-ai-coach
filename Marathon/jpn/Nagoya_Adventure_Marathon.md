# 名古屋探險馬拉松 (Nagoya Adventure Marathon)



### 名古屋探險馬拉松 (Nagoya Adventure Marathon)

1. **歷史背景**:
名古屋探險馬拉松是愛知縣內極具特色的市民賽事，不同於標準的城市路跑，它結合了莊內川河川敷的自然風光與都市叢林的探險感。賽事創辦初衷是為了讓跑者在繁忙的名古屋市區旁，體驗與大自然對話的樂趣。雖然規模不及名古屋女子馬，但其獨特的「非典型賽道」吸引了大量喜愛野趣與挑戰多變路面的跑友，是當地深耕多年的口碑賽事。

2. **賽道技術分析**:
賽道主要沿著莊內川河岸展開，混合了柏油路、泥土路與碎石路面。地形雖無極端爬升，但由於路面材質多變，對跑者的核心穩定性與**離心收縮**控制力有較高要求。建議配速策略採取保守開局，在河川敷平緩處維持節奏，進入非鋪裝路面時適度提高步頻以應對不穩定地表。若遇側風，建議尋求集團破風，避免獨自在開闊河岸消耗過多體力。

3. **補給特色**:
補給站充滿溫暖的在地人情味，除了標準的電解質補給，常出現名古屋特色的味噌點心或在地醃漬物。賽後更提供溫熱的湯品，幫助跑者在秋涼的河風中恢復體溫。這種「探險後的小確幸」是該賽事最令人難忘的環節。

4. **教練專業評論**:
本賽事定位為**LSD**訓練與心智淬煉的絕佳場合。由於地形與路面多變，並不適合追求極致 **PB**，但非常適合用於賽季初期的體能建構。其複雜的地形能有效刺激足底小肌肉群，強化路跑鮮少用到的穩定肌。跑者的 **RPE** (自覺用力程度) 在此賽道通常會比實際配速顯示的高，建議放下手錶數據，專注於呼吸與體感。對於目標 **BQ** 的跑者，此賽賽道可作為耐力儲備期的壓力測試。
