# 🇯🇴 約旦馬拉松賽事詳情 (Jordan Race Intelligence)

Last Updated: 2026-04-13
Author: AI Coach


### 安曼國際馬拉松 (Amman International Marathon)

1. **歷史背景**:
   安曼國際馬拉松由 Run Jordan 組織，是約旦最具代表性的體育賽事。賽事通常於每年 10 月舉行，旨在推廣健康生活並展示約旦首都的文化魅力。賽道起終點通常設於安曼市中心著名的羅馬劇場（Roman Theater）附近，跑者將在奔跑中親歷這座結合現代與古代遺跡的「七丘之城」。

2. **賽道技術分析**:
   安曼是一座充滿山丘的城市，賽道特點為持續的起伏與長緩坡。儘管全馬路線會盡量選擇相對平整的市區道路，但累積爬升仍不容小覷。建議跑者在賽前強化離心收縮 (Eccentric Contraction) 訓練，以應對下坡路段對股四頭肌的衝擊。配速策略上，前半程應保持保守的 RPE 5-6，避免在早期的上坡路段過度消耗。對於有經驗的跑者，可嘗試在下坡路段利用地心引力微幅加速，追求整體的負分割 (Negative Split)。

3. **補給特色**:
   官方補給站每 2.5km 至 5km 設置，提供充足的水與電解質飲料，部分站點備有當地特色的椰棗（Dates）作為高能量補給。約旦民眾的熱情在賽道兩旁體現無遺，民間私補常出現充滿地中海風味的堅果與點心。建議跑者預先進行腸胃訓練，確保能適應賽道提供的各類碳水補給。

4. **教練專業評論**:
   安曼馬拉松是一場心智與耐力的淬煉之旅，絕非純粹的 PB 賽道，而是理想的高品質 LSD (Long Slow Distance) 移地訓練場所。由於安曼的高海拔（約 700-1000m）與乾燥空氣，跑者需密切監測 RPE 以防過早脫水。對於追求挑戰的跑者，此賽事的「含金量」在於其地形多變性，是檢驗核心穩定度與步頻調整能力的絕佳實戰。
