# 🇬🇷 希臘賽事詳情 (Greece Race Details)

> **Last Updated**: 2026-04-13
> **Author**: AI Coach



### 阿克西奧斯跑步節 (Axios Running Festival)

1. **歷史背景**: 阿克西奧斯跑步節 (Axios Running Festival) 在希臘第二大城塞薩洛尼基 (Thessaloniki) 近郊舉行，賽道貫穿阿克西奧斯河三角洲國家公園 (Axios Delta National Park)。此區為希臘重要的濕地生態保護區，賽場旨在結合生態保育與體育競技，讓跑者在翠綠的河岸景觀與豐富的鳥類棲息地中，體驗獨特的自然韻律與地方文化，是希臘北部極具自然特色與環保意識的體育盛事。

2. **賽道技術分析**: 賽道海拔起伏極低，屬「極致平坦」的地貌，這意味著跑者的**離心收縮**負擔較小，極利於維持高度一致的跑姿與步頻。主要路面為平整柏油路，動能回饋極佳。由於緊鄰河岸，環境變量中需特別注意 5 月份的濕度與偶發的河谷風向。建議配速策略採**負分割 (Negative Split)**，前段穩定 **RPE**（自覺運動強度），後段若風向許可則大膽推進以衝刺個人 **PB**。

3. **補給特色**: 官方補給站間距約 2.5 至 5 公里，提供充足的電解質飲品、香蕉及能量膠。由於 5 月氣溫可能升至 20°C 以上，建議跑者嚴格執行補給計畫，避免電解質失衡。對於目標設定為 **BQ** (波士頓馬拉松資格) 的菁英跑者，應事先進行腸胃訓練，確保能穩定吸收高碳水化合物而不影響競技節奏。

4. **教練專業評論**: 本賽事定位為「春季 PB 殿堂」，是針對年度目標賽事進行強度測試的絕佳場合。平坦賽道能精確檢測跑者的心肺與配速掌握度。建議賽前二至三週將長距離訓練 (**LSD**) 的配速略微上提，模擬比賽強度。此賽道對核心穩定性要求極高，因單調的平緩地形反而容易造成特定肌群過度疲勞，需加強抗阻力訓練。
