### 奧林匹克日路跑 (Olympic Day Run Greece)

1. **歷史背景**: 塞薩洛尼基奧林匹克日路跑（Olympic Day Run Greece）是希臘境內唯一獲得國際奧委會（IOC）授權、並由希臘奧委會與塞薩洛尼基奧林匹克博物館共同主辦的官方賽事。賽事旨在傳承奧林匹克價值觀，不僅是一場競技，更是一場致敬體育精神的盛典。自舉辦以來，已成為地中海地區最具象徵意義的道路跑賽事之一。

2. **賽道技術分析**: 賽道沿著塞薩洛尼基著名的海岸線延伸，具備極致平坦的特性，總爬升微乎其微。起點設於亞里斯多德廣場，沿途經過象徵性的地標「白塔」，終點位於地標雕塑「雨傘」。技術上建議採取 **負分割 (Negative Split)** 策略，前半程應密切觀察 **RPE**（自覺運動強度），避免受海風（如 Vardar 風）影響節奏；後半程在寬闊平直的柏油路面上全力加速。

3. **補給特色**: 官方補給站提供充足的電解質飲品、水以及在地特色的能量補充品。由於賽事規格極高，補給節奏精確，跑者能在 5 公里後每隔 2.5 公里獲得支援。補給區志工展現了典型的希臘式熱情，對於維持長距離高強度下的心理韌性與體能恢復極有幫助。

4. **教練專業評論**: 此賽事是挑戰 **PB** 的絕佳舞台，其平直度在歐洲賽事中名列前茅。對於菁英跑者而言，它是追求高速表現的殿堂；對於市民跑者，則是進行 **LSD**（長距離慢跑）高強度實戰演練的理想選擇。建議賽前加強下肢的 **離心收縮** 訓練，以應對終點前的硬地衝刺。整體而言，這是一場兼具歷史榮譽感與速度博弈的 A 級賽事。
