# 🇬🇷 希臘賽事詳情庫 (Greece Race Intelligence)

### 波賽頓雅典半程馬拉松 (Poseidon Athens Half Marathon)

1. **歷史背景**: 以希臘海神波賽頓命名的賽事，每年於雅典南郊的法利羅灣（Phaliro Bay）盛大舉行。此賽事不僅是競技的舞台，更結合了現代都市美學與薩羅尼克灣的蔚藍海景，旨在推廣奧林匹克運動精神與環境保護意識。作為雅典春季最具代表性的長跑活動之一，它吸引了大量尋求海濱跑步體驗的國際跑者。

2. **賽道技術分析**: 賽道主要沿著法利羅灣的海岸線延伸，地形極度平坦，海拔起伏幾乎可以忽略。這種極致平坦的特性有利於跑者維持穩定的步頻與節奏。然而，濱海路段需密切關注薩羅尼克灣的海風影響，逆風時 RPE（自覺強度）會顯著提升。教練建議全程採用平穩配速策略，並在後半程根據風向嘗試「負分割」（Negative Split）衝刺，以挑戰個人 PB。

3. **補給特色**: 官方補給站每 2.5 公里設置一次，提供電解質飲料、礦泉水與新鮮香蕉，確保跑者在高濕度的海濱環境中能及時補充流失的水分。賽道沿途常有熱情的當地民眾提供特色補給，與充滿希臘風情的音樂共同營造出絕佳的競技節奏。

4. **教練專業評論**: 此賽道是追求半馬 PB 的絕佳場地，其平坦的地形將腿部肌肉的離心收縮損耗降至最低。對於精英跑者，這是檢驗冬訓成果的理想戰場；對於市民跑者，則可視為一場高品質的 LSD（長距離慢跑）訓練。建議參賽者加強核心穩定訓練，以應對濱海多變的側風，保持跑姿穩定。
