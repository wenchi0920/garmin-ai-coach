# 阿爾瑪提馬拉松 (Almaty Marathon)


### 阿爾瑪提馬拉松 (Almaty Marathon)

1. **歷史背景**: 自 2012 年首屆舉辦以來，阿爾瑪提馬拉松已成長為中亞地區規模最大的慈善體育賽事。賽事不僅旨在推廣健康生活，更將部分收入捐贈給慈善機構，支持當地的社會公益項目。賽道穿越哈薩克舊都阿爾瑪提的歷史核心區，背景則是壯麗的天山山脈 (Zailiyskiy Alatau)，讓參賽者在現代都市景觀與壯闊自然景觀間穿梭，體驗中亞絲綢之路的現代脈動。

2. **賽道技術分析**: 賽道海拔介於 700 米至 900 米之間，屬於亞高原地形。賽道起伏明顯，起點與終點通常設於共和國廣場，全程包含明顯的海拔爬升。由於海拔較高，空氣含氧量略低，對心肺耐力與 **RPE** (自覺用力程度) 的掌控要求極高。建議採取 **負分割 (Negative Split)** 策略，在前半段的緩坡攀升中保持保守，保留體力於後半段的緩降坡衝刺。下坡段需特別注意股四頭肌的 **離心收縮** 負荷，避免腿部過早力竭。

3. **補給特色**: 官方補給點密度極高，提供充足的飲用水、電解質飲料及香蕉、橘子等傳統補給。當地跑者習慣在賽後享用傳統的哈薩克美食，但在賽事過程中，建議國際跑者仍以自備的能量膠與鹽錠為主，特別是在亞高原環境下，每小時應攝取 30-60 克碳水化合物，並嚴格監控電解質平衡以防抽筋。

4. **教練專業評論**: 阿爾瑪提馬拉松雖然並非追求 **PB** 的平坦賽道，但其海拔與坡度變化使其成為極佳的 **LSD** 長距離訓練場與心智淬煉挑戰。對於志在 **BQ** 的精英跑者，在此賽事獲得優異成績將證明具備極強的抗缺氧能力與坡度適應力。這是一場視覺與生理雙重挑戰的賽事，建議賽前進行 2-3 週的亞高原熱適應訓練。
