### Garmin Run 台北 (Garmin Run Taipei)
*   **1. 歷史背景**：Garmin Run 台北由全球穿戴裝置品牌 Garmin 主辦，以科學化數據導引為核心，自創辦以來憑藉「From Zero to Hero」的精神，迅速成為亞洲具指標性的專業路跑賽事。每年 9 月在台北引爆熱潮，吸引數千名追求卓越的跑者，透過數據校準突破自我限制，展現了結合科技與競技的運動新紀元。
*   **2. 賽道技術分析**：賽道橫跨大直美堤河濱公園，地勢極平坦，是拼搏個人紀錄 (PB) 的黃金賽道。技術關鍵在於 9 月高溫高濕的 RPE 管理，以及河濱段長直線的步頻節奏維持。建議採取負分割 (Negative Split) 策略，後半程在維持核心穩定的前提下發力。需強化下肢肌肉的離心收縮耐受度，以對抗高強度配速下的疲勞。
*   **3. 補給特色**：補給規劃體現極致的科學精神。除了精準配置的電解質、鹽錠與香蕉，更提供針對專業跑者設計的能量包與賽後營養恢復餐。站點節奏精簡，旨在減少因停頓造成的配速波動。此外，提供專業按摩與數據分析回饋，讓補給從單純的能量補充升華為整體的科學修復體系。
*   **4. 教練專業評論**：從訓練科學視角視之，此賽是檢驗夏季訓練成果的最佳殿堂，具備極高的 PB 潛力與 BQ (Boston Qualify) 墊腳石價值。強大的專業 Pacer 陣容能提供穩定參考。教練建議將此賽定位為賽季前的「體感總複習」或高品質的 LSD 實戰演練。重點在於利用科學數據精確掌控心率區間，切忌因亢奮超速而導致後半程崩解。若能在此環境下穩定發揮，代表你的基礎耐力與配速紀律已臻成熟。
