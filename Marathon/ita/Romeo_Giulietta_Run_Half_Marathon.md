### 羅密歐與朱麗葉半程馬拉松 (Romeo&Giulietta Run Half Marathon)

1. **歷史背景**: 位於義大利維羅納（Verona），這座因莎士比亞筆下羅密歐與朱麗葉而聞名的「愛之城」。賽事結合了浪漫情懷與嚴謹的競技精神，終點設在具兩千年歷史的維羅納圓形競技場（Arena di Verona），是歐洲冬季備受矚目的半馬賽事，常與情人節氣氛相結合。

2. **賽道技術分析**: 賽道總體平緩，海拔起伏微小，非常適合追求 PB。然而，維羅納古城區內存在部分石板路（Cobblestones），對腳踝穩定性與「離心收縮」能力有一定要求。建議採取「負分割（Negative Split）」策略，前 10K 穩定節奏，後半程利用平直路段加速。二月氣溫較低（約 3-8°C），熱身必須充足，避免 RPE 在初期過高。

3. **補給特色**: 提供標準的電解質飲品與能量膠。義大利賽事特色在於終點後的豐富補給，常包含當地麵包、義式點心與熱飲，對恢復體力極有幫助，讓跑者在完賽後能迅速緩解低溫帶來的不適。

4. **教練專業評論**: 此賽事定位為「PB 聖殿」與「賽季啟動跑」。低溫環境極利於心肺發揮，是挑戰半馬個人紀錄的絕佳機會。訓練中應加強石板路模擬跑，並提升核心穩定度以應對不平整路面。對於全馬跑者，此賽事亦可作為春季大滿貫賽前的高強度 LSD 測試，檢測體能狀態。
