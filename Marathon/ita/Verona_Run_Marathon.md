### 維洛那馬拉松 (Verona Run Marathon)

1. **歷史背景**:
維洛那馬拉松起源於 2002 年，賽道設立在被列為 UNESCO 世界文化遺產的維洛那古城。這座城市因莎士比亞筆下的《羅密歐與朱麗葉》而聞名於世，被譽為「愛之城」。賽事的起終點位於布拉廣場（Piazza Bra），緊鄰著名的維洛那阿雷納（Arena di Verona）羅馬圓形劇場，讓跑者在完賽瞬間體驗兩千年的歷史震撼與浪漫氣息。

2. **賽道技術分析**:
賽道整體極為平緩，海拔起伏微小，非常有利於追求 **PB**。路線結合了阿迪傑河（Adige）沿岸的自然風光與中世紀古城的石板路。技術重點在於古城區的窄巷彎道，建議採取 **負分割 (Negative Split)** 策略，在前半程維持穩定的 **RPE**，後半程進入市區後利用群眾熱情進行最後衝刺。對於腳踝穩定性與**離心收縮**能力有一定要求，以應對部分古老的石板路段。

3. **補給特色**:
賽道每 5 公里設置標準補給站，提供水、電解質飲料及義大利特色點心（如乾果、餅乾）。後半程提供能量膠與水果（香蕉、橘子）。完賽後，跑者可享用豐富的義式點心與啤酒，在阿雷納劇場旁慶祝。建議跑者在長距離訓練（**LSD**）中預先測試義式補給品的耐受度，以確保腸胃適應。

4. **教練專業評論**:
這是一場兼具觀光美學與競技效率的優質賽事。賽道平整度高，極具 **BQ** 潛力。其定位不僅是挑戰極限的競技場，更是深度體驗北義文化的移動盛宴。建議將此賽事作為年度目標，並在訓練中加入模擬石板路面的穩定性訓練。對於初馬跑者而言，這裡的浪漫氛圍與平易近人的賽道難度，是絕佳的首選之地。
