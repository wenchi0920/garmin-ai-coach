# 🇪🇹 衣索比亞馬拉松賽事詳情 (Ethiopia Race Database)

> **Last Updated**: 2026-04-14
> **Author**: AI Coach


### 衣索比亞大跑步 10km (Great Ethiopian Run 10km)

1. **歷史背景**:
   衣索比亞大跑步 (Great Ethiopian Run) 由傳奇長跑巨星海勒·格布雷斯拉希 (Haile Gebrselassie) 於 2001 年創立。這場在首都阿迪斯阿貝巴 (Addis Ababa) 舉行的賽事，已發展成為非洲規模最大的路跑盛會。賽事初衷是為了推廣大眾體育與社會公益，吸引了來自世界各地的頂尖好手與數萬名當地跑者。街道上充滿了音樂、歡呼與鮮豔的黃綠紅國旗色，展現了「長跑王國」深厚的跑步文化底蘊，是感受非洲田徑熱情的絕佳窗口。

2. **賽道技術分析**:
   此賽道的核心挑戰在於「高海拔」。阿迪斯阿貝巴平均海拔高達 2,355 公尺，空氣稀薄對心肺功能與最大攝氧量 (VO2 Max) 提出極高要求，預期 RPE 會比平地顯著增加。賽道地形多起伏，包含數個長上坡與急彎，考驗跑者的核心穩定度與下肢力量。策略上，建議開賽採用保守配速，避免過早進入無氧區間。若要追求優異成績，賽前的「高海拔適應」或「熱適應」訓練至關重要，後半程應專注於呼吸節奏與動力鏈的維持。

3. **補給特色**:
   官方補給站提供充足的水分與電解質飲品，但在高海拔乾燥氣候下，跑者應更頻繁地進行小口補水。現場補給展現了東非特色，除了標準運動補給，終點附近常有當地特色補給與強烈民族風格的音樂慶祝。考慮到高海拔對消化系統的影響，賽前與賽中的腸胃訓練同樣不可忽視。建議跑者攜帶個人習慣的能量膠，以應對長爬坡後的高強度能量消耗。

4. **教練專業評論**:
   作為一名資深教練，我將此賽事定義為「心智與高海拔的終極試煉」。在此海拔高度，追求平地 PB (Personal Best) 是不切實際的，應將其定位為「高原能力檢測跑」或「深度馬拉松文化之旅」。關鍵在於離心收縮訓練以對抗下坡衝擊，以及利用負分割 (Negative Split) 策略管理心率。雖然不具備 BQ 潛力，但在此完成 10km 所獲得的心理韌性，將對未來任何平地馬拉松產生巨大的助益。
