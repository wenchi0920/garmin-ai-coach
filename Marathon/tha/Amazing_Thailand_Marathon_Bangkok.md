# 泰國曼谷馬拉松 (Amazing Thailand Marathon Bangkok)


### 泰國曼谷馬拉松 (Amazing Thailand Marathon Bangkok)

1. **歷史背景**: 作為泰國官方主導的首都市中心馬拉松，ATMBK 自 2018 年首屆以來，迅速成長為亞洲具備高度國際影響力的賽事。該賽事由泰國旅遊局（TAT）發起，旨在展現曼谷現代與傳統交融的魅力，並致力於打造符合 WA 菁英標誌（Elite Label）等級的專業競技平台。起跑於拉加曼加拉國家體育場，終點則設於極具歷史意義的大皇宮附近，象徵跑向這座城市的靈魂。

2. **賽道技術分析**: 賽道總體而言相對平緩，但熱帶的高溫與高濕度是跑者最大的考驗。賽程前期包含部分高速路段與跨橋路段，需強化核心穩定性以應對坡度變化帶來的肌肉衝擊。配速策略建議採用保守起手，在 30K 後進入市區前確保補給充足。由於清晨起跑，跑者應提前進行熱適應訓練。賽道後半段直路較多，若體力充沛可嘗試執行負分割 (Negative Split)，但需注意 GPS 訊號在摩天大樓群中可能產生的飄移。

3. **補給特色**: 官方補給展現了泰式熱忱，除了標準的電解質飲料與能量膠外，賽後補給區更提供芒果糯米飯、道地泰式炒河粉及各式清涼水果。針對長跑過程中對碳水補給的需求，賽道每 2 公里設有水站，每 5 公里備有運動飲料，確保跑者在高溫環境下維持生理機能。

4. **教練專業評論**: 曼谷馬拉松並非典型的 PB 聖殿，但絕對是心智淬煉與熱適應的最佳場域。其賽程定位可視為冬訓期間的 LSD 重點測試賽。對於追求 BQ 的跑者，此賽事對下肢的離心收縮訓練有一定要求，特別是應對路面轉換。建議跑者依據 RPE (自覺運動強度) 來微調配速，而非盲目跟隨手錶數據，以平衡極端環境帶來的生理負荷。
