# 🇹🇭 泰國馬拉松 (Thailand Marathon)

> **Last Updated**: 2026-04-19
> **Author**: AI Coach


### 泰國錫克跑 (Thai Sikh Run)

1. **歷史背景**: 
泰國錫克跑（Thai Sikh Run）由曼谷錫克教社群主辦，旨在紀念錫克教創始人拿那克大師（Guru Nanak Dev Ji）的誕辰。這場賽事是曼谷歷史最悠久的慈善路跑之一，起點設在曼谷著名的「小印度」拍乎叻（Phahurat），充滿濃厚的宗教包容與文化融合氣息。賽事歷史超過 30 年，不僅是體育競技，更是曼谷錫克社群展現慷慨與活力的年度盛事。

2. **賽道技術分析**: 
賽道從舊城區的心臟地帶出發，途經充滿歷史感的紀念大橋（Memorial Bridge），並環繞大皇宮與國柱廟。地勢整體平坦，但舊城區街道較為狹窄且彎道較多，對維持穩定步頻是一大挑戰。建議採取**負分割 (Negative Split)** 策略：前半段在狹窄街道中保持節奏（RPE 4-5），待通過紀念大橋進入較寬闊路段後再適度加速。注意橋面坡度雖然緩和，但多次往返會累積**離心收縮**的負擔。

3. **補給特色**: 
這場賽事以其「傳奇級」的賽後補給聞名。終點後提供豐富的道地印度素食盛宴，包括香料扁豆湯（Dal）、現烤烤餅（Roti）及各式印度甜點（Ladoo）。補給站點分佈密集，提供充足的水分與電解質。對於跑者而言，這是一場胃口與意志力同時得到滿足的比賽。

4. **教練專業評論**: 
泰國錫克跑雖然具備 **AIMS 認證**，但因舊城區路況與高濕度環境，並非追求極致 **PB** 的首選賽道。我將其定義為一場極佳的**文化體驗 LSD (Long Slow Distance)**。跑者在此可以練習在複雜地形中的體感監控與 **RPE** 管理。賽事氛圍極佳，非常適合做為賽季中的調整賽，或是帶領親友一同感受曼谷多元文化的洗禮。
