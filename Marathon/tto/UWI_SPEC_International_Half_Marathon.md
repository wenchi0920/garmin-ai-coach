# 特立尼達和多巴哥賽事詳情庫


### UWI SPEC 國際半程馬拉松 (UWI SPEC International Half-Marathon & 5K)

1. **歷史背景**:
自 2004 年由西印度大學 (UWI) 聖奧古斯丁分校運動與體育教育中心發起，UWI SPEC 半馬已成為特立尼達島歷史最悠久且最受尊崇的路跑賽事之一。賽事不僅旨在推廣競技體育，更與社會公益結合，每屆均吸引加勒比海地區頂尖長跑好手與國際選手同場競技，體現了學術機構對提升區域運動水平的承諾。

2. **賽道技術分析**:
賽道起點位於聖奧古斯丁 UWI SPEC，沿著東部主要幹道 (Eastern Main Road) 往返。地形整體相對平緩，但加勒比海典型的高溫與極高濕度是影響 RPE 的關鍵變量。建議配速策略採取保守的「負分割 (Negative Split)」，前半程應比目標配速慢 5-10 秒以保留體能應對回程體感溫度的上升。由於路面多為柏油路且缺乏遮蔭，跑者需注意核心穩定以維持高效率步頻。

3. **補給特色**:
官方補給站每 2-3 公里設置一次，提供充足的水與電解質飲品。鑒於極端氣候，賽道特色在於噴霧降溫站的設置。賽後補給融合當地特色，提供椰子水等天然電解質與水果，幫助跑者在劇烈流汗後迅速恢復。

4. **教練專業評論**:
這是一場典型的高難度「耐熱硬仗」。雖然賽道坡度起伏不大，但濕熱環境會顯著加速心率漂移。對於追求 PB 的跑者，此賽道難度極高，更適合作為進階馬拉松訓練中的 LSD (長距離慢跑) 質量課表，或作為檢測熱適應能力的年度期末考。建議加強「離心收縮」訓練以應對後半程疲勞帶來的肌肉衝擊。賽事定位：心智淬煉與耐熱測試。
